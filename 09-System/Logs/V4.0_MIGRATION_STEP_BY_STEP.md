## SECTION 1: Pre-Migration Safety Checklist

### 1.1 Vault State Scan (MANDATORY)

**Purpose:** Establish baseline, detect unexpected changes

```bash
#!/bin/bash
# Pre-Migration Vault Scan Script
# Run this BEFORE any migration steps

echo "=== PRE-MIGRATION VAULT SCAN ==="
echo "Date: $(date)"
echo ""

VAULT_DIR="/home/simon/Workspaces/TheOptimizedBrain"
cd "$VAULT_DIR"

# 1. Git Status
echo "1. Git Status:"
git status --short | grep -v ".obsidian" || echo "   Clean (except .obsidian)"
echo ""

# 2. Count All Files by Type
echo "2. File Counts:"
echo "   Markdown files: $(find . -name '*.md' -type f | wc -l)"
echo "   Zettels (04-Zettels/): $(ls 04-Zettels/*.md 2>/dev/null | wc -l)"
echo "   MOCs (55-MOCs/): $(ls 55-MOCs/*.md 2>/dev/null | wc -l)"
echo "   LitNotes (66b/): $(ls 66-Literature/66b-LitNotes/*.md 2>/dev/null | wc -l)"
echo "   TechRefs (66c/): $(ls 66-Literature/66c-TechRefs/*.md 2>/dev/null | wc -l)"
echo "   Templates (11/): $(ls 11-Templates/*.md 2>/dev/null | wc -l)"
echo ""

# 3. Check for Uncommitted Work
echo "3. Uncommitted Changes:"
UNCOMMITTED=$(git status --short | grep -v ".obsidian" | wc -l)
if [ "$UNCOMMITTED" -eq 0 ]; then
    echo "   ✅ None (safe to proceed)"
else
    echo "   ⚠️  $UNCOMMITTED uncommitted files:"
    git status --short | grep -v ".obsidian"
    echo "   ACTION: Commit these before migration"
fi
echo ""

# 4. Check for Open Files (Obsidian)
echo "4. Obsidian Workspace Check:"
echo "   ⚠️  Manual check required: Is Obsidian closed?"
echo "   If Obsidian is open during file moves, it may cache old paths."
echo ""

# 5. Verify Backups Exist
echo "5. Backup Verification:"
if [ -f "/tmp/TheOptimizedBrain-BACKUP-20260222-1825.tar.gz" ]; then
    echo "   ✅ External archive exists ($(du -h /tmp/TheOptimizedBrain-BACKUP-20260222-1825.tar.gz | cut -f1))"
else
    echo "   ❌ External archive NOT FOUND"
fi

if git branch | grep -q "backup-pre-v4.0-migration"; then
    echo "   ✅ Git backup branch exists"
else
    echo "   ❌ Git backup branch NOT FOUND"
fi

if git tag | grep -q "v3.0-final"; then
    echo "   ✅ Git tag exists"
else
    echo "   ❌ Git tag NOT FOUND"
fi
echo ""

# 6. Check Other Sessions
echo "6. Other Sessions Check:"
echo "   ⚠️  Manual check required: Are other agents/sessions active?"
echo "   - Check Telegram for RON messages"
echo "   - Check subagent status"
echo "   - Check 22-Dashboards/01-QUEUE.md for active work"
echo ""

# 7. Dashboard Links Audit
echo "7. Dashboard Links That Will Break:"
grep -n "file=02-Atomic\|file=66-Literature/Refs" ~/.openclaw/workspace/dashboard/*.html 2>/dev/null | head -5
echo ""

# 8. Generate Scan Report
echo "8. Generating scan report..."
SCAN_FILE="/tmp/vault-pre-migration-scan-$(date +%Y%m%d-%H%M).txt"
cat > "$SCAN_FILE" << EOF
PRE-MIGRATION SCAN REPORT
=========================
Date: $(date)
Vault: $VAULT_DIR

Git Status:
$(git status)

File Counts:
Markdown: $(find . -name '*.md' -type f | wc -l)
Commits ahead: $(git rev-list --count origin/main..main 2>/dev/null || echo "unknown")

Backup Status:
External: $(test -f /tmp/TheOptimizedBrain-BACKUP-20260222-1825.tar.gz && echo "EXISTS" || echo "MISSING")
Git branch: $(git branch | grep backup-pre-v4.0-migration | wc -l) found
Git tag: $(git tag | grep v3.0-final | wc -l) found

Ready for migration: $(test "$UNCOMMITTED" -eq 0 && echo "YES" || echo "NO - uncommitted work")
EOF

echo "   Report saved: $SCAN_FILE"
echo ""
echo "=== SCAN COMPLETE ==="
```

**Step-by-Step:**

```bash

# 3. Create checkpoint commit

echo "✅ Restore test passed"
```
