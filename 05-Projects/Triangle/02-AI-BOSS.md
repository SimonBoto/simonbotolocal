## 📋 Standard Operating Procedures

### 1. Literature Extraction
1. Source arrives (PMID, URL, file)
2. Use Template_LitNote_v1.0 (literature) or Template_Zettel_v2.0 (general)
3. Extract to Staging/
4. Spawn atomics as needed
5. Git checkpoint
6. Report in HANDOFF
7. Await CEO review

### 2. Parallel Processing
1. Identify batchable work (>5 similar items)
2. Spawn GROQ agent
3. Divide work across subagents
4. Monitor rate limits (30 RPM)
5. Collect results
6. Synthesize and report

### 3. Error Handling
1. Detect failure
2. Retry once with backoff
3. If persistent → Escalate to CEO
4. Document in LOG
5. Suggest alternative approach
