# Liver Function Panel - B-009-0001

## 1. IDENTIFICATION

| Marker | Full Name | What It Measures |
|--------|-----------|------------------|
| **AST (SGOT)** | Aspartate Aminotransferase | Hepatocellular injury |
| **ALT (SGPT)** | Alanine Aminotransferase | Hepatocellular injury (liver specific) |
| **GGT** | Gamma-Glutamyl Transferase | Biliary injury, detoxification, alcohol use |
| **ALP** | Alkaline Phosphatase | Biliary injury, bone turnover |
| **Bilirubin** | Total/Direct Bilirubin | Excretion function, hemolysis |
| **Albumin** | Serum Albumin | Synthetic function |
| **PT/INR** | Prothrombin Time | Synthetic function |

**Specimen:** Serum
**Collection:** No fasting required

## 2. SIMOS CLINICAL TARGETS

### AST (SGOT) - U/L
| Range | Interpretation |
|-------|----------------|
| **<20** | Optimal |
| **20-40** | Normal |
| **40-80** | Mild elevation |
| **>80** | Significant elevation |

**Note:** AST also found in heart, muscle, RBCs (less liver-specific)

### ALT (SGPT) - U/L
| Range | Interpretation |
|-------|----------------|
| **<20** | Optimal |
| **20-40** | Normal |
| **40-80** | Mild elevation (fatty liver likely) |
| **>80** | Significant elevation |

**Simos Target: ALT <40, ideally <30**
- **More liver-specific than AST**
- **Elevated in fatty liver, hepatitis, toxins**

### AST:ALT Ratio
| Ratio | Interpretation |
|-------|----------------|
| **<1.0** | Typical for fatty liver, viral hepatitis |
| **>2.0** | Alcoholic liver disease |
| **>3.0** | Strongly suggests alcohol |

### GGT - U/L
| Range | Interpretation |
|-------|----------------|
| **<20** | Optimal |
| **20-60** | Normal |
| **60-100** | Mild elevation |
| **>100** | Significant elevation |

**Simos Target: GGT <60**

**Clinical Utility:**
- **Sensitive to alcohol** (even moderate intake)
- **Biliary injury** (cholestasis)
- **Detoxification capacity**
- **Oxidative stress marker**

### ALP (Alkaline Phosphatase) - U/L
| Range | Interpretation |
|-------|----------------|
| **40-120** | Normal (adults) |
| **>120** | Elevated |

**Sources:**
- Liver (biliary)
- Bone (osteoblast activity)
- Placenta (pregnancy)

**Differentiation:**
- **GGT elevated + ALP elevated** = Liver source
- **GGT normal + ALP elevated** = Bone source

### Bilirubin - mg/dL
| Type | Range | Interpretation |
|------|-------|----------------|
| **Total** | 0.1-1.2 | Normal |
| **Direct** | 0-0.3 | Normal |
| **Indirect** | 0.1-0.9 | Normal |

**Patterns:**
- **Indirect elevated** = Hemolysis, Gilbert syndrome
- **Direct elevated** = Liver disease, biliary obstruction
- **Both elevated** = Hepatocellular injury

## 3. CLINICAL PATTERNS

### Fatty Liver (NAFLD) Pattern
| Marker | Finding |
|--------|---------|
| ALT | Elevated (40-100) |
| AST | Elevated but < ALT |
| AST:ALT | <1.0 |
| GGT | Mildly elevated |
| ALP | Normal |
| Bilirubin | Normal |

**Associated with:**
- Insulin resistance
- Obesity
- Metabolic syndrome
- Dyslipidemia

### Alcoholic Liver Disease
| Marker | Finding |
|--------|---------|
| AST | Elevated |
| ALT | Elevated but < AST |
| AST:ALT | >2.0 |
| GGT | Markedly elevated |
| MCV | Elevated (macrocytosis) |

### Viral Hepatitis
| Marker | Finding |
|--------|---------|
| ALT | Very high (>200) |
| AST | Very high |
| AST:ALT | <1.0 |
| Bilirubin | Elevated |

### Cholestasis (Biliary Obstruction)
| Marker | Finding |
|--------|---------|
| ALP | Markedly elevated |
| GGT | Markedly elevated |
| ALT | Mildly elevated |
| Bilirubin | Direct elevated |

## 4. SYNTHETIC FUNCTION MARKERS

### Albumin - g/dL
| Range | Interpretation |
|-------|----------------|
| **<3.5** | Low (chronic liver disease, malnutrition) |
| **3.5-5.0** | Normal |
| **>5.0** | High (dehydration) |

**Half-life:** 20 days
- **Reflects chronic status, not acute**

### PT/INR
| INR | Interpretation |
|-----|----------------|
| **<1.1** | Normal |
| **1.1-1.5** | Mild prolongation |
| **>1.5** | Significant (synthetic dysfunction) |

**Critical:** INR >1.5 in liver disease = poor prognosis

## 5. RED FLAGS

🚩 **Critical:**
- ALT/AST >1000 (severe hepatitis, toxin)
- INR >1.5 (synthetic failure)
- Bilirubin >10 (severe cholestasis)
- Encephalopathy (liver failure)

⚠️ **Urgent:**
- Rapidly rising transaminases
- Jaundice with elevated INR
- New ascites
