# Thyroid Panel - B-010-0001

## 1. IDENTIFICATION

| Marker | Full Name | What It Measures |
|--------|-----------|------------------|
| **TSH** | Thyroid Stimulating Hormone | Pituitary response to thyroid hormone |
| **FT4** | Free T4 | Active thyroid hormone (storage) |
| **FT3** | Free T3 | Most active thyroid hormone |
| **Anti-TPO** | Anti-Thyroid Peroxidase Antibodies | Autoimmune thyroiditis (Hashimoto's) |
| **Anti-TG** | Anti-Thyroglobulin Antibodies | Autoimmune thyroiditis |
| **Reverse T3** | Reverse T3 | Inactive form (stress marker) |

**Specimen:** Serum
**Collection:** Any time (TSH has diurnal variation, but clinical significance minimal)

## 2. SIMOS CLINICAL TARGETS

### TSH - mIU/L (μIU/mL)
| Range | Standard | Simos Optimal | Interpretation |
|-------|----------|---------------|----------------|
| **<0.5** | Low | Low | Hyperthyroidism, overtreatment |
| **0.5-2.0** | Normal | **OPTIMAL** | Ideal thyroid function |
| **2.0-4.0** | Normal | Suboptimal | Early hypothyroidism, symptoms possible |
| **4.0-10** | High | Hypothyroid | Overt hypothyroidism |
| **>10** | High | Severe | Significant hypothyroidism |

**Simos Target: TSH 0.5-2.0 mIU/L**

**Why Lower Than Standard (0.5-4.0)?**
- Many people symptomatic at TSH 2.5-4.0
- Optimal function typically TSH 1-2
- TSH >2.5 in pregnancy associated with adverse outcomes
- TSH >2 with symptoms = trial of treatment warranted

### FT4 (Free T4) - ng/dL (pmol/L)
| Range | Interpretation |
|-------|----------------|
| **<0.8** | Low (hypothyroid) |
| **0.8-1.0** | Low-normal |
| **1.0-1.5** | **OPTIMAL** |
| **1.5-1.8** | High-normal |
| **>1.8** | High (hyperthyroid) |

**Simos Target: FT4 1.0-1.5 ng/dL**

### FT3 (Free T3) - pg/mL (pmol/L)
| Range | Interpretation |
|-------|----------------|
| **<2.0** | Low (hypothyroid, conversion issue) |
| **2.0-2.5** | Low-normal |
| **2.5-3.5** | **OPTIMAL** |
| **3.5-4.5** | High-normal |
| **>4.5** | High (hyperthyroid) |

**Simos Target: FT3 2.5-3.5 pg/mL**

**T3 is the active hormone:**
- T4 converted to T3 in peripheral tissues
- T3 binds nuclear receptors
- Low T3 = hypothyroid symptoms even if T4 normal

### FT3:FT4 Ratio
| Ratio | Interpretation |
|-------|----------------|
| **<0.2** | Poor conversion (T4 → T3) |
| **0.2-0.3** | Normal conversion |
| **>0.3** | Good conversion |

**Low conversion seen in:**
- Chronic illness
- Stress
- Selenium deficiency
- Zinc deficiency
- Medications (beta blockers, steroids)

### Anti-TPO (Thyroid Peroxidase Antibodies)
| Level | Interpretation |
|-------|----------------|
| **<10** | Negative |
| **10-35** | Borderline |
| **>35** | Positive (Hashimoto's) |

**Clinical Significance:**
- **Positive = Autoimmune thyroiditis (Hashimoto's)**
- Most common cause of hypothyroidism
- May be elevated years before TSH rises
- Indicates ongoing immune attack

### Anti-TG (Thyroglobulin Antibodies)
| Level | Interpretation |
|-------|----------------|
| **<20** | Negative |
| **20-40** | Borderline |
| **>40** | Positive |

**Clinical Significance:**
- Also indicates autoimmune thyroiditis
- Less specific than Anti-TPO
- Often both positive in Hashimoto's

## 3. CLINICAL PATTERNS

### Optimal Thyroid Function
| Marker | Value |
|--------|-------|
| TSH | 0.5-2.0 |
| FT4 | 1.0-1.5 |
| FT3 | 2.5-3.5 |
| Anti-TPO | Negative |
| Anti-TG | Negative |

### Subclinical Hypothyroidism
| Marker | Value | Interpretation |
|--------|-------|----------------|
| TSH | 4-10 | Elevated |
| FT4 | Normal | Still adequate |
| FT3 | Normal or low | May be symptomatic |
| Anti-TPO | Often positive | Autoimmune cause |
| Action | | Treat if symptomatic or TPO+

### Overt Hypothyroidism
| Marker | Value |
|--------|-------|
| TSH | >10 |
| FT4 | Low |
| FT3 | Low |
| Anti-TPO | Often positive |
| Symptoms | Fatigue, weight gain, cold intolerance, depression |

### Hashimoto's Thyroiditis (Early)
| Marker | Value |
|--------|-------|
| TSH | Normal or slightly elevated |
| FT4 | Normal |
| FT3 | Normal |
| Anti-TPO | **Positive** |
| Anti-TG | Often positive |
| Action | Monitor, may need treatment later |

### Hyperthyroidism
| Marker | Value |
|--------|-------|
| TSH | <0.1 |
| FT4 | High |
| FT3 | High |
| Causes | Graves' disease, toxic nodule, thyroiditis |

### Low T3 Syndrome (Euthyroid Sick Syndrome)
| Marker | Value | Interpretation |
|--------|-------|----------------|
| TSH | Normal or low | |
| FT4 | Normal | |
| FT3 | **Low** | Poor peripheral conversion |
| rT3 | **High** | Stress response |
| Cause | Chronic illness, stress, starvation | |
| Action | Treat underlying cause, not thyroid | |

## 4. SYMPTOMS CORRELATION

### Hypothyroid Symptoms
| System | Symptoms |
|--------|----------|
| **Metabolic** | Weight gain, cold intolerance, fatigue |
| **Neurological** | Depression, brain fog, memory issues |
| **Dermatological** | Dry skin, hair loss, brittle nails |
| **GI** | Constipation |
| **Cardiovascular** | Bradycardia, elevated cholesterol |
| **Reproductive** | Irregular periods, infertility |

### Hyperthyroid Symptoms
| System | Symptoms |
|--------|----------|
| **Metabolic** | Weight loss, heat intolerance |
| **Neurological** | Anxiety, tremor, insomnia |
| **Cardiovascular** | Tachycardia, palpitations |
| **GI** | Diarrhea |
| **Ocular** | Bulging eyes (Graves') |

## 5. NUTRITIONAL CONNECTIONS

### Nutrients for Thyroid Function
| Nutrient | Role | Deficiency Effect |
|----------|------|-------------------|
| **Iodine** | Thyroid hormone synthesis | Goiter, hypothyroidism |
| **Selenium** | T4 → T3 conversion, antioxidant | Poor conversion, thyroiditis |
| **Zinc** | T3 receptor function, TSH production | Hypothyroidism |
| **Iron** | Thyroid peroxidase function | Impaired synthesis |
| **Vitamin D** | Immune regulation (autoimmune) | Hashimoto's risk |
| **Vitamin A** | T3 receptor function | Impaired signaling |
| **Tyrosine** | Thyroid hormone backbone | Low hormone production |

### Hashimoto's and Gluten
- Many Hashimoto's patients sensitive to gluten
- Molecular mimicry theory
- Trial of gluten-free diet often beneficial

## 6. SUPPLEMENTATION

### For Hypothyroidism (If Treated or Borderline)
| Nutrient | Dose | Purpose |
|----------|------|---------|
| **Selenium** | 200 mcg | T4→T3 conversion, reduce antibodies |
| **Zinc** | 15-30 mg | T3 receptor function |
| **Vitamin D** | 4000-10,000 IU | Immune regulation |
| **Iron** | If deficient | Thyroid peroxidase function |
| **Ashwagandha** | 300-600 mg | Thyroid support (traditional) |

### For Hashimoto's (Autoimmune)
| Intervention | Purpose |
|--------------|---------|
| **Selenium** | Reduce antibody levels |
| **Vitamin D** | Immune modulation |
| **Gluten-free trial** | Reduce immune trigger |
| **Low-dose naltrexone (LDN)** | Immune modulation (off-label) |
| **Curcumin** | Anti-inflammatory |

## 7. RED FLAGS

🚩 **Critical:**
- TSH >100 (myxedema coma risk)
- TSH <0.01 with high FT4 (thyroid storm risk)
- New thyroid nodule with rapid growth
- Compressive symptoms (dysphagia, dyspnea)

⚠️ **Urgent:**
- TSH >10
- Rapidly changing thyroid function
- Pregnancy with TSH >2.5

## 8. SPECIAL POPULATIONS

### Pregnancy
- **TSH target:** <2.5 (first trimester), <3.0 (later)
- **FT4:** Upper normal range
- **Anti-TPO:** Check (risk of miscarriage, postpartum thyroiditis)
- **Iodine:** 220-290 mcg daily

### Postpartum
- Risk of postpartum thyroiditis (especially TPO+)
- Check TSH at 6 weeks postpartum

### Elderly
- TSH may be slightly elevated (normal aging)
- Overtreatment risky (atrial fibrillation, osteoporosis)
- Target TSH 4-6 may be acceptable
