# HOMA-IR — Insulin Resistance Index

## 1. ETYMOLOGY

**HOMA-IR:** Homeostatic Model Assessment for Insulin Resistance

**Developed:** 1985 by Matthews et al. (Diabetologia)

**Purpose:** Simple estimate of insulin resistance from fasting glucose and insulin

source-litnote: Synthesized from clinical practice
---

## 2. PHYSIOLOGY

### Biological Function

**Mathematical model:**
```
HOMA-IR = (Fasting [[Glucose_Blood_Sugar_Energy_Z-005-0001]|Glucose] × Fasting [[Insulin_Pancreatic_Beta-Cell_Response_Z-005-0002]|Insulin]) / 405
```

*(Glucose in mg/dL, Insulin in μU/mL)*

**Interpretation:**
- Higher value = more insulin resistant
- Estimates steady-state beta cell function

### Comparison to Gold Standard

| Method | Description | Utility |
|--------|-------------|---------|
| **HOMA-IR** | Calculated from fasting values | Simple, widely used |
| **QUICKI** | 1 / (log [[Insulin_Pancreatic_Beta-Cell_Response_Z-005-0002]|insulin] + log [[Glucose_Blood_Sugar_Energy_Z-005-0001]|glucose]) | Alternative calculation |
| **OGTT** | 2-hour glucose challenge | More accurate |
| **Hyperinsulinemic-euglycemic clamp** | IV insulin/glucose | Gold standard, invasive |
| **[[HbA1c_Glycemic_Control_RBC_Memory_Z-005-0003]|HbA1c]** | Long-term glucose | Different information |

## 3. METABOLIC INTERPRETATION

### HOMA-IR Levels

| Value | Status | Interpretation |
|-------|--------|----------------|
| **<1.0** | Optimal | High insulin sensitivity |
| **1.0-1.9** | Normal | Normal insulin sensitivity |
| **2.0-2.9** | Borderline | Early insulin resistance |
| **3.0-4.9** | Elevated | Insulin resistance |
| **>5.0** | High | Significant insulin resistance |

### Clinical Significance

**Insulin resistance progression:**
1. **Normal:** HOMA-IR <2.0
2. **Compensated:** HOMA-IR 2.0-3.0, glucose normal (beta cells compensate)
3. **Decompensated:** HOMA-IR >3.0, glucose elevated (beta cell failure)

**Associated conditions:**
| Condition | Mechanism |
|-----------|-----------|
| **Type 2 diabetes** | Progressive insulin resistance |
| **Metabolic syndrome** | Core feature |
| **PCOS** | Common comorbidity |
| **NAFLD** | Hepatic insulin resistance |
| **Cardiovascular disease** | Atherogenic dyslipidemia |
| **Cancer** | Growth factor effects |
| **Alzheimer's** | "Type 3 diabetes" |

## 5. INTERVENTIONS

### For Elevated HOMA-IR

**Lifestyle (most effective):**
| Intervention | Expected Improvement |
|--------------|----------------------|
| **Weight loss (5-10%)** | 20-30% reduction |
| **Exercise** | 20-25% reduction |
| **Low-carb diet** | 30-40% reduction |
| **Time-restricted eating** | 15-20% reduction |
| **Sleep optimization** | 10-15% reduction |
| **Stress reduction** | 10-15% reduction |

**Medications:**
- Metformin (first line)
- GLP-1 agonists
- SGLT2 inhibitors
- Thiazolidinediones (TZDs)

**Target:** HOMA-IR <2.0

## 6. CONNECTED BIOMARKERS

| Biomarker | Relationship | Pattern |
|-----------|--------------|---------|
| **Fasting glucose** | Component of calculation | Elevated in IR |
| **Fasting insulin** | Component of calculation | Elevated in IR |
| **HbA1c** | Long-term glucose | Elevated in IR |
| **Triglycerides** | Metabolic syndrome | Elevated |
| **HDL** | Metabolic syndrome | Low |
| **ApoB** | Atherogenic | Elevated |

### Clinical Patterns

**Metabolic Syndrome:**
- HOMA-IR: Elevated (>2.5)
- Glucose: 100-125 mg/dL
- Triglycerides: >150 mg/dL
- HDL: <40 (M) / <50 (W)
- Waist: >40" (M) / >35" (W)

**Prediabetes with Insulin Resistance:**
- HOMA-IR: >3.0
- Glucose: 100-125 mg/dL
- Insulin: >15 μU/mL
- **Beta cells struggling to compensate**
