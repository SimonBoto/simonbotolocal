# Progesterone — Luteal Phase Hormone

## 1. ETYMOLOGY

**Progesterone:** From "pro-" (for) + "gestation" (pregnancy)

**Also called:**
- P4 (progesterone)
- Pregnanedione
- "Pregnancy hormone"

**Discovery:** 1934 by Adolf Butenandt

source-litnote: Synthesized from clinical practice
---

## 2. PHYSIOLOGY

### Biological Function

**Primary functions:**
| Function | Description |
|----------|-------------|
| **Menstrual cycle** | Prepares endometrium for implantation |
| **Pregnancy** | Maintains pregnancy, prevents contractions |
| **Breast** | Prepares for lactation |
| **Thermogenic** | Raises basal body temperature |
| **Neuroprotective** | Calming, anti-anxiety effects |
| **Bone** | Supports bone density |

### Production

**Sources:**
- **Ovary:** Corpus luteum (post-ovulation)
- **Placenta:** During pregnancy (major source)
- **Adrenal:** Small amount

### Menstrual Cycle Variation

| Phase | Progesterone (ng/mL) |
|-------|----------------------|
| **Follicular** | <1 |
| **Ovulation** | 1-3 |
| **Luteal (mid)** | 5-20 |
| **Pregnancy (1st trimester)** | 10-44 |
| **Pregnancy (3rd trimester)** | 50-280 |
| **Postmenopausal** | <1 |

## 3. METABOLIC INTERPRETATION

### Progesterone Levels

| Status | Range (ng/mL) |
|--------|---------------|
| **Follicular** | <1 |
| **Luteal (adequate)** | >10 (mid-luteal) |
| **Luteal (insufficient)** | <10 |
| **Pregnancy (1st trimester)** | 10-44 |
| **Postmenopausal** | <1 |

### Clinical Significance

**Low progesterone (luteal phase defect):**
- Infertility
- Recurrent miscarriage
- Premenstrual syndrome (PMS)
- Irregular cycles
- Estrogen dominance symptoms

**High progesterone:**
- Pregnancy
- Corpus luteum cyst
- Molar pregnancy (very high)
- Exogenous supplementation

### Progesterone/Estradiol Ratio

**Important for:**
- Estrogen dominance assessment
- Luteal phase adequacy
- PMS evaluation

**Optimal luteal ratio:** Progesterone 10-20x estradiol

## 4. COFACTORS

### Progesterone Production

| Factor | Role |
|--------|------|
| **LH** | Stimulates corpus luteum |
| **hCG** | Maintains corpus luteum in pregnancy |
| **Cholesterol** | Steroid precursor |

## 5. INTERVENTIONS

### For Low Progesterone

**Progesterone supplementation:**
- Oral (micronized)
- Vaginal (suppositories, gel)
- Intramuscular
- Topical

**Indications:**
- Luteal phase defect
- Recurrent miscarriage
- PMS
- Menopausal HRT (with estrogen)

**Natural support:**
- Vitamin B6
- Vitamin C
- Zinc
- Chasteberry (Vitex)

### For Estrogen Dominance

**Increase progesterone relative to estrogen:**
- Progesterone supplementation
- Reduce estrogen exposure
- Support liver detoxification
- Manage stress (cortisol competes with progesterone)
