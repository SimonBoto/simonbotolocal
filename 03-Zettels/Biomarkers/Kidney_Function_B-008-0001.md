# Kidney Function Panel - B-008-0001

## 1. IDENTIFICATION

| Marker | Full Name | What It Measures |
|--------|-----------|------------------|
| **Creatinine** | Serum Creatinine | Muscle metabolism waste product |
| **eGFR** | Estimated GFR | Kidney filtration rate |
| **BUN** | Blood Urea Nitrogen | Protein metabolism waste |
| **Uric Acid** | Uric Acid | Purine metabolism |
| **Cystatin C** | Cystatin C | Alternative filtration marker |
| **Electrolytes** | Na, K, Cl, CO2 | Kidney regulation |

**Specimen:** Serum
**Collection:** No fasting required

## 2. SIMOS CLINICAL TARGETS

### Creatinine - mg/dL
| Range | Interpretation | Notes |
|-------|----------------|-------|
| **<0.7** | Low | Low muscle mass, pregnancy |
| **0.7-1.0** | **OPTIMAL** | Normal function |
| **1.0-1.2** | High-normal | Monitor trend |
| **1.2-1.5** | Elevated | Reduced GFR |
| **>1.5** | High | Significant impairment |

### eGFR - mL/min/1.73m²
| Stage | eGFR | Interpretation |
|-------|------|----------------|
| **G1** | >90 | Normal or high |
| **G2** | 60-89 | **Mildly decreased** - Monitor |
| **G3a** | 45-59 | Mildly to moderately decreased |
| **G3b** | 30-44 | Moderately to severely decreased |
| **G4** | 15-29 | Severely decreased |
| **G5** | <15 | Kidney failure |

**Simos Target: eGFR >90 (G1), acceptable >60 (G2)**

### BUN - mg/dL
| Range | Interpretation |
|-------|----------------|
| **<7** | Low (malnutrition, liver disease) |
| **7-20** | **Normal** |
| **20-40** | Elevated (dehydration, high protein, renal impairment) |
| **>40** | High (significant renal impairment) |

**BUN:Creatinine Ratio:**
| Ratio | Interpretation |
|-------|----------------|
| **10-20:1** | Normal |
| **>20:1** | Prerenal (dehydration, CHF) |
| **<10:1** | Low protein intake, liver disease |

## 3. CLINICAL PATTERNS

### Dehydration Pattern
| Marker | Finding |
|--------|---------|
| BUN | Elevated |
| Creatinine | Normal or mildly elevated |
| BUN:Cr ratio | >20:1 |
| Uric acid | Elevated |

### Chronic Kidney Disease
| Feature | Finding |
|---------|---------|
| Duration | >3 months |
| eGFR | <60 |
| Creatinine | Gradually rising |

## 4. RED FLAGS

🚩 **Critical:**
- eGFR <15 (kidney failure)
- Creatinine >3.0 (severe impairment)
- Hyperkalemia >6.0 (cardiac risk)

⚠️ **Urgent:**
- Rapidly rising creatinine
- eGFR <30
