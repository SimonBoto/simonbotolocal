# Basophils — Allergic Response & Inflammation

## 1. ETYMOLOGY

**Basophil:** From Greek "basis" (base) + "philein" (to love)

**Named for:** Stains with basic dyes (blue-purple)

**Characteristic:** Large granules obscure nucleus

source-litnote: Synthesized from clinical practice
---

## 2. PHYSIOLOGY

### Biological Function

**Rarest WBC (<1%):**
- Granulocytic white blood cells
- Large, coarse granules
- S-lobed nucleus (often obscured)

**Primary functions:**
| Function | Description |
|----------|-------------|
| **Allergic reactions** | Release histamine, heparin |
| **Inflammation** | Mediate inflammatory response |
| **Immune regulation** | Interact with T-cells |
| **Parasite defense** | Contribute to anti-helminth response |

### Granule Contents

- **Histamine:** Vasodilation, bronchoconstriction
- **Heparin:** Anticoagulant
- **Leukotrienes:** Inflammatory mediators
- **IL-4, IL-13:** Th2 cytokines

## 3. METABOLIC INTERPRETATION

### Basophil Levels

| Level (%) | Status | Interpretation |
|-----------|--------|----------------|
| **<1** | Normal | Typical (0.5-1%) |
| **>2** | Elevated | Basophilia (rare) |
| **>3** | High | Significant elevation |

### Causes of Basophilia

**Myeloproliferative disorders:**
- Chronic myeloid leukemia (CML)
- Polycythemia vera
- Essential thrombocythemia
- Myelofibrosis

**Inflammatory:**
- Ulcerative colitis
- Rheumatoid arthritis (some)

**Infections:**
- Tuberculosis
- Varicella
- Influenza

**Hypothyroidism:**
- Myxedema

**Other:**
- Recovery from acute infection
- Splenectomy

### Clinical Significance

**Elevated basophils suggest:**
- Myeloproliferative disorder (most common serious cause)
- Chronic inflammation
- Hypothyroidism

**Low basophils:**
- Not clinically significant
- Can occur in acute infections, steroids

## 5. INTERVENTIONS

### For Elevated Basophils

**Identify cause:**
- Rule out myeloproliferative disorder (CBC, JAK2, BCR-ABL)
- Check thyroid function
- Evaluate for chronic inflammation

**Treatment:**
- Address underlying cause
- Hematology referral if MPN suspected
