# Pentose Phosphate Pathway Ribose NADPH

## Overview
The pentose phosphate pathway (PPP) generates ribose-5-phosphate for nucleotide synthesis and NADPH for biosynthesis and antioxidant defense.

## Two Phases of PPP

### Phase 1: Oxidative (Irreversible)
```
Glucose-6-P → 6-Phosphogluconolactone → 6-Phosphogluconate
                 ↓
        Ribulose-5-P + CO₂ + 2 NADPH
```

**Products per Glucose-6-P:**
- 2 NADPH (reducing power)
- 1 Ribulose-5-P (pentose precursor)
- 1 CO₂ (waste)

**Key Enzyme:** Glucose-6-phosphate dehydrogenase (G6PD)
- Rate-limiting step
- Inhibited by NADPH (product inhibition)

### Phase 2: Non-Oxidative (Reversible)
```
Ribulose-5-P ↔ Ribose-5-P ↔ Xylulose-5-P
         ↓
    Transketolase/Transaldolase reactions
         ↓
Glyceraldehyde-3-P + Fructose-6-P (rejoin glycolysis)
```

## Functions of PPP

### 1. Ribose Production
- **For:** DNA, RNA, ATP, NAD⁺, FAD, CoA synthesis
- **Tissues with high demand:** Bone marrow, skin, gut epithelium

### 2. NADPH Generation
**Functions of NADPH:**
| Function | Example |
|----------|---------|
| **Biosynthesis** | Fatty acid synthesis, cholesterol synthesis |
| **Antioxidant defense** | Glutathione reduction |
| **Immune function** | Respiratory burst in phagocytes |
| **Nitric oxide synthesis** | Vasodilation signaling |

### 3. Metabolic Flexibility
- Interconverts sugar phosphates
- Balances glycolytic vs. biosynthetic needs
- Links carbohydrate to nucleotide 

## Clinical Relevance

### G6PD Deficiency
| Feature | Description |
|---------|-------------|
| **Prevalence** | Most common enzyme deficiency (400M+ people) |
| **Inheritance** | X-linked |
| **Trigger** | Oxidative stress (fava beans, sulfa drugs, infection) |
| **Consequence** | Hemolytic anemia |
| **** | Insufficient NADPH → cannot maintain reduced glutathione → RBC lysis |
| **Protection** | Malaria resistance (heterozygote advantage) |

## Regulation
| Condition | Effect on PPP |
|-----------|---------------|
| **High NADP⁺/NADPH ratio** | Stimulates oxidative phase |
| **Need for ribose** | Activates both phases |
| **High ATP** | Inhibits (shunts to storage) |
| **** | Upregulates PPP enzymes |
