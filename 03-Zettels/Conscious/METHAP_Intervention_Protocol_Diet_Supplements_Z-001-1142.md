## Overview
The METHAP intervention combines targeted dietary modifications with evidence-based supplementation to improve insulin sensitivity and metabolic health in insulin-resistant participants.

## Dietary Intervention

### Macronutrient Targets
| Parameter | Target | Rationale |
|-----------|--------|-----------|
| **Carbohydrates** | 40-45% calories | Moderate reduction |
| **Protein** | 25-30% calories | Muscle preservation |
| **Fat** | 30-35% calories | Healthy fats emphasized |
| **Fiber** | >30g/day | Gut health, satiety |

### Carbohydrate Quality
| Recommendation | Implementation |
|----------------|----------------|
| **Low glycemic index** | <55 average |
| **Complex carbs** | Whole grains, legumes |
| **Limit refined sugar** | <25g added sugar/day |
| **Vegetable focus** | Non-starchy priority |

### Fat Composition
| Type | Target | Sources |
|------|--------|---------|
| **Saturated fat** | <10% calories | Limit red meat, dairy |
| **Monounsaturated** | 15-20% | Olive oil, nuts, avocado |
| **Polyunsaturated** | 10-15% | Fatty fish, seeds |
| **Omega-3** | 2-3g EPA+DHA/day | Fish, supplements |
| **Omega-6:3 ratio** | <4:1 | Reduce seed oils |

### Meal Timing
| Strategy | Protocol |
|----------|----------|
| **Regular meals** | 3 meals + 1-2 snacks |
| **Time-restricted eating** | 10-12 hour eating window |
| **Breakfast emphasis** | Protein-rich morning meal |
| **Evening restriction** | Light dinner, no late eating |

## Supplement Protocol

### Core Supplements
| Supplement | Dose | Timing | Rationale |
|------------|------|--------|-----------|
| **3** | 4000 IU/day | Morning | Insulin sensitivity |
| **Magnesium** | 400-600mg/day | Evening | Glucose  |
| **Omega-3 (EPA/DHA)** | 2-3g/day | With meals | Anti-inflammatory |
| **Chromium** | 200-1000mcg/day | Morning | Glucose transport |
| **Alpha-lipoic acid** | 600mg/day | Divided | Antioxidant, insulin |
| **Berberine** | 500mg 2-3x/day | With meals | AMPK activation |

### Optional/Targeted
| Supplement | Indication | Dose |
|------------|------------|------|
| **Myo-inositol** | PCOS, metabolic syndrome | 2-4g/day |
| **Cinnamon** | Glucose control | 1-3g/day |
| **Zinc** | If deficient | 15-30mg/day |
| **B-complex** | Metabolic support | As labeled |

## Intervention Duration

| Phase | Duration | Activities |
|-------|----------|------------|
| **Baseline** | Week 0 | Testing, education |
| **Active intervention** | 12 weeks | Diet + supplements |
| **Post-intervention** | Week 12+ | Retesting, analysis |

## Compliance Monitoring

### Dietary Assessment
| Method | Frequency |
|--------|-----------|
| **Food diaries** | Weekly (first month) |
| **24-hour recalls** | Bi-weekly |
| **Dietitian review** | Monthly |

### Supplement Tracking
| Method | Implementation |
|--------|----------------|
| **Pill counts** | Return at visits |
| **Self-report** | Daily checklist |
| ** verification** | Omega-3 index, 25(OH)D |

## Safety Monitoring

### Contraindications
| Condition | Action |
|-----------|--------|
| **Kidney disease** | Magnesium caution |
| **Bleeding risk** | Omega-3 monitoring |
| **Medication interactions** | Physician review |

### Adverse Events
| Symptom | Response |
|---------|----------|
| **GI upset** | Adjust supplement timing |
| **Hypoglycemia** | Monitor with meds |
| **Any serious AE** | Discontinue, report |

## Expected Outcomes

### Primary
| Outcome | Target |
|---------|--------|
| **HOMA-IR reduction** | >20% improvement |
| **Metabolomic shift** | Toward healthy pattern |

### Secondary
| Outcome | Target |
|---------|--------|
| **Weight loss** | 3-5% body weight |
| **HbA1c reduction** | 0.3-0.5% |
| **Lipid improvement** | TG↓, HDL↑ |
