# Amino Acids to Fumarate: Phenylalanine and Tyrosine

## The Pathway

### Phenylalanine → Tyrosine
```
Phenylalanine + O₂ + BH₄ → Tyrosine + BH₂ + H₂O
```
**Enzyme:** Phenylalanine hydroxylase (PAH)
**Cofactors:**
- **Tetrahydrobiopterin (BH₄):** Redox cofactor
- **Iron (Fe²⁺):** Required for hydroxylation

### Tyrosine → Fumarate
```
Tyrosine → p-hydroxyphenylpyruvate → Homogentisate → Maleylacetoacetate →
Fumarylacetoacetate → Fumarate + Acetoacetate
```
**Enzymes:** Tyrosine aminotransferase, p-hydroxyphenylpyruvate dioxygenase, etc.

## Classification: Both Glucogenic and Ketogenic
- **Fumarate:** Glucogenic (enters Krebs, can make glucose)
- **Acetoacetate:** Ketogenic (cannot make glucose)

## Clinical: Phenylketonuria (PKU)

### Cause
**Phenylalanine hydroxylase deficiency** (or BH₄ synthesis defect)

### Consequences
- Phenylalanine accumulation → neurotoxic
- Tyrosine becomes essential (cannot synthesize)
- Hypopigmentation (tyrosine is melanin precursor)
- Intellectual disability if untreated

### Treatment
- **Phenylalanine-restricted diet** (lifelong)
- **Sapropterin dihydrochloride** (BH₄ analog, for some variants)
- **Pegvaliase** (enzyme therapy, FDA approved)

## Clinical: Tyrosinemias

### Type I (Most Severe)
**Fumarylacetoacetate hydrolase deficiency**
- Accumulation of succinylacetone
- Liver failure, renal dysfunction
- Treatment: Nitisinone (inhibits upstream enzyme)

### Type II
**Tyrosine aminotransferase deficiency**
- Ocular and skin lesions
- Mental retardation

### Type III
**4-Hydroxyphenylpyruvate dioxygenase deficiency**
- Mild, often asymptomatic

## Folate and Iron Connection
**Phenylalanine hydroxylase requires:**
- BH₄ (tetrahydrobiopterin) — folate-related metabolism
- Iron (Fe²⁺) — for hydroxylation chemistry
