# Clinician Training Competency Checklist

## Overview
Competency verification at each training stage before independent practice authorization.

## Phase 1: Foundation Competency (Weeks 1-2)

### Biochemical Literacy
- [ ] Explain  as information flow
- [ ] Describe l function beyond "powerhouse"
- [ ] Define redox balance and NAD+/NADH
- [ ] Identify the five core pathways

### Assessment
- [ ] Pass oral exam on pathway function
- [ ] Identify three failure patterns from case examples
- [ ] Explain Krebs cycle to simulated patient

**Sign-off required:** Senior clinician

## Phase 2: OAT Mastery (Weeks 3-4)

### Pattern Recognition
- [ ] Interpret OAT by category without prompting
- [ ] Identify common patterns (mitochondrial, gut, B-vitamin)
- [ ] Distinguish artifact from true abnormality

### Case-Based Competency
- [ ] Correctly interpret 10 practice cases
- [ ] Achieve 80% concordance with senior clinician
- [ ] Complete case presentation with reasoning

**Sign-off required:** Senior clinician + competency exam

## Phase 3: Lipidomics (Weeks 5-6)

### Fatty Acid Interpretation
- [ ] Explain desaturase pathway
- [ ] Calculate and interpret AA/EPA ratio
- [ ] Identify dietary patterns from fatty acid profile
- [ ] Connect lipids to mitochondria and OAT

### Clinical Application
- [ ] Prescribe omega-3 dosing appropriately
- [ ] Explain "seed oil collapse" concept
- [ ] Integrate lipid data with OAT findings

**Sign-off required:** Senior clinician

## Phase 4: Integration (Weeks 7-8)

### Multi-Modal Integration
- [ ] Synthesize OAT + fatty acids + basic labs
- [ ] Apply three integration algorithms
- [ ] Build personalized treatment plans

### Clinical Practice
- [ ] Present 5 complete cases independently
- [ ] Demonstrate metabolic storytelling
- [ ] Handle patient objections appropriately
- [ ] Write standardized reports

### Communication Skills
- [ ] Explain findings to simulated patient
- [ ] Navigate difficult conversation scenario
- [ ] Receive satisfactory patient feedback (if available)

**Sign-off required:** Senior clinician + final competency exam

## Final Authorization Requirements

### Before Independent Practice
- [ ] All phase competencies signed off
- [ ] Minimum 20 supervised cases
- [ ] 90% concordance with senior clinician on interpretation
- [ ] Satisfactory patient communication assessments
- [ ] Completion of all training modules

### Ongoing Competency
- [ ] Annual case review
- [ ] Continuing education in 
- [ ] Quality metric participation

## Remediation
If competency not achieved:
1. Identify specific gaps
2. Additional supervised practice
3. Targeted education
4. Re-assessment
