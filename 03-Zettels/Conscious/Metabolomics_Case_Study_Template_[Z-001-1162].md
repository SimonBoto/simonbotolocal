#  Case Study Template

## Purpose

Standardized template for documenting and presenting clinical cases in metabolomics practice. Ensures comprehensive data capture and facilitates knowledge sharing across the clinical team.

## Usage Guidelines

### For Case Documentation
1. Complete all sections at initial visit
2. Update follow-up sections at each visit
3. Upload to team repository (GitHub)
4. Tag with relevant MOCs and conditions

### For 4-Arm Presentations
- **Arm 4 (Clinical Cases)** uses this template
- Present 1-2 cases per education cycle
- Focus on teaching points and pattern recognition
- Include metabolomics patterns and clinical correlations

### For Knowledge Base
- Anonymize all patient identifiers
- Include unique case identifier (non-PHI)
- Cross-link to relevant zettels and protocols
- Update with long-term outcomes
