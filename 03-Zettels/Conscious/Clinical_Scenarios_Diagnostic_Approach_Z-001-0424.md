#  Scenarios — Diagnostic Approach

## Red Flags for B12 Deficiency (Not MM)

| Feature | B12 Deficiency | MM |
|---------|---------------|-----|
| **Anemia morphology** | Macrocytic | Normocytic |
| **Neutrophils** | Hypersegmented | Normal |
| **Neurological symptoms** | Paresthesias, ataxia | Rare |
| **Risk factors** | Vegan, gastric surgery, PPI, elderly | Age >60 |
| **Bone pain** | Absent | Common |
| **** | Normal | Elevated |

## Diagnostic Algorithm

```
Suspected hematological malignancy
            ↓
    Check B12, MMA first
            ↓
    ┌───────┴───────┐
    ↓               ↓
Low B12/MMA      Normal B12/MMA
    ↓               ↓
Treat B12      Check SPEP, free light chains
    ↓               ↓
Monitor        M-protein present?
response       ↓
              Yes → MM workup
              No → Other investigation
```

## When to Biopsy

**Indications:**
- Unclear diagnosis
- Need to exclude MM definitively
- Concurrent conditions suspected
- Atypical features
