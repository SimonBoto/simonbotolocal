# Lipoproteins — The Transport System

## Core Claim
Lipoproteins are macromolecular complexes with distinct density classes (chylomicrons → VLDL → IDL → LDL → HDL), each serving specific transport functions, with apolipoproteins determining metabolic fate and receptor interactions.

## Evidence

### Lipoprotein Classes

| Class | Density | Origin | Primary Lipid | Core Function |
|-------|---------|--------|---------------|---------------|
| **Chylomicrons** | Lowest (<0.95) | Intestine | Triglycerides | Dietary fat transport to tissues |
| **VLDL** | Very low (0.95-1.006) | Liver | Triglycerides | Endogenous fat transport |
| **IDL** | Intermediate (1.006-1.019) | VLDL remnant | Cholesterol/TG | Transitional to LDL |
| **LDL** | Low (1.019-1.063) | IDL remnant | Cholesterol | Cell delivery |
| **HDL** | High (1.063-1.21) | Liver/intestine | Phospholipids | Reverse cholesterol transport |

### Structure
- **Core:** Hydrophobic lipids (triglycerides, cholesteryl esters)
- **Surface:** Amphipathic phospholipids, free cholesterol, apolipoproteins

### Apolipoproteins

| Apo | Associated Lipoprotein | Primary Function |
|-----|------------------------|------------------|
| **ApoB-48** | Chylomicrons | Intestinal fat absorption; structural |
| **ApoB-100** | VLDL, IDL, LDL | Structural; LDL receptor binding |
| **ApoA-I** | HDL | Structural; initiates reverse cholesterol transport |
| **ApoE** | Chylomicron remnants, IDL | Hepatic uptake via receptors |
| **ApoC-II** | VLDL, chylomicrons | LPL activation for triglyceride hydrolysis |

### Metabolic Pathways

**Exogenous (Dietary):**
```
Dietary fat → Chylomicrons → LPL releases TG → Chylomicron remnants → Liver uptake (apoE)
```

**Endogenous (Hepatic):**
```
Liver → VLDL → LPL releases TG → IDL → Hepatic lipase → LDL → Peripheral cells (LDL receptor)
```

## Clinical Implication

**Lipid Panel Interpretation:**
- Elevated TG suggests increased VLDL (hepatic overproduction or delayed clearance)
- Low HDL impairs reverse cholesterol transport
- ApoB measurement reflects total atherogenic particle number (VLDL + IDL + LDL)

**Therapeutic Targets:**
- Fibrates activate PPAR-α → reduce VLDL production
- Omega-3 fatty acids reduce hepatic VLDL secretion
- Statins upregulate LDL receptors, enhancing LDL clearance

## METHAP Relevance

**Clinical Assessment:**
- Advanced lipoprotein analysis (NMR) reveals particle size and number
- Small dense LDL pattern indicates metabolic syndrome/insulin resistance
- Lipoprotein(a) genetic testing for residual risk assessment
