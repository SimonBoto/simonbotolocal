# Clinical Metabolic Markers HOMA IR HbA1c

## Overview
Standard clinical metabolic markers provide validated, accessible measures of glucose  and insulin resistance—serving as both diagnostic tools and outcome measures in the METHAP trial.

## Fasting Glucose

### Normal Values
| Category | Fasting Glucose |
|----------|-----------------|
| **Normal** | <100 mg/dL (<5.6 mmol/L) |
| **Prediabetes** | 100-125 mg/dL (5.6-6.9 mmol/L) |
| **Diabetes** | ≥126 mg/dL (≥7.0 mmol/L) |

### METHAP Eligibility
| Group | Glucose Range |
|-------|---------------|
| **Healthy** | <100 mg/dL |
| **Insulin-resistant** | 100-125 mg/dL (prediabetes) |
| **Excluded** | ≥126 mg/dL (diabetes) |

## HOMA-IR (Homeostatic Model Assessment for Insulin Resistance)

### Calculation
```
HOMA-IR = (Fasting insulin μU/mL × Fasting glucose mmol/L) ÷ 22.5
```

### Interpretation
| HOMA-IR | Status |
|---------|--------|
| **<2.0** | Insulin sensitive |
| **2.0-2.5** | Borderline |
| **≥2.5** | Insulin resistant |
| **>5.0** | Severe insulin resistance |

### METHAP Criteria
| Group | HOMA-IR Threshold |
|-------|-------------------|
| **Healthy** | <2.5 |
| **Insulin-resistant** | ≥2.5 |

### Advantages
| Feature | Benefit |
|---------|---------|
| **Simple** | Single fasting blood draw |
| **Validated** | Extensive research support |
| **Widely available** | Standard laboratory test |
| **Cost-effective** | Inexpensive |

### Limitations
| Issue | Impact |
|-------|--------|
| **Fasting required** | Patient inconvenience |
| **Insulin assay variability** | Standardization issues |
| **Doesn't reflect postprandial** | Misses some abnormalities |

## HbA1c (Glycated Hemoglobin)

### Normal Values
| Category | HbA1c |
|----------|-------|
| **Normal** | <5.7% |
| **Prediabetes** | 5.7-6.4% |
| **Diabetes** | ≥6.5% |

### METHAP Eligibility
| Group | HbA1c Range |
|-------|-------------|
| **Healthy** | <5.7% |
| **Insulin-resistant** | 5.7-6.4% |
| **Excluded** | ≥6.5% |

### Advantages
| Feature | Benefit |
|---------|---------|
| **No fasting** | Any time of day |
| **3-month average** | Less day-to-day variability |
| **Standardized** | NGSP/IFCC standardized |

### Limitations
| Condition | Effect on HbA1c |
|-----------|-----------------|
| **Anemia** | May be falsely low/high |
| **Hemoglobinopathies** | Interference |
| **Recent transfusion** | Inaccurate |
| **Pregnancy** | Different targets |

## Lipid Panel

### Components
| Marker | Target | IR Association |
|--------|--------|----------------|
| **Total cholesterol** | <200 mg/dL | Variable |
| **LDL cholesterol** | <100 mg/dL | Often normal in IR |
| **HDL cholesterol** | >40 (M), >50 (F) mg/dL | Low in IR |
| **Triglycerides** | <150 mg/dL | Elevated in IR |

### TG:HDL Ratio
| Ratio | Risk Category |
|-------|---------------|
| **<2** | Low risk |
| **2-3** | Moderate risk |
| **>3** | High insulin resistance risk |

### METHAP Lipid Targets
| Marker | Baseline Assessment | Outcome Measure |
|--------|---------------------|-----------------|
| **Triglycerides** | ✓ | Expected to decrease |
| **HDL** | ✓ | Expected to increase |
| **TG:HDL ratio** | ✓ | Expected to improve |

## METHAP Testing Protocol

### Baseline (All Participants)
| Test | Collection | Analysis |
|------|------------|----------|
| **Fasting glucose** | Fasting 8-12h | Glucose oxidase |
| **Fasting insulin** | Fasting 8-12h | Immunochemiluminometric |
| **HbA1c** | Any time | HPLC |
| **Lipid panel** | Fasting 12h | Enzymatic |

### Post-Intervention (IR Groups)
| Test | Timing | Expected Change |
|------|--------|-----------------|
| **All baseline tests** | Week 12 | Improvement |
| **HOMA-IR calculation** | Week 12 | >20% reduction |
| **HbA1c** | Week 12 | 0.3-0.5% reduction |

## Reference Ranges Summary

### Healthy vs Insulin-Resistant
| Marker | Healthy | Insulin-Resistant | Diabetic |
|--------|---------|-------------------|----------|
| **Fasting glucose** | <100 | 100-125 | ≥126 |
| **HbA1c** | <5.7% | 5.7-6.4% | ≥6.5% |
| **HOMA-IR** | <2.5 | ≥2.5 | Variable |
| **TG** | <100 | >150 | >150 |
| **HDL** | >40/50 | <40/50 | <40/50 |
