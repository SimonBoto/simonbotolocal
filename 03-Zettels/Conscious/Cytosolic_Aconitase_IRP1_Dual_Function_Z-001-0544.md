# Cytosolic Aconitase — IRP1 Dual Function

> Cytosolic aconitase (ACO1) functions both as a metabolic enzyme and as the iron regulatory protein IRP1, with its activity controlled by Fe-S cluster status.

## Two Isoforms

| Feature | l | Cytosolic |
|---------|--------------|-----------|
| **Gene** | ACO2 | ACO1 |
| **Location** | Matrix | Cytosol |
| **Role** |  | Iron sensing |
| **Cluster** | Stable | Labile |

## IRP1 Function

**Fe-S Cluster as Sensor:**

**Iron abundant:**
- [4Fe-4S] cluster present
- Aconitase active
- Normal 

**Iron low:**
- Cluster lost
- Becomes IRP1
- Binds IREs (iron-responsive elements)

## Iron Homeostasis Control

| Target | Iron High | Iron Low |
|--------|-----------|----------|
| **Ferritin** | Translation blocked | Translation allowed |
| **Transferrin receptor** | mRNA degraded | mRNA stabilized |

**Result:** Same protein, two functions based on iron status.
