#  Quality Standards

## Overview
Quality checkpoints throughout the metabolomics testing and interpretation process.

## Pre-Analytical Quality

### Sample Collection
| Parameter | Standard | Acceptable Range |
|-----------|----------|------------------|
| **Collection timing** | First morning void | Any time if noted |
| **Fasting** | 8-12 hours preferred | Document if non-fasting |
| **Storage** | Refrigerated or frozen | < 24h room temp |
| **Transport** | Frozen with cold pack | Track temperature |

### Common Pre-Analytical Errors
| Error | Impact | Prevention |
|-------|--------|------------|
| Delayed processing | Bacterial growth | Process within 48h |
| Room temperature storage | Marker degradation | Refrigerate immediately |
| Inadequate volume | Insufficient sample | Collection instructions |

## Analytical Quality

### QC Metrics
| Metric | Target | Action if Outside |
|--------|--------|-------------------|
| **Internal standard recovery** | 80-120% | Re-extract sample |
| **Calibration curve R²** | > 0.99 | Recalibrate |
| **Blank contamination** | < limit of detection | Clean system |

### Proficiency Testing
- **Participate** in external QC programs
- **Compare** results to peer laboratories
- **Document** deviations and corrective actions

## Interpretation Quality

### Review Process
| Step | Responsible | Action |
|------|-------------|--------|
| **Initial interpretation** | Clinician | Pattern recognition |
| **Second review** | Senior clinician | Verify conclusions |
| **Report generation** | Staff | Standardized format |
| **Final sign-off** | Attending |  approval |

### Interpretation Standards
- **Reference ranges** — Age and sex specific
- **Pattern-based** — Not isolated markers
- **Evidence-linked** — Citations for claims
- **Actionable** — Specific recommendations

## Post-Analytical Quality

### Report Standards
| Element | Requirement |
|---------|-------------|
| **Patient identification** | Two identifiers |
| **Test date** | Collection and report dates |
| **Reference ranges** | Lab-specific |
| **Abnormal flags** | Clear highlighting |
| **Interpretation** | Clinical context |
| **Recommendations** | Evidence-based |

### Turnaround Times
| Test | Target TAT | Maximum TAT |
|------|------------|-------------|
| **OAT** | 5 business days | 7 days |
| **** | 5 business days | 7 days |
| **Combined report** | 7 business days | 10 days |

## Continuous Quality Improvement

### Metrics Tracked
- Pre-analytical error rate
- Analytical QC failure rate
- Report turnaround time
- Clinician interpretation agreement
- Patient satisfaction scores

### Target Benchmarks
| Metric | Target |
|--------|--------|
| Pre-analytical error rate | < 2% |
| QC failure rate | < 5% |
| TAT compliance | > 95% |
| Patient satisfaction | > 90% |
