# Metabolomics Quality Standards

Pre-analytical variables, sample integrity, interpretation consistency, and reporting accuracy. Quality standards ensure reliable, reproducible results that drive clinical decisions.

## Evidence
> "Quality in metabolomics isn't just lab accuracy—it's the entire chain from patient preparation to clinical interpretation." (Laboratory medicine)

## Pre-Analytical Quality

### Patient Preparation
| Variable | Standard | Rationale |
|----------|----------|-----------|
| **Fasting** | 8-12 hours overnight | Avoids dietary confounders |
| **First morning urine** | Upon waking | Most concentrated, consistent |
| **Hydration** | Normal, not excessive | Avoids dilution |
| **Exercise** | No strenuous activity 24h prior | Avoids metabolic stress markers |
| **Medications** | Document all | Identifies confounders |

### Collection Protocol
1. **Clean catch midstream** — Reduces contamination
2. **First morning void** — Standardizes timing
3. **Fill to line** — Ensures adequate volume
4. **Refrigerate immediately** — Preserves stability
5. **Ship within 24 hours** — Maintains integrity

### Common Pre-Analytical Errors
| Error | Impact | Prevention |
|-------|--------|------------|
| **Not first morning** | Variable dilution | Clear instructions |
| **Incomplete collection** | Insufficient volume | Visual fill line |
| **Delayed shipping** | Marker degradation | 24-hour shipping |
| **Contamination** | False elevations | Clean catch technique |

## Analytical Quality

### Laboratory Standards
| Parameter | Requirement |
|-----------|-------------|
| **CLIA certification** | Required |
| **Proficiency testing** | Quarterly participation |
| **Internal QC** | Daily with each batch |
| **Calibration** | Per manufacturer schedule |

### QC Acceptance Criteria
| Control Level | Acceptable Range |
|---------------|------------------|
| **Low** | ± 20% of target |
| **Medium** | ± 15% of target |
| **High** | ± 15% of target |

### QC Failure Response
1. **Stop processing** — No patient reports
2. **Investigate cause** — Reagent, instrument, technique
3. **Correct and rerun** — QC must pass before resuming
4. **Document** — QC log and corrective action

## Post-Analytical Quality

### Interpretation Standards
| Element | Standard |
|---------|----------|
| **Reference ranges** | Age/sex-matched, validated |
| **Pattern recognition** | Multiple markers, not isolated |
| **Clinical correlation** | Symptoms match markers |
| **Documentation** | Interpretation rationale recorded |

### Reporting Standards
| Requirement | Implementation |
|-------------|----------------|
| **Patient-friendly language** | No jargon without explanation |
| **Visual hierarchy** | Traffic light system |
| **Actionable recommendations** | Specific, not generic |
| **Follow-up plan** | Clear timeline |

## Interpretation Consistency

### Standardized Approach
1. **Review by category** — Systematic, not random
2. **Identify patterns** — Not isolated elevations
3. **Severity assessment** — Critical vs. optimization
4. **Clinical correlation** — Do markers match symptoms?
5. **Treatment mapping** — Specific markers → specific actions

### Inter-Rater Reliability
| Scenario | Action |
|----------|--------|
| **New clinician** | Supervised interpretation until competency |
| **Complex cases** | Second opinion for patterns |
| **Discrepancies** | Discussion and consensus |

## Quality Metrics & Monitoring

### Key Performance Indicators
| Metric | Target | Frequency |
|--------|--------|-----------|
| **Pre-analytical error rate** | < 2% | Monthly |
| **QC failure rate** | < 5% | Per batch |
| **Report turnaround** | < 5 days | Weekly |
| **Interpretation accuracy** | > 95% | Quarterly audit |
| **Patient satisfaction** | > 90% | Per visit |

### Quality Improvement Process
1. **Monitor metrics** — Track KPIs
2. **Identify trends** — Statistical process control
3. **Root cause analysis** — When targets missed
4. **Implement corrective action** — Systematic changes
5. **Re-monitor** — Verify improvement

## Documentation Requirements

### Chain of Custody
| Step | Documentation |
|------|---------------|
| **Collection** | Date, time, patient ID |
| **Receipt** | Condition, temperature |
| **Processing** | Technician, batch, QC |
| **Reporting** | Interpreter, date |

### Record Retention
| Record Type | Retention Period |
|-------------|------------------|
| **Raw data** | 7 years |
| **Reports** | 7 years |
| **QC logs** | 2 years |
| **Corrective actions** | Permanent |

## Red Flags: When to Pause

| Situation | Action |
|-----------|--------|
| **Multiple QC failures** | Stop, investigate, notify lab director |
| **Unusual pattern of results** | Verify instrument, reagents |
| **Patient complaint about accuracy** | Full case review |
| **Proficiency testing failure** | Immediate corrective action |
