# Purine Salvage Pathway Recycling 

## Overview
The purine salvage pathway recovers free purine bases from nucleotide degradation, converting them back into nucleotides for reuse—an energy-efficient alternative to de novo synthesis.

## Why Salvage?

### Energy Economics
| Pathway | ATP Cost per Purine | Efficiency |
|---------|---------------------|------------|
| **De novo synthesis** | ~6 ATP | Energy expensive |
| **Salvage pathway** | ~1-2 ATP | Energy efficient |

**Net savings:** 4-5 ATP per nucleotide recycled

### Cellular Context
- **Brain:** Highly dependent on salvage (limited de novo capacity)
- **Bone marrow:** Rapid cell division, high nucleotide demand
- **Erythrocytes:** No de novo synthesis, rely entirely on salvage

## The Salvage Reactions

### Hypoxanthine-Guanine Salvage
```
Hypoxanthine + PRPP → IMP + PPi
        (HGPRT enzyme)

Guanine + PRPP → GMP + PPi
        (HGPRT enzyme)
```

### Adenine Salvage
```
Adenine + PRPP → AMP + PPi
        (APRT enzyme)
```

### Key Enzymes
| Enzyme | Substrates | Product | Tissue Distribution |
|--------|------------|---------|---------------------|
| **HGPRT** | Hypoxanthine, Guanine | IMP, GMP | Ubiquitous (high in brain) |
| **APRT** | Adenine | AMP | Ubiquitous |

### PRPP: The Key Substrate
- **PRPP** (phosphoribosyl pyrophosphate): Activated ribose-5-phosphate
- Provides ribose-phosphate moiety for nucleotide synthesis
- Synthesized from ribose-5-phosphate (from PPP) + ATP

## Degradation vs. Salvage Balance

### Purine Catabolism (Degradation Path)
```
Nucleotides → Nucleosides → Bases → Uric acid

GMP → Guanosine → Guanine → Xanthine → Uric acid
IMP → Inosine → Hypoxanthine → Xanthine → Uric acid
AMP → Adenosine → Inosine → Hypoxanthine → Xanthine → Uric acid
```

### Salvage Branch Point
- Hypoxanthine and guanine can be salvaged OR degraded
- Adenine primarily salvaged (less efficient degradation)
- **Xanthine:** Point of no return (only degrades to uric acid)

## Clinical Disorders

### Lesch-Nyhan Syndrome (HGPRT Deficiency)
| Feature | Description |
|---------|-------------|
| **Enzyme** | HGPRT severely deficient |
| **Metabolic** | Purine overproduction, hyperuricemia |
| **Neurological** | Intellectual disability, dystonia |
| **Behavioral** | Self-injurious behavior (compulsive) |
| **Treatment** | Allopurinol (reduces uric acid), supportive |

**Mechanism:** Without salvage, purines shunted to degradation → excessive uric acid

### Adenine Phosphoribosyltransferase Deficiency
| Feature | Description |
|---------|-------------|
| **Enzyme** | APRT deficient |
| **Metabolic** | Adenine shunted to degradation |
| **Consequence** | 2,8-dihydroxyadenine stones |
| **Renal** | Chronic kidney disease |
| **Treatment** | Allopurinol, low-purine diet, hydration |

### Purine Nucleoside Phosphorylase Deficiency
- T-cell immunodeficiency
- Toxic metabolites in T-cells
- Normal B-cell function (initially)

## Evolutionary Perspective
- Salvage pathways likely evolved early
- Conserved across species
- Reflects evolutionary pressure for metabolic efficiency
- Brain dependence suggests critical function

## Therapeutic Targeting

### Salvage Inhibitors in 
- Some chemotherapies target salvage enzymes
- Tumor cells often upregulate salvage pathways
- Combination with de novo inhibitors effective

### Gout Treatment
- Allopurinol inhibits xanthine oxidase
- Prevents conversion of hypoxanthine → xanthine → uric acid
- Increases salvage substrates (hypoxanthine, xanthine)
