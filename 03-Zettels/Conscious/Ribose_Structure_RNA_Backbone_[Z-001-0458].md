# Ribose Structure and Biological Roles

## Structure
- **Formula:** C₅H₁₀O₅ (pentose)
- **Ring form:** Furanose (5-membered: 4 carbons + 1 oxygen)
- **Hydroxyl at 2' position:** Critical for RNA 

## D-Ribose vs D-Deoxyribose
| Feature | Ribose (RNA) | Deoxyribose (DNA) |
|---------|--------------|-------------------|
| 2' carbon | -OH | -H |
| Stability | Less stable | More stable |
| Function | Catalytic flexibility | Long-term storage |

## Biological Roles

### 1. RNA Backbone
- Nucleotides: Base + ribose + phosphate
- Phosphodiester bonds link ribose 3'-OH to 5'-phosphate

### 2. ATP Structure
- Adenine + ribose + 3 phosphates
- High-energy phosphoanhydride bonds

### 3. Coenzymes
All contain ribose:
- NAD⁺/NADP⁺ (redox)
- FAD (electron transfer)
- CoA (acyl transfer)

### 4. Second Messengers
- cAMP, cGMP (signaling)

## Biosynthesis
Primarily via **Pentose Phosphate Pathway** (not directly from glucose).
