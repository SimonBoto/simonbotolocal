# TP53 Abnormalities Multiple Myeloma del17p

## Overview
TP53 abnormalities, particularly deletion of chromosome 17p (del(17p)) containing the TP53 gene, represent the highest-risk cytogenetic feature in multiple myeloma, associated with poor prognosis and treatment resistance.

## Types of TP53 Abnormalities

### Deletion 17p (del(17p))
| Feature | Description |
|---------|-------------|
| **Frequency** | 5-10% newly diagnosed; 20-50% relapsed |
| **** | Loss of one TP53 allele |
| **Detection** | FISH (fluorescence in situ hybridization) |
| **Threshold** | ≥10% cells with del(17p) ly significant |

### TP53 Mutations
| Feature | Description |
|---------|-------------|
| **Types** | Missense, nonsense, frameshift |
| **Location** | DNA-binding domain most common |
| **Effect** | Loss of function or dominant-negative |
| **Coincidence** | Often with del(17p) = biallelic inactivation |

### Biallelic Inactivation
| Definition | Both TP53 alleles abnormal |
|------------|---------------------------|
| **Mechanism** | Del(17p) + mutation of remaining allele |
| **Consequence** | Complete p53 loss |
| **Prognosis** | Worst outcome |

## Prognostic Impact

### Survival Data
| Status | Median PFS | Median OS |
|--------|------------|-----------|
| **No del(17p)** | 30-40 months | 60-80 months |
| **Del(17p)** | 12-15 months | 24-36 months |
| **Biallelic loss** | <12 months | <24 months |

### Response to Treatment
| Therapy | Response in del(17p) |
|---------|---------------------|
| **Conventional chemo** | Poor |
| **IMiDs (lenalidomide)** | Reduced efficacy |
| **Bortezomib** | Modestly better than IMiDs alone |
| **ASCT** | High relapse rate |
| **CAR-T** | Variable, often short duration |

## Mechanisms of Resistance

### Genomic Instability
| Effect | Consequence |
|--------|-------------|
| **DNA repair impaired** | Mutations accumulate |
| **Chromosomal instability** | Clonal evolution |
| **Therapeutic resistance** | Rapid adaptation |

### Apoptosis Evasion
| Pathway | Impact |
|---------|--------|
| **p53-mediated apoptosis** | Lost |
| **Chemotherapy-induced death** | Resistant |
| **Radiation response** | Impaired |

## Detection and Monitoring

### FISH Testing
| Probe | Target |
|-------|--------|
| **TP53** | 17p13.1 locus |
| **Reference probe** | Centromere 17 or other chromosome |
| **Interpretation** | Loss of TP53 signal |

### Next-Generation Sequencing
| Application | Benefit |
|-------------|---------|
| **Mutation detection** | Identifies TP53 mutations |
| **Clonal tracking** | Monitor evolution |
| **Co-occurring alterations** | Comprehensive profiling |

### When to Test
| Timing | Rationale |
|--------|-----------|
| **Diagnosis** | Risk stratification |
| **Relapse** | Acquired abnormalities |
| **Clinical trial** | Eligibility |
