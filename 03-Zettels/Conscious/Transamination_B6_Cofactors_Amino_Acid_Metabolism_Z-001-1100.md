## Core Claim
**Transamination transfers amino groups between amino acids and alpha-keto acids using vitamin B6 (pyridoxal phosphate, PLP) as essential cofactor—this reversible reaction enables nitrogen conservation while directing carbon skeletons to energy pathways** — B6 deficiency impairs all amino acid metabolism.

## Evidence

### Transamination Reaction
```
Amino Acid₁ + Alpha-ketoglutarate ↔ Alpha-keto acid₁ + Glutamate
```
- **Enzyme:** Aminotransferase (transaminase)
- **Cofactor:** Pyridoxal phosphate (PLP — vitamin B6)
- **Mechanism:** PLP forms Schiff base, acts as electron sink

### PLP Function
| Property | Role |
|----------|------|
| **Aldehyde group** | Forms Schiff base with amino acid |
| **Electron sink** | Stabilizes carbanion intermediates |
| **Versatility** | Catalyzes transamination, decarboxylation, deamination |

### B6-Dependent Reactions
| Reaction Type | Examples |
|---------------|----------|
| **Transamination** | ALT, AST (liver enzymes) |
| **Decarboxylation** | Neurotransmitter synthesis (GABA, serotonin, dopamine) |
| **Deamination** | Serine, threonine catabolism |
| **Racemization** | D-amino acid synthesis |

### Complete Cofactor Requirements for AA Catabolism
| Cofactor | Function |
|----------|----------|
| **B6 (PLP)** | Transaminases, decarboxylases |
| **B12** | Methylmalonyl-CoA mutase (branched-chain AAs) |
| **B2** | Oxidative deaminases |
| **Iron** | Phenylalanine hydroxylase, tyrosine hydroxylase |

## Clinical Implication
- **B6 deficiency** → impaired transamination → elevated homocysteine, neurological symptoms
- **ALT/AST** (liver enzymes) are B6-dependent — low B6 can falsely normalize values
- **Neurotransmitter synthesis** requires B6 — deficiency linked to depression, cognitive impairment
- **METHAP relevance:** High-dose B6 (50-100mg) supports transamination, neurotransmitter synthesis, and glycogen metabolism
