## Core Claim
**Glutamate provides the first nitrogen atom for the urea cycle via N-acetylglutamate (NAG) synthesis — NAG activates carbamoyl phosphate synthetase I (CPS I), the rate-limiting enzyme of ureagenesis** — arginine synthesis from glutamate connects amino acid metabolism to nitrogen disposal, with aspartate providing the second nitrogen atom.

## Evidence

### Urea Cycle Overview
**Purpose:** Convert toxic ammonia to non-toxic urea for excretion

### First Nitrogen: Glutamate → NAG
```
Glutamate + NH₄⁺ + Acetyl-CoA → N-acetylglutamate (NAG) + CoA
```
| Feature | Detail |
|---------|--------|
| **Enzyme** | N-acetylglutamate synthase (NAGS) |
| **Activator** | Arginine (positive feedback) |
| **Function** | NAG activates CPS I |

### CPS I Activation
```
NAG + CPS I → Active CPS I
NH₄⁺ + HCO₃⁻ + 2 ATP → Carbamoyl phosphate
```

### Arginine Synthesis from Glutamate
```
Glutamate → NAG → Ornithine → Citrulline → Argininosuccinate → Arginine
```

### Second Nitrogen: Aspartate
```
Citrulline + Aspartate → Argininosuccinate (argininosuccinate synthetase)
```

### Arginine Cleavage → Urea
```
Arginine → Ornithine + Urea (arginase)
```

## Clinical Implication
- **NAGS deficiency:** Hyperammonemia, neonatal encephalopathy
- **Urea cycle defects:** Life-threatening hyperammonemia
- **Arginine supplementation:** Treatment for some urea cycle disorders
- **METHAP relevance:** Glutamate and aspartate are essential for nitrogen disposal; metabolic health supports ureagenesis
