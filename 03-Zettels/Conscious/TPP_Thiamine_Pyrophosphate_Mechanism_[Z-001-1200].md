# TPP — Thiamine Pyrophosphate  and Enzymes

## Overview

Thiamine pyrophosphate (TPP) is the active form of vitamin B1 (thiamine). It serves as an essential cofactor for several critical decarboxylation reactions in energy .

---

## TPP Structure

**Components:**
- Thiazole ring (contains reactive carbon)
- Pyrimidine ring
- Pyrophosphate group (for enzyme binding)

**Reactive Site:**
- Carbon 2 of thiazole ring
- Forms carbanion (negative charge on carbon)
- Nucleophilic attack on carbonyl carbons

---

## Mechanism of Action

### Step 1: Carbanion Formation
```
Thiazole C2-H → Thiazole C2⁻ (carbanion)
```
- Stabilized by adjacent nitrogen
- Resonance with thiazole ring

### Step 2: Nucleophilic Attack
```
Carbanion attacks carbonyl carbon of substrate
Forms covalent intermediate
```

### Step 3: Decarboxylation
```
Carboxyl group leaves as CO₂
Remaining electrons stabilized by TPP
```

### Step 4: Transfer to Acceptor
```
Hydroxyethyl-TPP transfers to lipoamide or acceptor
Regenerates TPP
```

---

## TPP-Dependent Enzymes

### 1. Pyruvate Dehydrogenase Complex
**Reaction:**
```
Pyruvate + CoA + NAD⁺ → Acetyl-CoA + CO₂ + NADH
```

**Components:**
- **E1 (pyruvate dehydrogenase):** Contains TPP
- **E2 (dihydrolipoyl transacetylase):** Contains lipoic acid
- **E3 (dihydrolipoyl dehydrogenase):** Contains FAD

**Function:** Links glycolysis to Krebs cycle

### 2. α-Ketoglutarate Dehydrogenase Complex
**Reaction:**
```
α-Ketoglutarate + CoA + NAD⁺ → Succinyl-CoA + CO₂ + NADH
```

**Structure:** Similar to pyruvate dehydrogenase
- E1 contains TPP
- Same E2/E3 architecture

**Function:** Critical Krebs cycle step

### 3. Transketolase (Pentose Phosphate Pathway)
**Reaction:**
```
Transfers 2-carbon unit between sugars
```

**Function:**
- Connects PPP to glycolysis
- Generates ribose for nucleotide synthesis

### 4. Branched-Chain α-Ketoacid Dehydrogenase (BCKDH)
**Reaction:**
```
Degrades branched-chain amino acid derivatives
```

**Clinical:** Maple syrup urine disease (deficiency)

## Thiamine Deficiency — Beriberi

### Causes
- Mal (polished rice diet)
- Alcoholism (malnutrition + impaired absorption)
- Dialysis (water-soluble vitamin loss)
- Bariatric surgery

### Clinical Features

**Wet Beriberi (Cardiovascular):**
- High-output cardiac failure
- Edema
- Tachycardia

**Dry Beriberi (Neurological):**
- Peripheral neuropathy
- Muscle weakness
- Wernicke-Korsakoff syndrome (alcoholics)

**Biochemical Markers:**
- Elevated blood pyruvate/lactate
- Reduced transketolase activity
- Clinical response to thiamine

### Wernicke-Korsakoff Syndrome
**Triad:**
- Confusion
- Ataxia
- Ophthalmoplegia

**Mechanism:** Thiamine deficiency + alcohol neurotoxicity

---

## Laboratory Assessment

### Functional Tests
**Transketolase Activation Test:**
- Measure erythrocyte transketolase activity
- Add TPP in vitro
- >25% increase indicates deficiency

### Direct Measurement
- Serum thiamine (less reliable)
- Erythrocyte thiamine pyrophosphate
- Urinary thiamine excretion

---

## Therapeutic Uses

### Deficiency Treatment
- Thiamine 100 mg IV/IM (Wernicke's)
- Oral supplementation for mild cases
- Prophylaxis in alcoholics, bariatric patients

### Metabolic Support
- High-dose thiamine in sepsis (energetic crisis)
- Thiamine-responsive megaloblastic anemia
