# Critical Appraisal Framework

> **Core Claim:** Systematic evaluation of scope, design, methodology, results, and ethics ensures comprehensive manuscript assessment.

## Initial Assessment Checklist

- [ ] **Scope/Appropriateness** — Fits journal aims?
- [ ] **Study Design** — Adequate for question?
- [ ] **Methodology** — Sufficient detail for replication?
- [ ] **Results-Conclusions** — Alignment verified?
- [ ] **Ethical Compliance** — IRB, consent, conflicts disclosed?

## Red Flags

| Category | Warning Signs |
|----------|---------------|
| **Methodological** | Inadequate controls, cherry-picked data |
| **Statistical** | Inappropriate tests, overstated significance |
| **Interpretation** | Causal claims from observational data |
| **Ethical** | Undisclosed conflicts of interest |

## Systematic Approach

```
1. Read abstract → form initial impression
2. Examine methods → assess rigor
3. Review results → verify claims
4. Check discussion → evaluate interpretation
5. Note concerns → categorize severity
```
