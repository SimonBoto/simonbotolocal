# Glucogenic Amino Acids Gluconeogenesis Support

## Overview
Glucogenic amino acids can be converted to glucose via gluconeogenesis—critical for maintaining blood glucose during fasting, starvation, and carbohydrate restriction.

## The 18 Glucogenic Amino Acids

### By Gluconeogenic Precursor

#### Pyruvate Precursors
| Amino Acid | Conversion Enzyme |
|------------|-------------------|
| **Alanine** | Alanine transaminase |
| **Serine** | Serine dehydratase |
| **Glycine** | Glycine cleavage system |
| **Cysteine** | Cysteine dioxygenase |

#### Oxaloacetate Precursors
| Amino Acid | Pathway |
|------------|---------|
| **Aspartate** | Direct transamination |
| **Asparagine** | Asparaginase → aspartate |

#### Alpha-ketoglutarate Precursors
| Amino Acid | Conversion |
|------------|------------|
| **Glutamate** | Glutamate dehydrogenase |
| **Glutamine** | Glutaminase |
| **Proline** | Proline oxidase → glutamate |
| **Arginine** | Arginase → ornithine → glutamate |
| **Histidine** | Histidase → glutamate |

#### Succinyl-CoA Precursors
| Amino Acid | Pathway |
|------------|---------|
| **Valine** | Propionyl-CoA → methylmalonyl-CoA |
| **Methionine** | Propionyl-CoA pathway |
| **Isoleucine** | Propionyl-CoA + acetyl-CoA |
| **Threonine** | Propionyl-CoA or glycine |

#### Fumarate Precursors
| Amino Acid | Pathway |
|------------|---------|
| **Phenylalanine** | Via tyrosine |
| **Tyrosine** | Fumarate + acetoacetate |

## Gluconeogenesis from Protein

### During Fasting
| Time | Source | Glucose Production |
|------|--------|-------------------|
| **12-24 hours** | Glycogen depletion begins | Liver starts gluconeogenesis |
| **24-48 hours** | Muscle protein catabolism | Alanine major substrate |
| **48+ hours** | Continued proteolysis | All glucogenic AAs used |
| **Prolonged** | Skeletal muscle loss | Conservation s activate |

### The Alanine Cycle
```
Muscle: Protein → Alanine
              ↓
Liver: Alanine → Pyruvate → Glucose
              ↓
Blood: Glucose → Muscle
              ↓
Muscle: Glycolysis → Pyruvate → Alanine
```

**Purpose:** Transport nitrogen from muscle to liver (as alanine) while generating glucose

## Clinical Context

### Starvation Adaptation
- **Early:** Glycogenolysis (hours)
- **Intermediate:** Gluconeogenesis from alanine (days)
- **Late:** Ketones replace glucose for brain; protein sparing

### High-Protein Diets
- Excess glucogenic amino acids → glucose
- May reduce ketosis despite low carb intake
- Explains variability in ketogenic diet responses

### Medical Applications
| Condition | Use of Glucogenic AAs |
|-----------|----------------------|
| **Hypoglycemia** | Rapid glucose generation |
| **Critical illness** | Maintain euglycemia |
| **Burns/trauma** | Increased gluconeogenesis |
| **Hepatic failure** | Impaired—causes hypoglycemia |

## Hormonal Regulation

| Hormone | Effect on Gluconeogenesis |
|---------|---------------------------|
| **Glucagon** | Stimulates (fasting) |
| **Cortisol** | Stimulates (stress, prolonged fasting) |
| **** | Inhibits (fed state) |
| **Epinephrine** | Stimulates (acute stress) |
