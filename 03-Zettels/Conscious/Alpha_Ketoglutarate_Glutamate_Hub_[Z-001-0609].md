# α-Ketoglutarate Entry: The Glutamate Hub

## Four Amino Acids Converge Here

**α-Ketoglutarate** is the most connected Krebs cycle entry point for amino acids — four amino acids feed into it through glutamate.

## The Pathways

### 1. Glutamate → Direct Transamination
```
Glutamate + NAD⁺ + H₂O → α-Ketoglutarate + NADH + NH₄⁺
```
- **Enzyme:** Glutamate dehydrogenase
- **Location:** Mitochondrial matrix
- **Direct entry** — no intermediate steps

### 2. Proline → Glutamate → α-Ketoglutarate
```
Proline → Δ¹-pyrroline-5-carboxylate (P5C) → Glutamate-γ-semialdehyde → Glutamate → α-Ketoglutarate
```
- **Key enzyme:** Proline dehydrogenase (PRODH)
- Also produces **hydroxyproline** from collagen breakdown

### 3. Arginine → Glutamate → α-Ketoglutarate
```
Arginine → Glutamate-γ-semialdehyde (via ornithine) → Glutamate → α-Ketoglutarate
```
- **Urea cycle connection:** Arginine catabolism intersects with urea synthesis
- **Shared intermediate:** Glutamate-γ-semialdehyde (with proline)

### 4. Histidine → Glutamate → α-Ketoglutarate
```
Histidine → Urocanate → Formiminoglutamate (FIGLU) → Glutamate → α-Ketoglutarate
```
- **FIGLU test:** Elevated formiminoglutamate indicates folate/B12 deficiency
- **Tetrahydrofolate (THF) required** for histidine → glutamate conversion

## Why This Hub Matters

### Metabolic Flexibility
- **Nitrogen handling:** Glutamate is the primary nitrogen carrier
- **Redox balance:** Glutamate dehydrogenase generates NADH
- **Carbon skeleton:** α-Ketoglutarate feeds directly into Krebs cycle

### Clinical Connections
- **Glutaric aciduria type I:** Defect in glutaryl-CoA dehydrogenase (lysine/tryptophan/ hydroxylysine metabolism)
- **Hyperprolinemia:** PRODH deficiency, neurotoxic effects
- **FIGLU excretion:** Folate deficiency marker

### Research Applications
- **Insulin secretion:** α-Ketoglutarate signals in β-cells
- **Hypoxia response:** Prolyl hydroxylases use α-ketoglutarate
- **Epigenetics:** α-Ketoglutarate is a cofactor for TET enzymes (DNA demethylation)
