# Aconitase Oxidative Stress Sensitivity

## Overview
Aconitase is the most sensitive TCA cycle enzyme to oxidative stress due to its iron-sulfur (Fe-S) cluster, making it a key marker of mitochondrial damage.

## Enzyme Function
- **Reaction:** Citrate ↔ Isocitrate (via cis-aconitate intermediate)
- **Location:** Mitochondrial matrix
- **Kinetics:** Near-equilibrium enzyme
- **Control point:** Early TCA cycle commitment

## Iron-Sulfur Cluster
- **Structure:** [4Fe-4S]²⁺ cluster
- **Function:** Catalytic (not structural)
- **Vulnerability:** Superoxide (O₂⁻) attacks Fe-S cluster

## Oxidative Inactivation
### Mechanism
1. Superoxide attacks [4Fe-4S]²⁺ cluster
2. Cluster releases Fe²⁺
3. Enzyme converts to [3Fe-4S]⁺ (inactive form)
4. Citrate accumulates; isocitrate production ↓

### Reversibility
- **Partially reversible** with antioxidant restoration
- **Chronic damage:** Requires cluster reassembly

## OAT Signature
| Pattern | Interpretation |
|---------|----------------|
| **Citrate ↑** | Aconitase inhibition |
| **Isocitrate ↓** | Downstream block |
| **Citrate/Isocitrate ratio >10** | Significant oxidative stress |

## Clinical Triggers of Aconitase Block
- **Reactive oxygen species** (mitochondrial dysfunction)
- **Heavy metals** (mercury, arsenic, cadmium)
- **Nitric oxide** (competes with Fe-S)
- **Hypoxia** (reductive stress)

## Mitochondrial Aconitase vs Cytosolic
| Feature | Mitochondrial | Cytosolic |
|---------|---------------|-----------|
| **Gene** | ACO2 | ACO1 |
| **Primary role** | TCA cycle | Iron homeostasis (IRP1) |
| **Sensitivity** | High oxidative sensitivity | Moderate |
| **OAT marker** | Citrate accumulation | Not directly measured |

## Therapeutic Implications
| Intervention | Rationale |
|--------------|-----------|
| **Antioxidants** | Protect Fe-S cluster (glutathione, ALA, NAC) |
| **Iron management** | Avoid excess iron; ensure adequate but not excessive |
| **Heavy metal chelation** | Remove competitive inhibitors |
| **Mitochondrial support** | CoQ10, B-vitamins for ETC efficiency |

## Clinical Pearl
> "Elevated citrate with normal/low isocitrate is an early warning of mitochondrial oxidative stress—often before other markers appear."
