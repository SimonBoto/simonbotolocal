# Z_MECH: Akt Substrate Phosphorylation

Fully activated Akt phosphorylates multiple substrates to execute insulin's metabolic effects.

**Activation requirement:**
- PDK1 phosphorylates Thr308 (kinase domain)
- mTORC2 phosphorylates Ser473 (hydrophobic motif)
- Both required for full activity

**Key metabolic substrates:**

| Substrate | Function | Akt Effect |
|-----------|----------|------------|
| **AS160/TBC1D4** | GAP for Rab proteins | Inhibition → GLUT4 translocation |
| **GSK3β** | Inhibits glycogen synthase | Inhibition → glycogen synthesis |
| **FOXO1** | Transcription factor | Exclusion from nucleus → ↓ gluconeogenesis |
| **TSC2** | Rheb GAP | Inhibition → mTORC1 activation |

**Consensus motif:** Akt prefers RxRxxS/T phosphorylation sites.
