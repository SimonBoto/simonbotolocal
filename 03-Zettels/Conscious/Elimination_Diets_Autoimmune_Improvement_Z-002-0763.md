## Core Claim
**Elimination diets, particularly the Autoimmune Protocol (AIP), produce significant symptom improvement and disease modification in autoimmune conditions** — including IBD, Hashimoto's, rheumatoid arthritis, and psoriasis — by removing dietary triggers and supporting gut healing.

## The AIP Protocol

### Elimination Phase (30-90 days)
| Remove | Why |
|--------|-----|
| **Grains** | Gluten, lectins |
| **Dairy** | Casein, lactose |
| **Legumes** | Lectins |
| **Nightshades** | Glycoalkaloids |
| **Eggs** | Common sensitivity |
| **Nuts/seeds** | Lectins |
| **Processed foods** | Additives, inflammation |
| **Seed oils** | Omega-6 excess |
| **Refined sugar** | Inflammation |
| **Alcohol** | Gut permeability |
| **Coffee** | Cross-reactivity for some |

### What You CAN Eat
| Category | Foods |
|----------|-------|
| **Meat** | Grass-fed, pasture-raised |
| **Fish** | Wild-caught |
| **Vegetables** | All except nightshades |
| **Fruit** | Moderate amounts |
| **Healthy fats** | Olive oil, coconut oil, avocado |
| **Fermented foods** | Probiotics |
| **Bone broth** | Gut healing |

### Reintroduction Phase
| Step | Timeline |
|------|----------|
| **One food at a time** | Every 3-7 days |
| **Small amount first** | Test tolerance |
| **Monitor 72 hours** | Delayed reactions |
| **Keep a food diary** | Track symptoms |
