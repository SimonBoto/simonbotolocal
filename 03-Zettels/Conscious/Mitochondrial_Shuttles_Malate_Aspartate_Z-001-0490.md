# l Shuttles: Malate and Aspartate

## The Problem
**Oxaloacetate cannot cross the inner  membrane.**

Since OAA is made in mitochondria (by pyruvate carboxylase) but PEPCK is cytosolic, we need a **shuttle system**.

## Malate Shuttle (Malate-Aspartate Shuttle)

### 
**Mitochondrial side:**
```
OAA + NADH + H⁺ → Malate + NAD⁺ (malate dehydrogenase)
```

**Transport:**
- Malate transported to cytosol (dicarboxylate carrier)

**Cytosolic side:**
```
Malate + NAD⁺ → OAA + NADH + H⁺ (malate dehydrogenase)
```

### Key Feature
**Transfers reducing equivalents:**
- NADH from mitochondria → cytosol
- Needed for gluconeogenesis (glyceraldehyde-3-P dehydrogenase)

### Balance
- 1 NADH per OAA transported
- Maintains redox balance

## Aspartate Shuttle

### Mechanism
**Mitochondrial side:**
```
OAA + Glutamate → Aspartate + α-ketoglutarate
(aspartate aminotransferase)
```

**Transport:**
- Aspartate exported (glutamate imported)

**Cytosolic side:**
```
Aspartate + α-ketoglutarate → OAA + Glutamate
(aspartate aminotransferase)
```

### Key Feature
**No net redox transfer:**
- Just carbon skeleton (OAA) moved
- No NADH transfer
- α-ketoglutarate/glutamate cycle

### When Used?
- When NADH not needed in cytosol
- Alternative to malate shuttle

## Comparison

| Feature | Malate Shuttle | Aspartate Shuttle |
|---------|---------------|-------------------|
| **Redox transfer** | Yes (NADH) | No |
| **Substrates** | Malate/OAA | Aspartate/Glu |
| **Enzymes** | MDH | AST |
| **Energy cost** | None | None |
| **Primary use** | Gluconeogenesis | Urea cycle, protein synthesis |

## Clinical Relevance

### Shuttle Deficiencies
- Rare but cause metabolic disorders
- Impaired gluconeogenesis
- Elevated lactate

### Metformin Effect
- Inhibits mitochondrial glycerophosphate shuttle
- Increases reliance on malate shuttle
- Contributes to anti-gluconeogenic effect
