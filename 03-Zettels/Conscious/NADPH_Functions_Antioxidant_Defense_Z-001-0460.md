# NADPH: The Reducing Power

## Core Function
NADPH provides **reducing equivalents** for:
1. Biosynthesis (anabolism)
2. Antioxidant defense
3. Immune function

## Sources
- **~60%:** Pentose Phosphate Pathway
- **Remainder:** Malic enzyme, isocitrate dehydrogenase,  

## Role 1: Biosynthesis
**NADPH-dependent pathways:**
- Fatty acid synthesis
-  synthesis
- Nucleotide synthesis
- Steroid hormone synthesis

**Stoichiometry:** Most biosynthetic reductions require NADPH, not NADH.

## Role 2: Antioxidant Defense
### Glutathione System
```
GSSG + NADPH + H⁺ → 2 GSH + NADP⁺
```
- **Glutathione reductase:** Maintains GSH pool
- GSH: Major intracellular antioxidant

### Thioredoxin System
- Reduces protein disulfides
- DNA synthesis (ribonucleotide reductase)

## Role 3: Immune Function
**NADPH Oxidase (Respiratory Burst):**
```
NADPH + 2O₂ → 2O₂⁻ + NADP⁺ + H⁺
```
- Generates superoxide for pathogen killing
- Critical for phagocyte function
- **Chronic Granulomatous Disease:** NADPH oxidase deficiency

## NADPH vs NADH
| Feature | NADH | NADPH |
|---------|------|-------|
| Primary role | Energy (ETC) | Reduction (biosynthesis) |
| Standard potential | -0.32 V | -0.32 V |
| Pool ratio | Oxidized favored | Reduced favored |
| Metabolic state | Catabolism | Anabolism |
