# Glutamine-Glutamate-Ammonia Cycle

## The Cycle

### Glutamine → Glutamate (Glutaminase)
```
Glutamine + H₂O → Glutamate + NH₄⁺
```
**Enzyme:** Glutaminase
**Tissue-specific roles:**
- **Kidney:** NH₄⁺ for acid excretion (titratable acidity)
- **Brain:** Neurotransmitter synthesis
- **Immune cells:** Energy and biosynthesis
- **Intestine:** Preferred fuel

### Glutamate → Glutamine (Glutamine Synthetase)
```
Glutamate + NH₄⁺ + ATP → Glutamine + ADP + Pi
```
**Enzyme:** Glutamine synthetase
**Location:**
- **Brain:** Ammonia detoxification
- **Liver:** Periportal hepatocytes
- **Muscle:** Nitrogen storage/transport

## Functions

### 1. Ammonia Transport
**Glutamine as safe ammonia carrier:**
- Less toxic than free ammonia
- Transported in blood
- Delivered to liver or kidney for disposal

### 2. Nitrogen Shuttle
**Muscle → Liver:**
- Muscle: Glutamate + NH₄⁺ → Glutamine
- Blood: Glutamine transport
- Liver: Glutamine → Glutamate + NH₄⁺ (→ urea)

### 3. pH Regulation (Kidney)
**Acidosis response:**
- ↑ Glutaminase activity
- ↑ NH₄⁺ production
- NH₄⁺ excreted in urine (buffers acid)

## Tissue-Specific Roles

| Tissue | Primary Role | Enzyme Emphasis |
|--------|--------------|-----------------|
| **Liver** | Urea synthesis | Both (depending on zone) |
| **Kidney** | Acid excretion | Glutaminase |
| **Brain** | Neurotransmitter, detox | Glutamine synthetase (astrocytes) |
| **Muscle** | Nitrogen storage | Glutamine synthetase |
| **Immune** | Energy, proliferation | Glutaminase |
| **Intestine** | Fuel | Glutaminase |

## [[KEYWORD-Clinical|clinical]]: Hyperammonemia
**Impaired glutamine synthesis:**
- Ammonia accumulation
- Neurological dysfunction
- Causes: Liver failure, urea cycle defects, organic acidemias
