# Amino Acids to Krebs Cycle: Complete Metabolic Mapping Table

## The Master Table

| Krebs Cycle Metabolite | Connected Amino Acids |
|------------------------|----------------------|
| **Acetyl-CoA** | Leucine, Isoleucine, Tryptophan |
| **Citrate** | None directly |
| **Isocitrate** | None directly |
| **α-Ketoglutarate** | Glutamate, Proline, Arginine, Histidine |
| **Succinyl-CoA** | Methionine, Isoleucine, Valine, Threonine |
| **Succinate** | None directly |
| **Fumarate** | Phenylalanine, Tyrosine |
| **Malate** | None directly |
| **Oxaloacetate** | Aspartate |

## Key Patterns

### Direct Entry Points
- Only **3 Krebs metabolites** have direct amino acid connections: α-ketoglutarate, succinyl-CoA, oxaloacetate
- **Acetyl-CoA** accepts ketogenic amino acids
- **Fumarate** accepts aromatic amino acid metabolites

### No Direct Amino Acid Connections
- Citrate, isocitrate, succinate, malate — no amino acids feed directly into these
- These are intermediates between entry points

##  Relevance

### Metabolic Disease Connections
- **Maple syrup urine disease:** Defective BCAA → succinyl-CoA pathway
- **Methylmalonic acidemia:** Block in propionyl-CoA → succinyl-CoA conversion
- **Glutaric aciduria type I:** Glutamate/lysine  defect

### Research Applications
- Quantifying amino acid oxidation rates
- Tracing metabolic flux in  resistance
- Understanding gluconeogenesis from protein
