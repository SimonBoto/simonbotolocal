# Amino Acids to Succinyl-CoA: B12 Criticality

## Four Amino Acids

| Amino Acid | Pathway | Notes |
|------------|---------|-------|
| **Isoleucine** | → Propionyl-CoA → Methylmalonyl-CoA → Succinyl-CoA | Branched-chain (also ketogenic) |
| **Valine** | → Propionyl-CoA → Succinyl-CoA | Branched-chain |
| **Methionine** | → S-adenosylmethionine → Homocysteine → Propionyl-CoA → Succinyl-CoA | Methyl donor first |
| **Threonine** | → α-ketobutyrate → Propionyl-CoA → Succinyl-CoA | Threonine dehydrogenase |

## Common Pathway: Propionate 

### Critical Step: Methylmalonyl-CoA Mutase
```
Methylmalonyl-CoA → Succinyl-CoA
```
**Enzyme:** Methylmalonyl-CoA mutase (MMCM)
**Cofactor:** **Vitamin B12 (adenosylcobalamin)**

**B12 deficiency consequences:**
- Methylmalonic acid (MMA) accumulation
- Metabolic acidosis
- Neurological damage (irreversible if prolonged)
- Megaloblastic anemia

### Testing for B12 Deficiency
**Methylmalonic acid (MMA):**
- More sensitive than serum B12
- Elevated in tissue B12 deficiency even with normal serum levels

## Branched-Chain Amino Acids (BCAAs)

### Shared First Step
```
Valine/Isoleucine/Leucine → Branched-chain α-keto acid
```
**Enzyme:** Branched-chain α-keto acid dehydrogenase (BCKDH)

**Maple Syrup Urine Disease (MSUD):**
- BCKDH deficiency
- Accumulation of BCAAs and their α-keto acids
- "Maple syrup" odor in urine
- Severe neurological damage if untreated

### Divergent Fates
| BCAA | Pathway | Classification |
|------|---------|----------------|
| Valine | → Succinyl-CoA | Glucogenic |
| Isoleucine | → Succinyl-CoA + Acetyl-CoA | Both |
| Leucine | → HMG-CoA → Acetyl-CoA + Acetoacetate | Ketogenic |

## Clinical Pearl
**Methylmalonic acidemia (MMA):**
- Can be B12-responsive or non-responsive
- B12-responsive: Supplementation helps
- Non-responsive: Requires dietary restriction of propionate precursors
