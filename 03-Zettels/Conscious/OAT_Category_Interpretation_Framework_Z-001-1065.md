# OAT Category Interpretation Framework

## Overview
Systematic approach to OAT interpretation by marker category.

## The Seven Categories

### 1. Glycolysis Markers
| Marker | Normal | Elevated Indicates |
|--------|--------|-------------------|
| **Lactate** | < 2.0 mmol/mol Cr | Hypoxia, mitochondrial dysfunction |
| **Pyruvate** | < 1.5 mmol/mol Cr | B1 deficiency, PDH block |
| **L/P ratio** | < 20 | Redox status, oxygenation |

### 2. TCA Cycle Markers
| Marker | Interpretation |
|--------|----------------|
| **Citrate** | Elevated = aconitase block (oxidative stress) |
| **Succinate** | Elevated = Complex II dysfunction |
| **α-Ketoglutarate** | Elevated = B-vitamin cofactor insufficiency |

### 3. Fatty Acid Oxidation
| Marker | Interpretation |
|--------|----------------|
| **Suberate** | Medium-chain β-oxidation impairment |
| **Adipate** | General fatty acid oxidation stress |
| **Ethylmalonate** | Multiple acyl-CoA dehydrogenase deficiency |

### 4. B-ependent
| Marker | Vitamin | Interpretation |
|--------|---------|----------------|
| **Methylmalonate** | B12 | Functional B12 deficiency |
| **FIGLU** | Folate | One-carbon  block |
| **Kynurenate** | B6 | Tryptophan metabolism impairment |

### 5. Mitochondrial Distress
| Marker | Interpretation |
|--------|----------------|
| **Lactate/pyruvate** | Redox imbalance |
| **Elevated Krebs intermediates** | Mitochondrial dysfunction |

### 6. Gut-Derived
| Marker | Interpretation |
|--------|----------------|
| **Hippurate** | Low = dysbiosis |
| **Indican** | Elevated = protein malabsorption |
| **Phenylacetate** | Elevated = bacterial overgrowth |

### 7. Oxidative Stress
| Marker | Interpretation |
|--------|----------------|
| **8-OHdG** | DNA oxidation |
| **Oxidized lipids** | Membrane damage |

## Interpretation Sequence
1. **Start with energy markers** (lactate, pyruvate)
2. **Check mitochondrial function** (TCA markers)
3. **Assess cofactor status** (B-vitamin markers)
4. **Evaluate gut contribution** (gut markers)
5. **Note oxidative stress** (oxidative markers)
