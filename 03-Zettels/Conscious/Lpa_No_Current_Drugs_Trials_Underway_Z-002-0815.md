# Lp(a) No Current Drugs — Trials Underway

## Core Claim
**No drugs currently available to lower Lp(a)** — but clinical trials are active with agents that can reduce Lp(a) up to 95%; outcome data expected in coming years.

## Current Status

| Aspect | Status |
|--------|--------|
| **Approved Lp(a) drugs** | None |
| **Trials ongoing** | Yes |
| **Reduction potential** | Up to 95% |
| **Outcome data** | Pending |

## Interim Strategy

While awaiting specific therapies:

| Action | Rationale |
|--------|-----------|
| **Aggressive base risk control** | Hypertension, diabetes, weight |
| **Measure once** | Lp(a) stable over lifetime |
| **Consider earlier intervention** | If Lp(a) elevated |

## Emerging Therapies

| Drug Class | Mechanism | Reduction |
|------------|-----------|-----------|
| **Antisense oligonucleotides** | Reduce Apo(a) production | ~80-95% |
| **siRNA** | Silence LPA gene | ~95% |
