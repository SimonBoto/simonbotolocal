# Succinyl-CoA — Krebs Cycle Anaplerosis

> Succinyl-CoA serves as the entry point for propionate-derived carbons into the Krebs cycle, providing anaplerotic substrate that maintains cycle flux and enables gluconeogenesis.

## Anaplerotic Function

**Definition:**
- "Filling up" the Krebs cycle
- Replenishes intermediates removed for biosynthesis
- Maintains cycle flux

**Sources:**
- Propionate (via methylmalonyl-CoA mutase)
- Odd-chain fatty acids
- Amino acids (isoleucine, methionine, threonine, valine)

## Krebs Cycle Entry

**Succinyl-CoA Position:**
- Intermediate in Krebs cycle
- Between α-ketoglutarate and succinate

**Pathway:**
```
Succinyl-CoA → Succinate (succinyl-CoA synthetase)
    ↓
Fumarate → Malate → Oxaloacetate
    ↓
Can exit to gluconeogenesis
```

## Energy Yield

**From Propionate:**
- Cost: 1 ATP (carboxylation)
- Yield: GTP + FADH₂ + NADH
- Net: Positive energy balance

**Gluconeogenesis:**
- Oxaloacetate → PEP → Glucose
- Net glucose production possible
