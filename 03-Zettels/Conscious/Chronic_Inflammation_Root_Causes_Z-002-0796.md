# Chronic Inflammation Root Causes

## Core Claim
**Chronic low-grade inflammation arises from ongoing metabolic damage combined with nutritional deficiencies** — creating a pro-inflammatory state measurable by elevated hsCRP that drives insulin resistance, endothelial dysfunction, and atherosclerosis.

## Two Primary Drivers

### 1. Ongoing Damage (Inflammatory Stimuli)
| Source | Mechanism |
|--------|-----------|
| **Ultra-processed foods** | Advanced glycation end-products, oxidized lipids, additives |
| **Smoking** | Oxidative stress, endothelial injury |
| **Excess alcohol** | Gut dysbiosis, liver inflammation, oxidative stress |
| **Chronic stress** | Cortisol dysregulation, sympathetic overactivity |
| **Poor sleep** | Impaired resolution, cytokine elevation |

### 2. Deficiencies (Impaired Resolution)
| Nutrient | Role in Inflammation Resolution |
|----------|--------------------------------|
| **Omega-3 fatty acids** | Resolvins, protectins — active resolution |
| **Vitamin D** | Immune modulation, anti-inflammatory |
| **Magnesium** | Cofactor for anti-inflammatory enzymes |
| **Zinc** | Immune function, wound healing |
| **Antioxidants** (C, E, polyphenols) | Neutralize oxidative stress |

## The Vicious Cycle
```
Damage → Inflammation → Deficiencies → Impaired Healing → Chronic Inflammation
```

## Consequences Beyond CVD
- **Insulin resistance** → Type 2 diabetes
- **Autoimmune activation** → Autoimmune diseases
- **Cellular transformation** → Cancer
- **Neuroinflammation** → Cognitive decline

## Clinical Approach
Address **both sides**: Remove inflammatory stimuli + replenish resolution nutrients.
