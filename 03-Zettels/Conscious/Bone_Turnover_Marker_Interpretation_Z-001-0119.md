# Bone Turnover Marker Interpretation

## Antiresorptive Therapy Response
- **Expected:** >50% reduction in resorption markers (CTX, NTX) within 3 months
- **Non-response:** <30% reduction suggests poor adherence or inadequate dosing
- **Clinical correlation:** Resorption marker suppression correlates with fracture risk reduction

## Anabolic Therapy Response
- **Expected:** >50% increase in formation markers (P1NP, OC) within 3 months
- **Non-response:** <30% increase suggests possible non-adherence or biological resistance
- **Temporal pattern:** Formation markers rise before resorption markers fall

## Discordant Patterns
| Pattern | Interpretation | Action |
|---------|----------------|--------|
| **↓ Resorption, ↑ Formation** | Optimal response | Continue therapy |
| **↓ Resorption, ↔ Formation** | Antiresorptive only effect | Consider sequential therapy |
| **↔ Resorption, ↑ Formation** | Anabolic only effect | Monitor for bone gain |
| **↔ Resorption, ↔ Formation** | Non-response | Assess adherence, consider alternative therapy |
