# Anaplerosis: Replenishing Krebs Cycle Intermediates

## The Problem
**Gluconeogenesis depletes oxaloacetate from Krebs cycle.**

When OAA is converted to PEP for glucose synthesis, the Krebs cycle would stall without replenishment.

## Anaplerotic Reactions
**"Filling up" reactions** that add intermediates to Krebs cycle.

### 1. Pyruvate Carboxylase (Major)
```
Pyruvate + CO₂ + ATP → Oxaloacetate + ADP + Pi
```

**Characteristics:**
- **Enzyme:** Pyruvate carboxylase (PC)
- **Cofactor:** Biotin (B7)
- **Activator:** Acetyl-CoA (feed-forward)
- **Location:**  (liver, kidney)

**Regulation:**
- ↑ Acetyl-CoA stimulates PC
- Coordinates with fatty acid oxidation

### 2. Aspartate Aminotransferase
```
Aspartate + α-ketoglutarate → Oxaloacetate + Glutamate
```
- Uses amino acid carbon skeletons
- Connects protein  to Krebs cycle

### 3. Glutamate Dehydrogenase
```
Glutamate + NAD⁺ → α-ketoglutarate + NADH + NH₄⁺
```
- Adds α-ketoglutarate (not OAA, but Krebs intermediate)
- Major route for amino acid entry

## Anaplerosis vs Cataplerosis

| Process | Direction | Purpose |
|---------|-----------|---------|
| **Anaplerosis** | IN → Krebs | Replenish intermediates |
| **Cataplerosis** | OUT ← Krebs | Remove for biosynthesis |

**Balance:**
- Gluconeogenesis is cataplerotic (removes OAA)
- Pyruvate carboxylase is anaplerotic (adds OAA)
- Must balance for cycle to function

##  Relevance

### Pyruvate Carboxylase Deficiency
**Rare, severe:**
- Cannot make OAA from pyruvate
- **Lactic acidosis** (pyruvate accumulates)
- **Hypoglycemia** (cannot do gluconeogenesis)
- **Neurological damage** (brain depends on glucose)
- **Treatment:** Triheptanoin (anaplerotic odd-chain fat)

### Biotin Deficiency
- Impairs pyruvate carboxylase
- Similar but milder symptoms
- Treatable with biotin supplementation

### Ketogenic Diet
- **Reduced anaplerotic demand:** Ketones spare glucose
- **Brain uses ketones:** Less need for hepatic gluconeogenesis
- **OAA can remain** in Krebs cycle

## Quantitative Balance
**In fasting liver:**
- Gluconeogenesis removes OAA
- Pyruvate carboxylase adds OAA
- Net: OAA diverted to glucose, acetyl-CoA diverted to ketones

**Equation:**
```
Anaplerosis = Cataplerosis (steady state)
OR
Anaplerosis < Cataplerosis (OAA depletion → ketogenesis)
```
