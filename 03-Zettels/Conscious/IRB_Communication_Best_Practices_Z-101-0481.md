# IRB Communication Best Practices

## Overview
Effective communication with ethics committees is essential for timely approval and ongoing compliance.

## Initial Submission

### Pre-Submission
| Action | Purpose |
|--------|---------|
| **Pre-submission meeting** | Clarify requirements, identify concerns early |
| **Template review** | Ensure complete application |
| **Colleague feedback** | Catch errors before official submission |
| **Confirm receipt** | Verify application received and logged |

### Application Completeness
- [ ] All required sections completed
- [ ] Informed consent form attached
- [ ] Protocol detailed and clear
- [ ] Risk assessment thorough
- [ ] Recruitment materials included
- [ ] CVs and training documentation attached

## During Review

### Expected Timelines
| Committee Type | Typical Timeline |
|----------------|------------------|
| **Expedited review** | 1-2 weeks |
| **Full board review** | 4-6 weeks |
| **Continuing review** | 2-4 weeks |

### Follow-Up Protocol
| Timing | Action |
|--------|--------|
| **Week 2** | Confirm receipt if no acknowledgment |
| **Week 4** (full review) | Polite inquiry on status |
| **Week 6** | More direct follow-up |
| **Week 8+** | Escalation to supervisor/ombudsman |

## Responding to Feedback

### Revision Approach
1. **Read completely** — Understand all concerns
2. **Prioritize** — Address major issues first
3. **Be specific** — Point-by-point response
4. **Document changes** — Track revisions clearly
5. **Resubmit promptly** — Don't delay

### Response Letter Format
```
Dear Committee,

Thank you for your review of [Protocol #]. We have carefully 
considered your feedback and revised the protocol as follows:

Point 1: [Committee concern]
Response: [How addressed]
Revision: [Specific change made]

[Repeat for each point]

We believe these revisions address all concerns and respectfully 
request re-review.

Sincerely,
[Name]
```

## Problem Resolution

### Common Issues and Solutions
| Issue | Solution |
|-------|----------|
| **No response** | Escalation pathway; multiple channels |
| **Unclear feedback** | Request clarification meeting |
| **Disagreement** | Professional dialogue; find compromise |
| **Major revisions** | Revise thoroughly; don't argue minor points |
| **Continuing delays** | Involve supervisor; institutional support |

### Escalation Pathway
1. Committee administrator
2. Committee chair
3. Department head
4.  integrity office
5. Institutional ombudsman

## Ongoing Communication

### After Approval
- Submit amendments promptly
- Report adverse events immediately
- Complete continuing reviews on time
- Notify of study closure

### Relationship Building
- Attend IRB meetings when possible
- Volunteer for committee service (post-PhD)
- Thank committee members
- Provide feedback on process

## Templates

### Status Inquiry Email
```
Subject: Status Inquiry — Protocol [Number]

Dear [Committee Administrator/Chair],

I am writing to inquire about the status of my ethics application 
[Protocol #], submitted on [Date]. 

[Context: PhD timeline, funding requirements, etc.]

I would appreciate any update you can provide on the review timeline 
or any additional information needed.

Thank you for your time and assistance.

Best regards,
[Name, Title, Institution]
[Contact information]
```
