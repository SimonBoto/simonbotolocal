## Core Claim
**Aspartate and asparagine are glucogenic amino acids that enter the Krebs cycle at oxaloacetate — aspartate undergoes direct transamination to OAA (B6-dependent), while asparagine is first deamidated to aspartate** — this direct entry to the Krebs cycle makes them efficient glucose precursors.

## Evidence

### Aspartate → Oxaloacetate
```
Aspartate + α-ketoglutarate ↔ Oxaloacetate + Glutamate
```
| Feature | Detail |
|---------|--------|
| **Enzyme** | Aspartate aminotransferase (AST) |
| **Cofactor** | Vitamin B6 (pyridoxal phosphate) |
| **Isozymes** | Mitochondrial and cytosolic |

### Asparagine → Aspartate
```
Asparagine + H₂O → Aspartate + NH₃
```
| Feature | Detail |
|---------|--------|
| **Enzyme** | Asparaginase |
| **Location** | Liver, lymphocytes |

### AST Clinical Marker
| Condition | AST Pattern |
|-----------|-------------|
| **Normal** | AST < ALT |
| **Alcoholic liver disease** | AST > ALT (2:1 ratio) |
| **Myocardial infarction** | Elevated AST |

## Clinical Implication
- **Glucogenic capacity:** Both yield glucose via OAA
- **AST/ALT ratio:** Diagnostic for liver disease etiology
- **Asparaginase therapy:** Used in ALL (acute lymphoblastic leukemia) — depletes asparagine
- **METHAP relevance:** B6 required for transamination; adequate B6 supports amino acid metabolism
