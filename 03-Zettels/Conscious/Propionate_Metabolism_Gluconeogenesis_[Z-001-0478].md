# Propionate  and Gluconeogenesis

## Propionate: A Short-Chain Fatty Acid
**Structure:** CH₃-CH₂-COOH (3-carbon carboxylic acid)
**NOT a keto acid** (no ketone group)

## Sources
1. **Gut microbiome fermentation** (dietary fiber)
2. **Amino acid catabolism** (Met, Val, Ile, Thr)
3. **Dairy products** (natural content)

## Metabolism: Propionate → Glucose
**Gluconeogenic pathway (liver only):**

| Step | Conversion | Enzyme | Cofactor |
|------|------------|--------|----------|
| 1 | Propionate → Propionyl-CoA | Propionyl-CoA synthetase | ATP, CoA |
| 2 | Propionyl-CoA → Methylmalonyl-CoA | Propionyl-CoA carboxylase | **Biotin (B7)** |
| 3 | Methylmalonyl-CoA → Succinyl-CoA | Methylmalonyl-CoA mutase | **B12 (cobalamin)** |
| 4 | Succinyl-CoA → Krebs cycle | Various | - |
| 5 | Succinyl-CoA → Oxaloacetate | TCA cycle | - |
| 6 | OAA → Glucose | Gluconeogenesis | - |

**Net:** Propionate is **gluconeogenic** → yields glucose

## ependencies
- **Biotin (B7):** Carboxylation step
- **B12 (cobalamin):** Mutase reaction

**Deficiency consequences:**
- Propionic acidemia (biotin-related)
- Methylmalonic acidemia (B12-related)

## Physiological Roles
1. **Energy substrate** (via gluconeogenesis)
2. **Gut hormone signaling** (GLP-1, PYY)
3. **Metabolic health** (improves  sensitivity)
4. **Cholesterol synthesis** (indirect, via succinyl-CoA)

## Comparison: Pyruvate vs Propionate
| Feature | Pyruvate | Propionate |
|---------|----------|------------|
| Class | α-Keto acid | Carboxylic acid |
| Ketone group | Yes | No |
| Primary fate | Acetyl-CoA | Glucose |
| Key vitamins | B1, B2, B3, B5 | B7, B12 |
