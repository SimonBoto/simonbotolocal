# Methionine Cycle — Activation and Regeneration

> The methionine cycle activates methionine to SAM for  reactions, then converts the byproduct SAH to homocysteine, which is either remethylated to methionine or enters transsulfuration.

## Step 1: SAM Synthesis

**Reaction:**
```
Methionine + ATP → SAM + PPi + Pi
```

**Enzyme:** Methionine adenosyltransferase (MAT)
- Commits methionine to metabolic fate
- Irreversible reaction

## Step 2: Methyl Donation

**Reaction:**
```
SAM + Acceptor → Methylated product + SAH
```

**Methyltransferases:** >100 different enzymes
**Function:** DNA methylation, histone methylation, neurotransmitter synthesis

## Step 3: SAH Hydrolysis

**Reaction:**
```
SAH ↔ Homocysteine + Adenosine
```

**Enzyme:** SAH hydrolase (reversible)
- Product removal drives forward

## Step 4: Branch Point

**Option A: Remethylation**
- Homocysteine + 5-MTHF → Methionine
- B12-dependent

**Option B: Transsulfuration**
- Homocysteine → Cysteine
- B6-dependent
