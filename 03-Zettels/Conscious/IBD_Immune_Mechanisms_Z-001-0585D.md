## Core Claim
The immune mechanisms underlying IBD differ fundamentally between Crohn's disease (Th1/Th17 dominance) and ulcerative colitis (Th2 dominance), explaining the distinct inflammatory patterns, serological markers, and treatment responses.

## Key Concepts

### Comparative Features Framework

**Category:** Clinical Differentiation

**Key Process:** 
```
Distribution: Any GI vs Colon only → Pattern: Skip vs Continuous → Depth: Transmural vs Mucosal → Rectal: Variable vs Universal → Granulomas: Present vs Absent
```

**Mechanism/Steps:**
1. Distribution: Crohn's affects any GI segment; UC only colon
2. Pattern: Crohn's shows skip lesions; UC continuous
3. Depth: Crohn's transmural; UC mucosal only
4. Rectal involvement: Crohn's variable; UC always involves rectum
5. Granulomas: Crohn's present (40-60%); UC absent

**Purpose/Significance:** Provides the clinical criteria for differential diagnosis that guides diagnostic testing and management decisions
