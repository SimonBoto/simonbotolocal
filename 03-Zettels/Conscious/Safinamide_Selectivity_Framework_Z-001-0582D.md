# Safinamide Selectivity Framework

> **Core Claim:** Safinamide (Xadago) exemplifies clean MAO-B selectivity with additional sodium/calcium channel blocking properties for Parkinson's disease.

## Drug Profile

| Property | Characteristic |
|----------|---------------|
| **Generic name** | Safinamide |
| **Brand name** | Xadago |
| **Target** | MAO-B (highly selective) |
| **Use** | Parkinson's disease add-on therapy |

## Selectivity Advantage

| Feature | Benefit |
|---------|---------|
| **MAO-B selective** | Preserves serotonin/norepinephrine |
| **Minimal MAO-A inhibition** | Reduced tyramine sensitivity |
| **No DAO effect** | No histamine interaction |
| **Fewer dietary restrictions** | Better patient compliance |

## Additional Mechanisms

Beyond MAO-B inhibition:
- **Sodium channel blockade**
- **Calcium channel modulation**

## Clinical Value

Demonstrates how enzyme selectivity in drug development:
- Minimizes off-target effects
- Reduces side effects
- Improves safety profile
