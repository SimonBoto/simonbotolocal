## Core Claim
**Cholesterol serves as the universal precursor for all steroid hormones — synthesized through tissue-specific pathways producing glucocorticoids (cortisol), mineralocorticoids (aldosterone), androgens (testosterone), and estrogens (estradiol) — with the initial conversion to pregnenolone by CYP11A1 being the committed step in steroidogenesis.**

## Evidence

### Steroidogenesis Overview
| Hormone Class | Primary Hormone | Site | Key Function |
|---------------|-----------------|------|--------------|
| **Glucocorticoids** | Cortisol | Adrenal cortex (zona fasciculata) | Stress response, metabolism |
| **Mineralocorticoids** | Aldosterone | Adrenal cortex (zona glomerulosa) | Sodium/potassium balance |
| **Androgens** | Testosterone/DHEA | Testes, ovaries, adrenal | Sexual development, anabolism |
| **Estrogens** | Estradiol | Ovaries, adipose (aromatase) | Female reproduction, bone |

### Pathway Steps

#### Common Initial Step
```
Cholesterol → Pregnenolone (CYP11A1, side-chain cleavage)
```

#### Glucocorticoid Pathway (Cortisol)
```
Pregnenolone → Progesterone → 17-OH-Progesterone → 
11-Deoxycortisol → Cortisol
```
| Enzyme | Step | Regulation |
|--------|------|------------|
| 3β-HSD | Pregnenolone → Progesterone | ACTH stimulated |
| CYP17 | 17-hydroxylation | Tissue-specific |
| CYP21A2 | 21-hydroxylation | Essential |
| CYP11B1 | 11-hydroxylation | ACTH dependent |

#### Mineralocorticoid Pathway (Aldosterone)
```
Pregnenolone → Progesterone → 11-Deoxycorticosterone → 
Corticosterone → Aldosterone
```
| Factor | Effect |
|--------|--------|
| **Angiotensin II** | Stimulates aldosterone synthase |
| **Potassium** | Direct stimulation |
| **ACTH** | Acute stimulation |
| **Sodium** | Inhibits (negative feedback) |

#### Androgen Pathway
```
Pregnenolone → 17-OH-Pregnenolone → DHEA → 
Androstenedione → Testosterone
```
| Enzyme | Function | Location |
|--------|----------|----------|
| CYP17 | 17,20-lyase | Adrenal, gonads |
| 3β-HSD | DHEA → Androstenedione | Multiple tissues |
| 17β-HSD | Androstenedione → Testosterone | Gonads |

#### Estrogen Pathway
```
Testosterone → Estradiol (aromatase/CYP19)
```
| Tissue | Aromatase Activity | Clinical Relevance |
|--------|-------------------|-------------------|
| **Ovaries** | High | Primary estrogen source |
| **Adipose** | Moderate | Obesity → estrogen ↑ |
| **Brain** | Low | Neuroprotection |
| **Bone** | Low | Local estrogen effects |

### Steroid Hormone Regulation
| Factor | Cortisol | Aldosterone | Testosterone | Estradiol |
|--------|----------|-------------|--------------|-----------|
| **Primary regulator** | ACTH | Angiotensin II | LH | FSH/LH |
| **Circadian rhythm** | Yes (morning peak) | No | Yes | Cyclic |
| **Stress response** | ↑↑↑ | ↑ | Variable | Variable |
| **Feedback** | Negative | Negative | Negative | Negative |

### Clinical Correlations
| Condition | Mechanism | Presentation |
|-----------|-----------|--------------|
| **Cushing's syndrome** | Cortisol excess | Weight gain, hypertension |
| **Addison's disease** | Cortisol deficiency | Fatigue, hypotension |
| **Hyperaldosteronism** | Aldosterone excess | Hypertension, hypokalemia |
| **Hypogonadism** | Testosterone deficiency | Fatigue, low libido |
| **Aromatase excess** | Estrogen overproduction | Gynecomastia (males) |

## Clinical Implication
- **Statin effects:** Cholesterol reduction may impact hormone synthesis
- **Tissue specificity:** Same precursor, different end products
- **Feedback loops:** Complex regulation prevents excess/deficiency
- **Metabolic syndrome:** Obesity alters aromatase activity
- **METHAP relevance:** Hormonal balance is critical in metabolic syndrome; statin therapy requires monitoring for hormonal side effects
