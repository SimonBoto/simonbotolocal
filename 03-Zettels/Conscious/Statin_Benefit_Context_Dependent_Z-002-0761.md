# Statin Benefit Is Context-Dependent — Risk Stratification Essential

## Core Claim
**Statin benefit is highly context-dependent** — the same LDL reduction produces dramatically different absolute risk reductions depending on baseline CVD risk, CAC score, and metabolic health. Risk stratification is essential for appropriate statin prescribing.

## Number Needed to Treat (NNT)

### High-Risk Patients
| Risk Level | NNT (5 years) |
|------------|---------------|
| **Secondary prevention** | 20-30 |
| **Diabetes + CVD** | 25-35 |
| **High CAC (>100)** | 30-40 |

### Low-Risk Patients
| Risk Level | NNT (5 years) |
|------------|---------------|
| **Primary prevention, healthy** | 100-200 |
| **CAC = 0** | >200 |
| **Young, no risk factors** | >500 |

## Risk Stratification Framework

### Very High Risk
| Criteria | Action |
|----------|--------|
| **Established CVD** | High-intensity statin |
| **LDL goal** | <55 mg/dL |
| **Add ezetimibe/PCSK9 if needed** | Yes |

### High Risk
| Criteria | Action |
|----------|--------|
| **Diabetes + risk factors** | Moderate-high intensity |
| **Severe hypercholesterolemia** | Treat aggressively |
| **LDL goal** | <70 mg/dL |

### Moderate Risk
| Criteria | Action |
|----------|--------|
| **Multiple risk factors** | Moderate intensity |
| **Consider CAC** | Yes |
| **LDL goal** | <100 mg/dL |

### Low Risk
| Criteria | Action |
|----------|--------|
| **CAC = 0** | Lifestyle only |
| **Young, healthy** | Defer statin |
| **LDL goal** | <130 mg/dL (lifestyle) |

## CAC Score Guides Treatment

| CAC Score | Statin Benefit |
|-----------|----------------|
| **0** | Minimal — lifestyle first |
| **1-10** | Low — shared decision |
| **11-100** | Moderate — consider statin |
| **>100** | High — definite benefit |
| **>400** | Very high — aggressive therapy |

## Clinical Implication

### Don't Treat Numbers, Treat Patients
| Approach | Rationale |
|----------|-----------|
| **CAC-guided** | Direct plaque measurement |
| **Risk-based** | Absolute benefit matters |
| **Shared decision** | Patient preferences count |
| **Lifestyle first** | Foundation for all |

### When to Question Statin
| Scenario | Consider |
|----------|----------|
| **CAC = 0, age <50** | Defer |
| **Side effects** | Reassess risk/benefit |
| **Metabolic syndrome** | Address insulin first |
| **Frail elderly** | Individualize |
