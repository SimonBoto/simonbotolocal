# Cysteine — Metabolic Fates

> Cysteine serves as precursor for glutathione, taurine, and inorganic sulfate, or is degraded to pyruvate for energy production.

## Pathway 1: Glutathione Synthesis

**Reaction:**
```
Cysteine + Glutamate + Glycine → Glutathione
```

**Function:**
- Primary cellular antioxidant
- Detoxification
- Redox regulation

## Pathway 2: Taurine Synthesis

**Pathway:**
```
Cysteine → Cysteine sulfinic acid → Hypotaurine → Taurine
```

**Function:**
- Bile acid conjugation
- Osmoregulation
- Neuromodulation

## Pathway 3: Sulfate Production

**Fate:**
- Cysteine → Inorganic sulfate + Pyruvate
- Sulfation reactions
- Excreted in urine

## Pathway 4: Energy Production

**Degradation:**
```
Cysteine → Pyruvate → Acetyl-CoA → Krebs cycle
```

**Result:**
- Glucogenic amino acid
- ATP production
- CO₂ + H₂O
