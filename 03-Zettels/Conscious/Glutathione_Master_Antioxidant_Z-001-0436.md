# Glutathione — The Master Antioxidant

> Glutathione (GSH) is a tripeptide (γ-glutamyl-cysteinyl-glycine) synthesized from cysteine, glutamate, and glycine, serving as the primary intracellular antioxidant and detoxification agent.

## Synthesis

**Step 1:**
```
Glutamate + Cysteine → γ-Glutamylcysteine (GCL enzyme)
```

**Step 2:**
```
γ-Glutamylcysteine + Glycine → Glutathione (GSS enzyme)
```

**Rate-limiting:** Cysteine availability

## Functions

### Antioxidant
```
2 GSH + H₂O₂ → GSSG + 2 H₂O
```
- Neutralizes ROS
- Regenerates vitamin C, E

### Detoxification
- Conjugates toxins
- Phase II 
- Heavy metal chelation

### Redox Regulation
- GSH:GSSG ratio indicates cellular redox state
- ~100:1 in healthy cells

## Concentration

**Intracellular:** 1-10 mM (most abundant antioxidant)
**High in:** Liver, kidney, lens, RBCs
