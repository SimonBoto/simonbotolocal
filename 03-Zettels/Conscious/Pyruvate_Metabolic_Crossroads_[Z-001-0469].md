# Pyruvate: Metabolic Crossroads

## Structure
- **3-carbon α-keto acid**
- **Formula:** CH₃COCOOH
- **Simplest α-keto acid**

## Glycolytic Endpoint
```
Glucose → ... → 2 Pyruvate
```

## Metabolic Fates

### Aerobic: l Oxidation
```
Pyruvate → Acetyl-CoA → Krebs cycle
```
**Enzyme:** Pyruvate dehydrogenase complex (PDH)
**Cofactors:** B1, B2, B3, B5, lipoate

### Anaerobic: Lactate Fermentation
```
Pyruvate + NADH + H⁺ → Lactate + NAD⁺
```
**Enzyme:** Lactate dehydrogenase (LDH)
**Purpose:** Regenerates NAD⁺ for continued glycolysis

### Yeast: Alcoholic Fermentation
```
Pyruvate → Acetaldehyde → Ethanol + CO₂
```

### Gluconeogenic: Pyruvate Carboxylase
```
Pyruvate + CO₂ + ATP → Oxaloacetate
```
**Enzyme:** Pyruvate carboxylase (biotin)
**Purpose:** Gluconeogenesis, anaplerosis

### Transamination
```
Pyruvate + Glutamate → Alanine + α-ketoglutarate
```
**Enzyme:** ALT (alanine aminotransferase)

## Central Role
Pyruvate connects:
- **Glycolysis** (carbohydrates)
- **Krebs cycle** (aerobic energy)
- **Gluconeogenesis** (glucose synthesis)
- **Amino acid ** (transamination)
- **Lipogenesis** (acetyl-CoA → )

## Clinical
**Pyruvate dehydrogenase deficiency:**
- Lactic acidosis, neurological impairment
- Treat with ketogenic diet (bypasses PDH)
