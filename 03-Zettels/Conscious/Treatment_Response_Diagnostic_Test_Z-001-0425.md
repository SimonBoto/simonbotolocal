## Core Principle

> Response to B12 therapy is itself diagnostic: reticulocytosis peaks at 5-7 days, hemoglobin rises by 2 months; no response suggests alternative diagnosis.

## Expected B12 Response

**Timeline:**
| Time | Response |
|------|----------|
| **Day 3-5** | Reticulocytosis peaks (evidence of response) |
| **Week 2-4** | Hemoglobin begins rising |
| **Month 2-3** | Hemoglobin normalizes |
| **Month 6-12** | Neurological improvement (if early treatment) |

## Core Principle

> Response to B12 therapy is itself diagnostic: reticulocytosis peaks at 5-7 days, hemoglobin rises by 2 months; no response suggests alternative diagnosis.

## No Response Indicates

**Alternative Diagnosis:**
- Not B12 deficiency
- Concurrent deficiency (iron, )
- Non-compliance
- Misdiagnosis

**MM Will NOT Respond:**
- Confirms alternative pathology
- Avoids delay in appropriate therapy
- Critical for patient outcome

## Core Principle

> Response to B12 therapy is itself diagnostic: reticulocytosis peaks at 5-7 days, hemoglobin rises by 2 months; no response suggests alternative diagnosis.

##  Use

**Diagnostic Trial:**
- Start B12 while awaiting full workup
- Monitor reticulocyte count
- Rapid response confirms B12 deficiency
- No response → pursue MM/other diagnosis

**Caution:**
- Don't delay MM workup if high suspicion
- B12 trial shouldn't replace biopsy if indicated
- Use as adjunct to other diagnostics
