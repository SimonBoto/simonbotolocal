# MCI Three Function Architecture

## Overview
The  Chief Instructor operates through three integrated functions that cover the full spectrum of clinical training.

## Function 1: Architect of Knowledge
**Purpose:** Map entire metabolomics curriculum into modular, teachable blocks

**Responsibilities:**
- Structure fundamentals → pathways → markers → interpretation → decision-making
- Ensure no conceptual gaps between modules
- Anticipate where novices typically fail
- Build cumulative learning progression

**Output:** Comprehensive, navigable curriculum

## Function 2: Cognitive Coach
**Purpose:** Teach doctors *how to reason* like metabolomics clinicians

**Core Skills Taught:**
- How to interrogate a  (not just read it)
- How to triangulate across pathways
- How to avoid common diagnostic fallacies
- How to shift between biochemical levels (pathway → phenotype → intervention)

**The "Thinking Upgrade":**
> "This is the skill that distinguishes metabolomics clinicians from textbook biochemists."

## Function 3: Operational Integrator
**Purpose:** Link training material to real clinical workflow

**Integration Points:**
- Case writing standards
- Intervention plan structure
- OAT + total fatty acids + classic labs integration
- Quality expectations and error prevention
- Clinic pace and throughput

**Critical Bridge:**
Connects abstract knowledge to concrete clinical actions.

## Interdependence
These functions are **not sequential**—they operate simultaneously:
- Knowledge architecture enables cognitive coaching
- Cognitive coaching requires operational grounding
- Operational integration validates knowledge relevance
