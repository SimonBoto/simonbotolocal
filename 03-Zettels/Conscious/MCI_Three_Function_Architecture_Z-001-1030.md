## Core Claim
**The Metabolomics Chief Instructor operates through three integrated functions — Architect of Knowledge (curriculum design), Cognitive Coach (reasoning development), and Operational Integrator (systems thinking) — each addressing distinct training needs while supporting the others for comprehensive clinician development.**

## Evidence

### Function 1: Architect of Knowledge
| Aspect | Implementation |
|--------|----------------|
| **Curriculum mapping** | Modular, teachable, cumulative blocks |
| **Content sequencing** | Prerequisites → Foundation → Application → Integration |
| **Resource creation** | Teaching materials, frameworks, hierarchies |

**Curriculum Architecture:**
```
Foundation (Weeks 1-4) → Application (Weeks 5-8) → Integration (Weeks 9-12)
```

### Function 2: Cognitive Coach
| Aspect | Implementation |
|--------|----------------|
| **Socratic questioning** | Guide discovery, not provide answers |
| **Error analysis** | Mistakes as learning opportunities |
| **Pattern recognition** | Explicit reasoning modeling |

### Function 3: Operational Integrator
| Aspect | Implementation |
|--------|----------------|
| **Workflow integration** | Link lessons to clinical practice |
| **Quality systems** | Pre-analytical discipline, reporting standards |
| **Outcome tracking** | Patient results feed back to training |

### Integration
| Function Pair | Synergy |
|---------------|---------|
| **Architect + Coach** | Curriculum builds reasoning skills |
| **Coach + Integrator** | Reasoning applies to real cases |
| **Integrator + Architect** | Practice informs curriculum updates |

## Clinical Implication
- **Training design:** Each function requires distinct skills and time allocation
- **Assessment:** Evaluate trainees on knowledge, reasoning, AND operational competence
- **Faculty development:** MCIs need training in all three functions
- **METHAP relevance:** Research team training requires similar architecture — knowledge of protocols, reasoning about data, and operational execution
