# Histone  — Chromatin Structure

> Histone methylation at specific lysine and arginine residues alters chromatin structure and gene accessibility, with distinct marks associated with transcriptional activation or repression.

## Histone Structure

**Core Histones:**
- H2A, H2B, H3, H4 (two each = octamer)
- Form nucleosome core
- DNA wraps around nucleosome

**N-Terminal Tails:**
- Extend from nucleosome
- Rich in modification sites
- Lysine and arginine residues

## Key Methylation Marks

### Activating Marks

| Mark | Name | Function |
|------|------|----------|
| **H3K4me3** | Trimethylated H3K4 | Active promoters, transcription start sites |
| **H3K36me3** | Trimethylated H3K36 | Active gene bodies, elongation |
| **H3K79me2** | Dimethylated H3K79 | Active transcription |

### Repressive Marks

| Mark | Name | Function |
|------|------|----------|
| **H3K27me3** | Trimethylated H3K27 | Polycomb repression, development |
| **H3K9me3** | Trimethylated H3K9 | Heterochromatin, silencing |
| **H4K20me3** | Trimethylated H4K20 | DNA repair, replication |

## Enzymes

**Histone Methyltransferases (HMTs):**
- Add methyl groups (SAM-dependent)
- Specific for lysine or arginine
- Site-specific (e.g., H3K4, H3K27)

**Histone Demethylases (HDMs):**
- Remove methyl groups
- Allows dynamic regulation
- Responds to cellular signals

## Functional Consequences

**Gene Expression:**
- **Activating marks:** Open chromatin, transcription factors access DNA
- **Repressive marks:** Condensed chromatin, gene silenced

**Cellular Memory:**
- Marks maintained through cell division
- Defines cell identity
- Explains differentiation stability

## Nutrient Connection

**SAM Availability:**
- HMTs require SAM as methyl donor
- B12/ deficiency → Reduced SAM
- Reduced SAM → Altered histone methylation
- Result: Epigenetic dysregulation

**Example:**
- H3K4me3 at metabolic gene promoters
- B12 deficiency → Reduced H3K4me3
- Reduced expression of metabolic genes
