# Transsulfuration Pathway — Cysteine Synthesis

> The transsulfuration pathway converts homocysteine to cysteine via cystathionine, requiring vitamin B6 and providing the sulfur atom for cysteine and glutathione synthesis.

## Step 1: Cystathionine β-Synthase (CBS)

**Reaction:**
```
Homocysteine + Serine → Cystathionine
```

**Enzyme:** CBS
**Cofactor:** Pyridoxal phosphate (vitamin B6)
**Location:** Liver, kidney, pancreas, brain

## Step 2: Cystathionine γ-Lyase (CGL)

**Reaction:**
```
Cystathionine → Cysteine + α-Ketobutyrate + NH₃
```

**Enzyme:** CGL (cystathionase)
**Cofactor:** Pyridoxal phosphate (vitamin B6)

## Regulation

**SAM Activation:**
- High SAM activates CBS
- Pushes homocysteine toward transsulfuration when methionine abundant

**Low SAM:**
- CBS inhibited
- Conserves methionine
- Favors re

**Vitamin B6:**
- Required for both steps
- Deficiency impairs transsulfuration
- Elevated homocysteine results
