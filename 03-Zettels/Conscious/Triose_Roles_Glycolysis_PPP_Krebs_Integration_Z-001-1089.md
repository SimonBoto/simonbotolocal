# Triose Roles  PPP Krebs Integration

## Overview
Trioses serve as metabolic crossroads, connecting glycolysis, the pentose phosphate pathway, and the  through interconversion and shared intermediates.

## Glycolysis: The Triose Phase

### The Split
```
Fructose-1,6-bisphosphate (6C)
           ↓ Aldolase
Dihydroxyacetone-P + Glyceraldehyde-3-P (2 × 3C)
```

### Interconversion
| Reaction | Enzyme | Direction |
|----------|--------|-----------|
| DHAP ↔ G3P | Triose phosphate isomerase | Reversible |

### Yield from One Glucose
- 2 G3P molecules per glucose
- Each generates: 2 ATP, 1 NADH, 1 Pyruvate

## Pentose Phosphate Pathway Connection

### Non-Oxidative Phase
```
Ribose-5-P ↔ Ribulose-5-P ↔ Xylulose-5-P
                    ↓
            Transketolase/Transaldolase
                    ↓
        Glyceraldehyde-3-P + Fructose-6-P
```

### Metabolic Flexibility
| Cellular Need | PPP Direction | Output |
|---------------|---------------|--------|
| High NADPH | Oxidative | NADPH + Ribose-5-P |
| High ribose | Oxidative | Ribose-5-P |
| Need both | Full PPP | Balanced output |
| Excess ribose | Non-oxidative | Glycolytic intermediates |

## Krebs Cycle Connection

### Entry Points
| Intermediate | Source | Krebs Entry |
|--------------|--------|-------------|
| **Pyruvate** | Glycolysis | Acetyl-CoA (via PDH) |
| **G3P** | Glycolysis/PPP | Pyruvate → Acetyl-CoA |

### Carbon Flow
```
Glucose → G3P → Pyruvate → Acetyl-CoA → Krebs
   ↓         ↓
   PPP ←─────┘ (if ribose/NADPH needed)
```

## Evolutionary Integration
- Glycolysis: Ancient, universal
- PPP: Evolved to provide biosynthetic precursors
- Krebs: Aerobic expansion of metabolic capacity
- **Trioses as conserved interface** between all three
