# IBD Environmental Triggers

> **Source:** ChatGPT conversation [[CGPT_0209_Gastroenterology_Registrar_Assessment_CGPT-0209]] (November 18, 2025)

## Core Claim
Environmental factors including diet, smoking, medications, and microbial exposures interact with genetic susceptibility to trigger and modulate IBD activity, with distinct patterns observed between Crohn's disease (smoking increases risk) and ulcerative colitis (smoking is protective).

## Key Concepts

### Pathophysiology Overview Framework

**Category:** Gastrointestinal Pathology

**Key Process:** 
```
Dysregulated immune response → Gut microbiota interaction → Barrier defect → Genetic + environmental triggers → Chronic relapsing inflammation
```

**Mechanism/Steps:**
1. Dysregulated immune activation against gut microbiota
2. Defective intestinal barrier function
3. Genetic susceptibility (NOD2/CARD15, IL23R)
4. Environmental triggers (diet, smoking, medications)
5. Results in chronic relapsing inflammation throughout GI tract

**Purpose/Significance:** Provides the unifying pathophysiological framework that explains the shared clinical features of IBD while distinguishing underlying mechanisms
