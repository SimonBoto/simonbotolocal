# [[KEYWORD-Magnesium|magnesium]] Metabolic Cofactor ATP Function

## Overview
Magnesium is essential for ATP function and serves as cofactor for over 300 enzymes—particularly those involving phosphate transfer.

## ATP-Magnesium Complex

### Structure
- **Mg-ATP** is the true substrate, not free ATP
- Mg²⁺ coordinates with phosphate oxygens
- Stabilizes negative charges on ATP
- Essential for high-energy bond reactivity

### Why Mg²⁺?
| Property | Importance |
|----------|------------|
| **Charge density** | High charge/radius ratio |
| **Coordination** | Octahedral coordination with phosphate |
| **Abundance** | Most abundant divalent cation in cells |
| **Competition** | Outcompetes Ca²⁺ for ATP binding |

## Enzyme Classes Requiring Mg²⁺

### 1. Kinases
| Enzyme | Pathway |
|--------|---------|
| **Hexokinase** | Glycolysis initiation |
| **Phosphofructokinase** | Glycolysis rate-limiting step |
| **Pyruvate kinase** | Glycolysis completion |
| **Protein kinases** | Signal transduction |

### 2. Phosphatases
| Enzyme | Function |
|--------|----------|
| **ATPases** | Energy coupling |
| **Phosphatases** | Signal termination |

### 3. DNA/RNA Enzymes
| Enzyme | Function |
|--------|----------|
| **DNA polymerase** | DNA replication |
| **RNA polymerase** | Transcription |
| **Topoisomerases** | DNA structure |
| **Ribozymes** | RNA catalysis |

### 4. Other Mg-Dependent Enzymes
| Enzyme | Pathway |
|--------|---------|
| **Isocitrate dehydrogenase** | Krebs cycle |
| **Alpha-ketoglutarate dehydrogenase** | Krebs cycle |
| **Phospholipase C** | Signal transduction |
| **Adenylate cyclase** | cAMP synthesis |

## Clinical Relevance

### Hypomagnesemia Effects
| System | Symptoms |
|--------|----------|
| **Neuromuscular** | Tremor, tetany, seizures |
| **Cardiovascular** | Arrhythmias, hypertension |
| **Metabolic** | [[KEYWORD-Insulin|insulin]] resistance, hypokalemia |
| **Psychiatric** | Anxiety, depression, insomnia |

### Causes of Deficiency
- Poor dietary intake
- Chronic diarrhea
- Alcoholism
- Diuretic use
- PPI (proton pump inhibitor) use

### Assessment
- **Serum Mg:** Poor marker (only 1% of body Mg)
- **RBC Mg:** Better indicator of tissue status
- **Symptoms:** Often precede lab abnormalities
