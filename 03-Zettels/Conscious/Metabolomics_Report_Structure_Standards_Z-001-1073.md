## Core Claim
**Standardized metabolomics report formats — whether delivered live in consultation or via email — must follow a concise, visual, and actionable structure that prioritizes patient comprehension over technical completeness, using clear headings, color coding, and plain language to transform complex data into meaningful health insights.**

## Evidence

### Report Structure Framework
| Section | Content | Length | Purpose |
|---------|---------|--------|---------|
| **Executive Summary** | 3-5 key findings | 150 words | High-level overview |
| **The Metabolic Story** | Narrative explanation | 300 words | Contextualize findings |
| **Detailed Findings** | Category breakdown | 500 words | Specific abnormalities |
| **Recommendations** | Action plan | 300 words | Next steps |
| **Appendix** | Technical details | As needed | For reference |

### Live Report Delivery Protocol
| Element | Timing | Technique |
|---------|--------|-----------|
| **Opening hook** | 30 seconds | Connect to their concerns |
| **Key findings** | 2-3 minutes | Top 3 abnormalities |
| **Story** | 2-3 minutes | Metabolic narrative |
| **Action plan** | 2 minutes | Clear next steps |
| **Q&A** | 5-10 minutes | Address questions |
| **Summary** | 1 minute | Reiterate key points |

### Email Report Format Standards
| Section | Format | Content |
|---------|--------|---------|
| **Subject line** | Clear, specific | "Your metabolic health results - [Name]" |
| **Opening** | Warm, personal | Brief greeting |
| **Executive summary** | Bullet points | Top findings |
| **Visuals** | Charts, graphs | Color-coded markers |
| **Action items** | Numbered list | Specific steps |
| **Support contact** | Clear contact info | Questions welcome |

### Visual Standards
| Element | Color Code | Meaning |
|---------|------------|---------|
| **Normal** | Green | Within range |
| **Mild elevation** | Yellow | Monitor |
| **Moderate elevation** | Orange | Address |
| **Severe elevation** | Red | Priority |

### Language Guidelines
| Principle | Application | Examples |
|-----------|-------------|----------|
| **Plain English** | Avoid jargon | "Energy production" not "ATP synthesis" |
| **Active voice** | Direct statements | "We recommend..." not "It is recommended..." |
| **Patient-centered** | Focus on their life | "You may experience..." not "Patients experience..." |
| **Action-oriented** | Clear next steps | "Take these 3 steps" not "Further investigation suggested" |
| **Positive framing** | Opportunity mindset | "Your body needs support" not "Your body is broken" |

### Common Report Pitfalls
| Pitfall | Why It Fails | Solution |
|---------|--------------|----------|
| **Information overload** | Overwhelms patient | Prioritize top findings |
| **Technical jargon** | Creates confusion | Use analogies, plain language |
| **No clear action** | Paralysis by analysis | Always provide next steps |
| **Negative framing** | Demoralizes patient | Focus on optimization |
| **Lengthy appendices** | Unreadable | Optional reference material |

### Quality Checklist
- [x] Executive summary present
- [x] Visual elements included
- [x] Plain language throughout
- [x] Actionable recommendations
- [x] Contact information clear
- [x] Patient-centered tone

## Clinical Implication
- **Compliance:** Clear reports drive follow-through
- **Trust:** Professional formatting builds credibility
- **Efficiency:** Standard structure saves report creation time
- **Communication:** Bridge between data and patient understanding
- **METHAP relevance:** Research participant communication requires similar clarity — result reports that participants understand improve retention and engagement
