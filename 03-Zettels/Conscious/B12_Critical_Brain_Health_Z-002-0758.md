# Vitamin B12 Critical for Brain Health — Mechanism in APOE4 Carriers

## Core Claim
**Vitamin B12 is critical for brain health** — and meat is the only reliable dietary source. APOE4 carriers may have **increased B12 requirements** or **impaired B12 utilization**, making meat consumption essential for preventing cognitive decline.

## B12 Functions in Brain

### Neurological
| Function | Mechanism |
|----------|-----------|
| **Myelin synthesis** | Maintains nerve insulation |
| **Neurotransmission** | Synthesis of serotonin, dopamine |
| **DNA synthesis** | Neuronal repair and replication |
| **Homocysteine metabolism** | Prevents neurotoxicity |

### Deficiency Consequences
| System | Effect |
|--------|--------|
| **Cognitive** | Memory loss, confusion |
| **Neurological** | Peripheral neuropathy |
| **Psychiatric** | Depression, psychosis |
| **Structural** | Brain atrophy |

## B12 and APOE4

### Why APOE4 Carriers Need More B12
| Factor | Explanation |
|--------|-------------|
| **Impaired lipid metabolism** | May affect B12 transport |
| **Increased oxidative stress** | Higher B12 demand |
| **Methylation issues** | B12 critical for methylation |
| **Brain energy metabolism** | B12 supports mitochondrial function |

### Meat as B12 Source
| Source | B12 Bioavailability |
|--------|---------------------|
| **Meat (heme)** | High |
| **Fish** | High |
| **Eggs** | Moderate |
| **Dairy** | Moderate |
| **Plant foods** | None (unless fortified) |

## Clinical Implications

### For APOE4 Carriers
- **Ensure adequate B12 intake** — test levels regularly
- **Meat consumption** — most reliable source
- **Supplementation** — if not consuming meat
- **Monitor homocysteine** — marker of B12 status

### B12 Testing
| Marker | Target |
|--------|--------|
| **Serum B12** | >400 pg/mL |
| **Homocysteine** | <10 μmol/L |
| **Methylmalonic acid** | Normal range |
