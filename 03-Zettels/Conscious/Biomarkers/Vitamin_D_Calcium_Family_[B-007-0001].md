# Vitamin D - Calcium - PTH Axis - B-007-0001

## 1. IDENTIFICATION

| Marker | Full Name | What It Measures |
|--------|-----------|------------------|
| **25(OH)D** | 25-Hydroxyvitamin D | Vitamin D status (storage form) |
| **PTH** | Parathyroid Hormone | Calcium homeostasis regulator |
| **Calcium** | Serum Calcium | Mineral balance |
| **Ionized Calcium** | Free Calcium | Biologically active calcium |
| **Magnesium** | Serum Magnesium | Cofactor for PTH, vitamin D |
| **Phosphate** | Serum Phosphate | Bone mineral partner |

**Specimen:** Serum
**Collection:** No fasting required; morning preferred for PTH (diurnal variation)

## 2. SIMOS CLINICAL TARGETS

### 25(OH)Vitamin D - ng/mL (nmol/L)
| Range | Simos Category | Clinical Significance |
|-------|----------------|----------------------|
| **<30** | Deficient | Rickets/osteomalacia risk, immune dysfunction |
| **30-50** | Insufficient | Suboptimal for most functions |
| **>50** | Minimum acceptable | Basic sufficiency |
| **80-100** | **ACCEPTED** | Good status |
| **80-150** | **OPTIMAL** | **Ideal for metabolic, immune, bone health** |
| **>150** | High | Monitor, reduce dose if >200 |
| **>250** | Toxicity risk | Hypercalcemia possible |

**Simos Rules:**
- **Optimal: 80-150 ng/mL**
- **Accepted: 80-100 ng/mL**
- **Minimum: >50 ng/mL**
- **Target for most patients: 100 ng/mL**

**Why Higher Than IOM Recommendations:**
- IOM (50 ng/mL) based on bone health only
- Immune function requires 80-100+
- Metabolic benefits (insulin sensitivity) at 80-100+
- Cancer prevention data suggests 100-150
- Safety margin up to 150 well-established

### PTH (Parathyroid Hormone) - pg/mL
| Range | Simos Category | Interpretation |
|-------|----------------|----------------|
| **<20** | **IDEAL** | Optimal suppression, adequate D |
| **20-30** | **Good** | Well-suppressed |
| **30-40** | **ACCEPTED** | Borderline, watch trend |
| **>40** | **Elevated** | Insufficient D, hyperparathyroidism |
| **>65** | High | Primary or secondary hyperparathyroidism |

**Simos Rules:**
- **Ideal: <20 pg/mL**
- **Accepted: <40 pg/mL**
- **Target: <30 pg/mL**

**Why PTH Matters:**
- PTH rises when vitamin D insufficient
- Elevated PTH = bone resorption (osteoporosis risk)
- PTH suppression confirms adequate vitamin D
- Even with "normal" 25(OH)D, elevated PTH suggests functional deficiency

### Calcium - mg/dL (mmol/L)
| Range | Interpretation |
|-------|----------------|
| **<8.5** | Low (hypocalcemia) |
| **8.5-9.0** | Low-normal |
| **9.0-10.0** | **OPTIMAL** |
| **10.0-10.5** | High-normal |
| **>10.5** | High (hypercalcemia) |
| **>11.0** | Critical |

**Simos Rule: Calcium should be 9-10 mg/dL**

**Albumin Correction:**
- Calcium bound to albumin
- Low albumin → falsely low calcium
- Corrected Ca = Measured Ca + 0.8 × (4.0 - Albumin)

**Ionized Calcium (if available):**
- More accurate than total calcium
- Range: 4.5-5.3 mg/dL (1.15-1.33 mmol/L)
- Not affected by albumin

### Magnesium - mg/dL (mmol/L)
| Range | Interpretation | Note |
|-------|----------------|------|
| **<1.8** | Deficient | Common, often missed |
| **1.8-2.0** | Low-normal | Suboptimal |
| **2.0-2.5** | **OPTIMAL** | Ideal |
| **>2.5** | High | Rare, check renal function |

**Simos Rule: Magnesium should be 2.0-2.5 mg/dL**

**Why Magnesium is Critical:**
- Required for PTH secretion
- Required for vitamin D activation (25-hydroxylation and 1α-hydroxylation)
- Low Mg → functional vitamin D deficiency even with adequate 25(OH)D
- Low Mg → PTH resistance
- **Check and replete magnesium BEFORE or WITH vitamin D supplementation**

## 3. THE VITAMIN D-PTH-CALCIUM AXIS

### Physiology

**Vitamin D Synthesis:**
```
Skin (UVB) or Diet → Vitamin D3 → Liver (25-hydroxylase) → 25(OH)D → Kidney (1α-hydroxylase) → 1,25(OH)2D (active)
```

**Calcium Homeostasis:**
```
Low Calcium → PTH release → 
   - Bone resorption (releases Ca)
   - Kidney reabsorption of Ca
   - Kidney activation of vitamin D (increases Ca absorption)
   → Calcium normalized
```

**PTH Suppression by Vitamin D:**
- Adequate 25(OH)D (>50 ng/mL) suppresses PTH
- Optimal 25(OH)D (80-100+) suppresses PTH to <30
- PTH <20 = excellent vitamin D status

### Clinical Patterns

**Optimal Pattern:**
| Marker | Value |
|--------|-------|
| 25(OH)D | 80-150 ng/mL |
| PTH | <20 pg/mL |
| Calcium | 9-10 mg/dL |
| Magnesium | 2.0-2.5 mg/dL |

**Vitamin D Deficiency Pattern:**
| Marker | Value | Interpretation |
|--------|-------|----------------|
| 25(OH)D | <30 | Deficient |
| PTH | >40 | Compensatory elevation |
| Calcium | Normal (maintained) | PTH doing its job |
| Magnesium | Variable | Check and replete |

**Functional Vitamin D Deficiency (Low Magnesium):**
| Marker | Value | Interpretation |
|--------|-------|----------------|
| 25(OH)D | Normal (30-50) | Adequate storage |
| PTH | >40 | Not suppressed |
| Magnesium | <1.8 | Can't activate D or secrete PTH properly |
| Action | | Replete magnesium first |

**Primary Hyperparathyroidism:**
| Marker | Value | Interpretation |
|--------|-------|----------------|
| Calcium | >10.5 | Elevated |
| PTH | >65 or inappropriately normal | Autonomous secretion |
| 25(OH)D | Variable | Often low (consumed) |
| Action | | Refer to endocrinology |

**Secondary Hyperparathyroidism (Vitamin D deficiency):**
| Marker | Value | Interpretation |
|--------|-------|----------------|
| Calcium | Normal or low | |
| PTH | >40 | Elevated due to low D |
| 25(OH)D | <30 | Deficient |
| Action | | Replete vitamin D and magnesium |

## 4. CLINICAL APPLICATIONS

### Bone Health
- Vitamin D essential for calcium absorption
- PTH elevation causes bone resorption
- Target: 25(OH)D >50 (minimum), PTH <30
- Osteoporosis prevention and treatment

### Immune Function
- Vitamin D modulates innate and adaptive immunity
- Deficiency associated with:
  - Autoimmune diseases (MS, RA, IBD)
  - Infections (respiratory, viral)
  - Cancer risk
- Target: 25(OH)D 80-100+ for immune optimization

### Metabolic Health
- Vitamin D improves insulin sensitivity
- Deficiency associated with:
  - Insulin resistance
  - Type 2 diabetes
  - Metabolic syndrome
- Target: 25(OH)D 80-100 for metabolic benefits

### Cardiovascular Health
- Deficiency associated with:
  - Hypertension
  - Heart disease
  - Stroke
- Mechanisms: RAAS modulation, endothelial function, inflammation
- Target: 25(OH)D 80-100

### Mood and Neurological
- Vitamin D receptors in brain
- Deficiency associated with:
  - Depression
  - Cognitive decline
  - MS
- Target: 25(OH)D 80-100

### Pregnancy
- Critical for fetal bone development
- Deficiency associated with:
  - Preeclampsia
  - Gestational diabetes
  - Low birth weight
- Target: 25(OH)D >50 (minimum), ideally 80-100

## 5. SUPPLEMENTATION PROTOCOL

### Vitamin D3 (Cholecalciferol)

**Dosing Strategy:**
| Starting Level | Dose | Duration to Optimal |
|----------------|------|---------------------|
| <20 ng/mL | 10,000 IU daily | 8-12 weeks |
| 20-30 | 10,000 IU daily | 6-8 weeks |
| 30-50 | 5,000-10,000 IU | 4-6 weeks |
| 50-80 | 5,000 IU | 4 weeks |
| Maintenance | 2,000-5,000 IU | Ongoing |

**Simos Standard:**
- **Loading:** 10,000 IU daily until 80-100 ng/mL
- **Maintenance:** 5,000 IU daily (individual variation)
- **Some need:** 10,000 IU daily long-term (obesity, malabsorption)

**Form:**
- D3 (cholecalciferol) preferred
- D2 (ergocalciferol) less effective
- Oil-based capsules best absorbed
- Take with fat-containing meal

**Cofactors (CRITICAL):**
| Cofactor | Dose | Why |
|----------|------|-----|
| **Magnesium** | 400-600 mg | Required for D activation |
| **Vitamin K2 (MK-7)** | 100-200 mcg | Directs calcium to bone, not arteries |
| **Zinc** | 15-30 mg | Cofactor for D receptors |
| **Boron** | 3-6 mg | Extends D half-life |

### Magnesium (Must Replete)
**Why First:**
- Without magnesium, vitamin D can't be activated
- Without magnesium, PTH can't function
- Low magnesium = functional D deficiency

**Forms:**
- Magnesium glycinate (best absorbed, gentle)
- Magnesium citrate (good absorption, laxative)
- Magnesium oxide (poor absorption, avoid)

**Dose:** 400-600 mg elemental magnesium daily

### Monitoring
| Timeline | Check | Target |
|----------|-------|--------|
| Baseline | 25(OH)D, PTH, Calcium, Magnesium | Document |
| 8 weeks | 25(OH)D, Calcium | 80-100 ng/mL |
| 12 weeks | PTH | <30 pg/mL |
| 6 months | Complete panel | All optimal |
| Ongoing | 25(OH)D every 6 months | Maintain 80-100 |

### Safety Monitoring
- Calcium at 8-12 weeks (hypercalcemia rare but check)
- 25(OH)D >150 → reduce dose
- Calcium >10.5 → reduce dose, check PTH
- Symptoms of hypercalcemia: Nausea, constipation, confusion

| Source | Amount | Notes |
|--------|--------|-------|
| **Sun exposure** | 10,000-20,000 IU/20 min | Best source, variable by latitude, skin tone |
| **Cod liver oil** | 400-1,000 IU/tsp | Also provides A and omega-3 |

## 7. RED FLAGS

🚩 **Critical:**
- Calcium >11.0 mg/dL (hypercalcemia)
- 25(OH)D >250 ng/mL (toxicity risk)
- PTH >100 pg/mL (severe hyperparathyroidism)
- Symptomatic hypocalcemia (tetany, seizures)

⚠️ **Urgent:**
- Calcium >10.5 with elevated 25(OH)D (overdose)
- PTH >65 (hyperparathyroidism workup)
- 25(OH)D <20 with elevated PTH (severe deficiency)
