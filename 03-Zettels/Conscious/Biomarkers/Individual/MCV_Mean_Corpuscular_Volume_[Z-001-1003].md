# MCV — Mean Corpuscular Volume

## 1. ETYMOLOGY

**MCV:** Mean Corpuscular Volume

**Corpuscular:** From Latin "corpusculum" (small body)

**Also called:**
- Mean cell volume
- RBC size index

source-litnote: Synthesized from clinical practice
---

## 2. PHYSIOLOGY

### Biological Function

**Average size of red blood cells:**
- Calculated: Hct ÷ RBC count × 10
- Normal: 80-100 femtoliters (fL)

**Clinical utility:**
- Classifies anemia by RBC size
- Guides diagnostic workup

## 3. METABOLIC INTERPRETATION

### MCV Categories

| Category | MCV (fL) | Description |
|----------|----------|-------------|
| **Microcytic** | <80 | Small RBCs |
| **Normocytic** | 80-100 | Normal size |
| **Macrocytic** | >100 | Large RBCs |

### Causes by Category

**Microcytic (<80 fL):**
| Cause | Mechanism |
|-------|-----------|
| **Iron deficiency** | Most common; impaired heme synthesis |
| **Thalassemia** | Genetic hemoglobin defect |
| **Anemia of chronic disease** | Iron sequestration |
| **Sideroblastic anemia** | Iron utilization defect |
| **Lead poisoning** | Heme synthesis inhibition |

**Normocytic (80-100 fL):**
| Cause | Mechanism |
|-------|-----------|
| **Acute blood loss** | Normal RBCs lost |
| **Hemolysis** | Premature destruction |
| **Anemia of chronic disease** | Early/mixed |
| **Bone marrow failure** | Reduced production |
| **Chronic kidney disease** | EPO deficiency |

**Macrocytic (>100 fL):**
| Cause | Mechanism |
|-------|-----------|
| **B12 deficiency** | Impaired DNA synthesis |
| **Folate deficiency** | Impaired DNA synthesis |
| **Liver disease** | Lipid membrane changes |
| **Hypothyroidism** | Reduced metabolism |
| **Alcoholism** | Multiple mechanisms |
| **Myelodysplasia** | Bone marrow disorder |
| **Medications** | Methotrexate, phenytoin, AZT |

## 4. COFACTORS

### Factors Affecting RBC Size

| Factor | Effect |
|--------|--------|
| **Iron** | Deficiency → microcytic |
| **B12/Folate** | Deficiency → macrocytic |
| **DNA synthesis** | Impaired → macrocytic |
| **Heme synthesis** | Impaired → microcytic |
| **Thyroid** | Hypo → macrocytic |
| **Liver function** | Impaired → macrocytic |

## 5. INTERVENTIONS

### Based on MCV Classification

**Microcytic:**
- Iron studies (ferritin, TIBC, transferrin sat)
- Hemoglobin electrophoresis (thalassemia)
- Lead level

**Normocytic:**
- Reticulocyte count
- Hemolysis workup
- Chronic disease evaluation

**Macrocytic:**
- B12, folate levels
- Liver function tests
- TSH
- Alcohol history
- Medication review

## 6. CONNECTED BIOMARKERS

| Biomarker | Relationship | Pattern |
|-----------|--------------|---------|
| **MCH** | Hb per RBC | Low in microcytic |
| **MCHC** | Hb concentration | Low in microcytic |
| **RDW** | Size variation | High in mixed deficiencies |
| **Ferritin** | Iron stores | Low in iron deficiency |
| **B12/Folate** | DNA synthesis | Low in macrocytic |

### Clinical Patterns

**Iron Deficiency:**
- MCV: Low (<80)
- MCH: Low
- RDW: High
- Ferritin: Low

**B12 Deficiency:**
- MCV: High (>100)
- MCH: High
- RDW: Normal/high
- B12: Low

**Mixed Deficiency:**
- MCV: Normal (80-100)
- RDW: Very high
- **Iron + B12/folate deficiency**
