# WBC — White Blood Cell Count

## Overview
Total count of white blood cells (leukocytes) in circulation.

**See complete guide:** [[WBC_[B-001-0001].md|WBC Comprehensive Clinical Guide]]

## Quick Reference

| Level | Value | Interpretation |
|-------|-------|----------------|
| **Optimal** | 4.5-8.0 | Normal immune function |
| **Accepted** | 4.5-10.0 | Standard reference |
| **High** | >11 | Infection, inflammation, stress |
| **Low** | <3.5 | Immunosuppression, viral, deficiency |

## Clinical Context
- Always interpret with differential
- Trend matters more than single value
- Check absolute counts, not just percentages
