# Urine Organic Acid — Krebs Cycle Markers

> Elevations in citrate and aconitate indicate aconitase dysfunction, while 3-HMGA elevation suggests mevalonate pathway and fatty acid oxidation impairment.

## Pattern Recognition

| Marker | Indicates | Enzyme Step |
|--------|-----------|-------------|
| **Citrate ↑** | Block at citrate→isocitrate | Aconitase dysfunction |
| **Aconitate ↑** | Confirms Fe-S cluster problem | Aconitase intermediate |
| **3-HMGA ↑** | Mevalonate/fatty acid issues | HMG-CoA lyase, carnitine |

## Accumulation Cascade

```
Acetyl-CoA → Citrate (accumulates)
    ↓
Aconitate (accumulates) ← BLOCK
    ↓
Isocitrate → α-KG → Succinyl-CoA
```

##  Significance

- l energy production compromised
- Back-pressure affects upstream 
- Multiple pathway involvement (TCA + mevalonate)
