# METHAP Study Timeline Testing Schedule

## Overview
The METHAP trial follows a structured timeline with baseline assessment, 12-week intervention, and post-intervention retesting to measure metabolic changes.

## Study Timeline

### Phase 1: Screening and Baseline (Weeks -4 to 0)
| Week | Activity |
|------|----------|
| **-4 to -2** | Recruitment, screening, consent |
| **-1** | Baseline testing (all participants) |
| **0** | Randomization (IR group), intervention start |

### Phase 2: Intervention (Weeks 1-12)
| Week | Activity |
|------|----------|
| **1** | Education, dietitian consult, supplement initiation |
| **2-4** | Weekly check-ins, compliance monitoring |
| **6** | Mid-point assessment (symptoms, adherence) |
| **8-10** | Continued monitoring, dietary review |
| **12** | End of intervention, post-testing |

### Phase 3: Post-Intervention (Weeks 13-16)
| Week | Activity |
|------|----------|
| **13** | Post-intervention testing (IR groups) |
| **14-15** | Data entry, preliminary analysis |
| **16** | Results feedback to participants |

## Testing Schedule

### Baseline Testing (All Participants)
| Category | Tests |
|----------|-------|
| **Clinical** | FBC, HbA1c, fasting glucose, fasting insulin |
| **Lipids** | Total cholesterol, LDL, HDL, TG |
| **Metabolic** | HOMA-IR calculation, uric acid |
| **Thyroid** | TSH |
| **Iron** | Ferritin |
| **Calcium** | Serum calcium |
| **** | Urine organic acids (OAT), plasma total fatty acids (TFA) |

### Post-Intervention Testing (IR Groups Only)
| Category | Tests |
|----------|-------|
| **Clinical** | All baseline clinical markers repeated |
| **Metabolomics** | Urine OA, plasma TFA repeated |
| **Adverse events** | Safety monitoring |
| **Compliance** | Supplement count, dietary recall |

## Sample Collection Protocol

### Urine Organic Acids (OAT)
| Parameter | Specification |
|-----------|---------------|
| **Collection** | First morning urine |
| **Volume** | 10-20 mL |
| **Container** | Urine collection cup, preservative |
| **Storage** | Refrigerate immediately, freeze within 24h |
| **Analysis** | GC-MS/MS |

### Plasma Total Fatty Acids (TFA)
| Parameter | Specification |
|-----------|---------------|
| **Collection** | Fasting 12 hours |
| **Tube** | EDTA (lavender top) |
| **Volume** | 5-10 mL |
| **Processing** | Centrifuge within 2 hours |
| **Storage** | Plasma separated, -80°C |
| **Analysis** | Gas chromatography |

### Clinical Blood Tests
| Parameter | Specification |
|-----------|---------------|
| **Collection** | Fasting 8-12 hours |
| **Tube** | Serum (red top), EDTA |
| **Analysis** | Standard clinical chemistry |
