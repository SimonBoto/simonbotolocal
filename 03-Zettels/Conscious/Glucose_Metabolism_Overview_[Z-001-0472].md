# Glucose Metabolism Overview

## The Primary Fuel
**Glucose:** Most abundant monosaccharide, primary energy source for most organisms.

## Blood Glucose Regulation
- **Fasting:** 70-100 mg/dL (3.9-5.6 mmol/L)
- **Postprandial:** <140 mg/dL (<7.8 mmol/L)
- **Tight control:** Critical for brain function (no glycogen stores)

## Cellular Uptake

### Insulin-Dependent (GLUT4)
- Skeletal muscle
- Adipose tissue
- **Mechanism:** Insulin → GLUT4 translocation to membrane

### Insulin-Independent (GLUT1, GLUT2, GLUT3)
- Brain (GLUT3)
- RBCs (GLUT1)
- Liver/pancreas (GLUT2)

## Metabolic Fates

### 1. Glycolysis (All Cells)
```
Glucose → Pyruvate + 2 ATP + 2 NADH
```

### 2. Glycogen Synthesis (Liver, Muscle)
```
Glucose → G6P → Glycogen (storage)
```

### 3. Pentose Phosphate Pathway
```
G6P → NADPH + Ribose (biosynthesis/redox)
```

### 4. Lipogenesis (Liver, Adipose)
```
Glucose → Acetyl-CoA → Fatty acids
```

## Regulation
- **Insulin:** Promotes uptake, storage
- **Glucagon:** Promotes release (liver)
- **Epinephrine:** Emergency glucose mobilization
