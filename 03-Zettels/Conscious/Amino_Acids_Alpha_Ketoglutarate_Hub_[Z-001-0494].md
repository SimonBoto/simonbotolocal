#  to α-Ketoglutarate: The Glutamate Hub

## Four Amino Acids Converge on Glutamate

| Amino Acid | Pathway to Glutamate | Key Enzymes |
|------------|----------------------|-------------|
| **Glutamate** | Direct | Glutamate dehydrogenase |
| **Arginine** | → Ornithine → Glutamate-γ-semialdehyde → Glutamate | Arginase, P5C dehydrogenase |
| **Proline** | → Pyrroline-5-carboxylate (P5C) → Glutamate | Proline dehydrogenase, P5C dehydrogenase |
| **Histidine** | → Urocanate → Formiminoglutamate (FIGLU) → Glutamate | Histidase, urocanase, formiminotransferase |

## Glutamate ↔ α-Ketoglutarate

### Transamination (Reversible)
```
Glutamate + α-keto acid ↔ α-ketoglutarate + amino acid
```
**Enzyme:** Various transaminases (all B6-dependent)

### Oxidative Deamination
```
Glutamate + NAD⁺ + H₂O → α-ketoglutarate + NH₄⁺ + NADH
```
**Enzyme:** Glutamate dehydrogenase (GDH)

**Regulation:**
- Activated by: ADP, leucine
- Inhibited by: ATP, GTP, NADH

## Central Hub Function
**Glutamate receives amino groups from:**
- Most amino acids (via transamination)
- Then transfers to:
  - Urea cycle (for disposal)
  - Other α-keto acids (for amino acid synthesis)

## : Vitamin B6
**All four pathways B6-dependent:**
- Transaminases (all)
- Histidase
- **Deficiency:** Impaired  of all four amino acids
