# Metabolic Cluster as Insulin Resistance Manifestations

## Core Claim
**Hypertension + high triglycerides + low HDL + sdLDL + obesity** are not separate risk factors but **manifestations of insulin resistance** — a unified metabolic problem.

## The Cluster

| Marker | Mechanism |
|--------|-----------|
| **Hypertension** | Insulin-mediated sodium retention |
| **High TG** | VLDL overproduction |
| **Low HDL** | Impaired reverse transport |
| **sdLDL** | Insulin-driven pattern B |
| **Obesity** | Insulin-resistant adipose |

## Unified Approach
| Instead of | Address |
|------------|---------|
| Treating each separately | Root insulin resistance |
