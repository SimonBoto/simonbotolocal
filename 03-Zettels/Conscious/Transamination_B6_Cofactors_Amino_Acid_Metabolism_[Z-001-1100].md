# Transamination B6 Cofactors Amino Acid 

## Overview
Transamination is the process of transferring amino groups between amino acids and alpha-keto acids—requiring vitamin B6 (pyridoxal phosphate, PLP) as essential cofactor.

## The Transamination Reaction

### Basic Equation
```
Amino Acid₁ + Alpha-keto acid₂ ↔ Alpha-keto acid₁ + Amino Acid₂

Example:
Alanine + Alpha-ketoglutarate ↔ Pyruvate + Glutamate
```

### 
1. **Substrate binding:** Amino acid binds to enzyme-PLP complex
2. **Amino group transfer:** Amino group moves to PLP (forms pyridoxamine phosphate)
3. **Keto acid release:** First alpha-keto acid released
4. **Second substrate:** New keto acid binds
5. **Reverse transfer:** Amino group moves to new keto acid
6. **Product release:** New amino acid released

## Vitamin B6 (PLP) as Cofactor

### PLP Structure
- **Active form:** Pyridoxal phosphate (PLP)
- **Derived from:** Pyridoxine, pyridoxal, pyridoxamine
- **Function:** Electron sink—stabilizes reaction intermediates

### Why PLP is Essential
| Property | Function |
|----------|----------|
| **Aldehyde group** | Forms Schiff base with amino acid |
| **Electron sink** | Stabilizes carbanion intermediates |
| **Versatility** | Catalyzes multiple reaction types |

### B6-Dependent Reactions
| Reaction Type | Examples |
|---------------|----------|
| **Transamination** | Most common |
| **Decarboxylation** | Neurotransmitter synthesis |
| **Deamination** | Serine, threonine |
| **Racemization** | D-amino acid synthesis |
| **Cleavage** | Tryptophan kynurenine pathway |

## Complete Cofactor Requirements

### For Amino Acid Catabolism
| Cofactor | Enzyme/Function |
|----------|-----------------|
| **B6 (PLP)** | Transaminases, decarboxylases |
| **B12** | Methylmalonyl-CoA mutase (branched-chain AAs) |
| **B2** | Oxidative deaminases (D-amino acid oxidase) |
| **B3** | Multiple dehydrogenases |
| **B5** | Acyl-CoA synthesis |
| **Iron** | Phenylalanine hydroxylase, tyrosine hydroxylase |

### B-Vitamin Interdependence
- **B6 deficiency:** Impairs all transamination
- **B12 deficiency:** Blocks odd-chain fatty acid pathway
- **Multiple deficiencies:** Combined effects worse than single

## Clinical Deficiency States

### Vitamin B6 Deficiency
| Effect | Mechanism |
|--------|-----------|
| **Elevated glycine** | Impaired glycine cleavage |
| **Elevated xanthurenate** | Impaired tryptophan metabolism |
| **Neurological symptoms** | Impaired neurotransmitter synthesis |
| **Sideroblastic anemia** | Impaired heme synthesis |

### Elevated OAT Markers
| Marker | Deficiency |
|--------|------------|
| **Kynurenate ↑** | B6 |
| **Xanthurenate ↑** | B6 |
| **Glycine ↑** | B6 |
| **Methylmalonate ↑** | B12 |

## Drug Interactions

### B6 Depletors
| Drug | Effect |
|------|--------|
| **Isoniazid** | B6 antagonist (TB treatment) |
| **Penicillamine** | B6 chelation |
| **Oral contraceptives** | Increased B6 requirement |
| **Hydralazine** | B6 antagonist |
