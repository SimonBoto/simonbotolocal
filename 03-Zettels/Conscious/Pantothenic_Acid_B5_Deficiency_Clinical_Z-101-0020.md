# Pantothenic Acid (B5) Deficiency Clinical Aspects

## Core Claim
**Pantothenic acid (vitamin B5) deficiency is rare due to widespread availability in foods, but when present causes non-specific symptoms including fatigue, irritability, paresthesias, and impaired wound healing — with severe deficiency potentially causing adrenal insufficiency and impaired fatty acid oxidation — while the related disorder PKAN (pantothenate kinase-associated neurodegeneration) represents a genetic defect in CoA synthesis causing severe neurological deterioration and brain iron accumulation.**

## Evidence

### B5 Functions
| Function | Mechanism |
|----------|-----------|
| **CoA synthesis** | Precursor for coenzyme A |
| **Fatty acid metabolism** | Acyl group transfer |
| **Krebs cycle** | Acetyl-CoA, succinyl-CoA |
| **Steroid synthesis** | Cholesterol, hormones |
| **Neurotransmitter synthesis** | Acetylcholine |

### Dietary Sources
| Source | B5 Content |
|--------|-----------|
| **Liver** | Very high |
| **Yeast** | Very high |
| **Egg yolk** | High |
| **Legumes** | Moderate |
| **Whole grains** | Moderate |
| **Meat** | Moderate |

### Deficiency Manifestations
| System | Symptoms |
|--------|----------|
| **General** | Fatigue, malaise |
| **Neurological** | Paresthesias, burning feet |
| **GI** | Nausea, abdominal cramps |
| **Skin** | Impaired wound healing |
| **Adrenal** | Insufficiency (severe deficiency) |

### PKAN (Pantothenate Kinase-Associated Neurodegeneration)
| Feature | Description |
|--------|-------------|
| **Gene** | PANK2 |
| **Defect** | Pantothenate kinase dysfunction |
| **Result** | Impaired CoA synthesis, brain iron accumulation |
| **Onset** | Childhood (classic) or adult (atypical) |
| **Features** | Dystonia, parkinsonism, optic atrophy |
| **MRI** | "Eye of the tiger" sign |

## Clinical Implication
- **Rare deficiency:** Due to widespread availability
- **Non-specific symptoms:** Difficult to diagnose
- **No toxicity:** Water-soluble, excess excreted
- **PKAN severity:** Genetic defect, not dietary deficiency
- **METHAP relevance:** B5 status affects CoA availability and metabolic capacity
