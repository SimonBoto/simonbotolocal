# Trials.gov Compliance Checklist

## Pre-Registration Requirements

### IRB/Ethics Approval
- [ ] Final IRB approval obtained
- [ ] Approval letter available
- [ ] IRB registration number documented
- [ ] Ethics committee contact information confirmed

### Study Documentation
- [ ] Protocol finalized
- [ ] Informed consent form approved
- [ ] Statistical analysis plan complete
- [ ] Case report forms designed

### Administrative Setup
- [ ] ClinicalTrials.gov account created
- [ ] Organization profile verified
- [ ] Principal Investigator credentials confirmed
- [ ] Sponsor information documented

## Registration Quality Check

### Section 1-3: Identification and Status
- [ ] Official title <600 characters
- [ ] Brief title <300 characters
- [ ] Protocol ID unique and consistent
- [ ] Dates realistic and achievable

### Section 4-5: Oversight and Description
- [ ] IRB information complete
- [ ] Brief summary understandable to layperson
- [ ] Detailed description scientifically accurate
- [ ] No promotional language

### Section 6-8: Design and Interventions
- [ ] All conditions from MeSH/ICD-10
- [ ] Keywords comprehensive
- [ ] Study design internally consistent
- [ ] Arms clearly differentiated
- [ ] Interventions fully specified

### Section 9: Outcomes
- [ ] Primary outcome ≤2 (recommended)
- [ ] All outcomes have timeframes
- [ ] Measures are specific and objective
- [ ] Endpoints clearly defined

### Section 10: Eligibility
- [ ] Age range specified
- [ ] Sex/gender clearly stated
- [ ] Inclusion criteria exhaustive
- [ ] Exclusion criteria justified

### Section 11-14: Contacts and Administration
- [ ] Contact information current
- [ ] Facility address complete
- [ ] IPD statement compliant with journal requirements
- [ ] Responsible party clearly identified

## Common Pitfalls to Avoid

### Content Issues
| Pitfall | Solution |
|---------|----------|
| Vague outcome measures | Specify exact measurement + unit |
| Missing timeframes | Add specific timepoints for all outcomes |
| Inconsistent design | Verify allocation matches masking |
| Promotional language | Use neutral, scientific phrasing |

### Technical Issues
| Pitfall | Solution |
|---------|----------|
| Duplicate registration | Check if trial already registered |
| Wrong study type | Verify interventional vs. observational |
| Incomplete locations | Add all participating sites |
| Outdated contacts | Verify before submission |

## Post-Registration Obligations

### Updates Required
| Trigger | Action | Timeline |
|---------|--------|----------|
| Recruitment status change | Update status | Within 30 days |
| Protocol amendment | Update description | Within 30 days |
| Completion | Post results | Within 12 months |
| Publication | Add reference | Within 30 days |

### Results Reporting
- Required within 12 months of primary completion
- Includes participant flow, demographics, outcomes
- Adverse events must be reported
- Statistical analysis results

- [ ] Funding sources disclosed
- [ ] Commercial relationships stated

- [ ] No undisclosed conflicts
