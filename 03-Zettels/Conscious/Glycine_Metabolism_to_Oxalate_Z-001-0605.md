## Pathway Overview

**Glycine is a major endogenous source of oxalate:**
```
Glycine → Glyoxylate → Oxalate
```

## Step 1: Glycine to Glyoxylate

**Enzyme:** Glycine dehydrogenase (decarboxylating)  
**Cofactor:** Pyridoxal phosphate (B6)  
**Location:** 

```
Glycine + NAD⁺ → Glyoxylate + NH₄⁺ + NADH
```

## Step 2: Glyoxylate to Oxalate

**Enzyme:** Lactate dehydrogenase (minor) or spontaneous oxidation  
**Alternative:** Glyoxylate reductase (converts back to glycolate)

```
Glyoxylate → Oxalate (oxidation)
```

## Regulation

**Fates of glyoxylate:**
1. **→ Oxalate** (irreversible, excreted)
2. **→ Glycolate** (glyoxylate reductase)
3. **→ Glycine** (transamination)
4. **→ Formate** (decarboxylation)

**Balance determines oxalate production.**

##  Relevance

**Primary hyperoxaluria Type 1 (PH1):**
- AGXT gene defect (alanine-glyoxylate aminotransferase)
- Impaired glyoxylate → glycine conversion
- Glyoxylate shunted to oxalate
- Severe oxalate overproduction

**High glycine intake:**
- Collagen supplements
- Glycine powders
- May increase oxalate burden

## Quantitative Significance

**Endogenous oxalate production:**
- ~50% from glycine metabolism
- ~10-20% from hydroxyproline
- ~10-50% from vitamin C (variable)
