# Amino Acids to α-Ketoglutarate — The Glutamate Hub

## Core Claim
**Four amino acids (glutamate, arginine, proline, histidine) converge on the glutamate hub before entering the Krebs cycle as α-ketoglutarate — glutamate is the central collection point for amino groups, and its conversion to α-ketoglutarate (via transamination or oxidative deamination) links amino acid metabolism to energy production** — all transaminations require B6.

## Evidence

### Convergence Pathways
| Amino Acid | Pathway to Glutamate | Key Enzymes |
|------------|----------------------|-------------|
| **Glutamate** | Direct entry | Glutamate dehydrogenase |
| **Arginine** | → Ornithine → Glutamate-γ-semialdehyde → Glutamate | Arginase, P5C dehydrogenase |
| **Proline** | → Pyrroline-5-carboxylate → Glutamate | Proline dehydrogenase, P5C dehydrogenase |
| **Histidine** | → Urocanate → Formiminoglutamate → Glutamate | Histidase, urocanase, formiminotransferase |

### Glutamate ↔ α-Ketoglutarate
| Reaction | Mechanism |
|----------|-----------|
| **Transamination** | Glutamate + α-keto acid ↔ α-KG + amino acid (B6-dependent) |
| **Oxidative deamination** | Glutamate + NAD⁺ + H₂O → α-KG + NH₄⁺ + NADH |

### Krebs Cycle Entry
```
α-Ketoglutarate → Succinyl-CoA → ... → Oxaloacetate
```
- **ATP yield:** ~12 ATP per α-KG
- **Glucogenic:** Yes (via OAA)

## Clinical Implication
- **Glutamate hub:** Central to nitrogen metabolism
- **B6 deficiency:** Impairs all transaminations
- **Hyperammonemia:** Impaired glutamate → α-KG conversion
- **METHAP relevance:** Glutamate/α-KG balance reflects metabolic state; B6 supports this critical hub
