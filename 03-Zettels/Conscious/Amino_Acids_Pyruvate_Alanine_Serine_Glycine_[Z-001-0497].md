# Amino Acids to Pyruvate: Alanine and Serine-Glycine

## Six Amino Acids → Pyruvate

| Amino Acid | Conversion | Key Enzyme | Notes |
|------------|------------|------------|-------|
| **Alanine** | Direct transamination | Alanine transaminase (ALT) | Glucose-alanine cycle |
| **Cysteine** | → Pyruvate + H₂S | Cysteine desulfurase | Sulfur metabolism |
| **Serine** | → Glycine → Pyruvate | Serine hydroxymethyltransferase (SHMT) | Folate-dependent |
| **Glycine** | → Serine → Pyruvate | SHMT (reverse) | Folate-dependent |
| **Threonine** | → Glycine → Pyruvate | Threonine aldolase | Split pathway |
| **Tryptophan** | → Alanine → Pyruvate | Multiple (kynurenine pathway) | Also ketogenic |

## Alanine and the Glucose-Alanine Cycle

### The Cycle
**Muscle → Liver:**
```
Muscle: Pyruvate + Glutamate → Alanine + α-ketoglutarate
Alanine transported in blood
Liver: Alanine + α-ketoglutarate → Pyruvate + Glutamate
Pyruvate → Glucose (gluconeogenesis)
Glucose → Muscle (via blood)
```

**Purpose:**
- Transport nitrogen from muscle to liver (as alanine)
- Carbon skeleton becomes glucose
- Maintains blood glucose during fasting

### Enzyme: ALT
**Alanine aminotransferase:**
- **Clinical marker:** Liver function
- **B6-dependent**
- Elevated in liver damage

## Serine-Glycine Interconversion

### The Folate Connection
```
Serine + Tetrahydrofolate (THF) ↔ Glycine + 5,10-methylene-THF
```
**Enzyme:** Serine hydroxymethyltransferase (SHMT)
**Cofactor:** Pyridoxal phosphate (B6)
**Folate carrier:** Tetrahydrofolate (THF)

### Significance
- **One-carbon metabolism:** 5,10-methylene-THF donates methyl groups
- **Nucleotide synthesis:** Provides carbons for purines, thymidylate
- **Folate deficiency:** Impaired serine/glycine metabolism

## Clinical: Folate Deficiency
**Consequences:**
- Megaloblastic anemia (impaired DNA synthesis)
- Neural tube defects (pregnancy)
- Elevated homocysteine

**Sources:**
- Leafy greens
- Fortified grains
- Supplementation
