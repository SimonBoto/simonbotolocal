# Clinic Patient Flow Operations

## Overview
Standard operating procedures for each stage of the  patient journey.

## Stage 1: Assessment (Intake)

### SOP
1. **Schedule appointment** — Allow 60 minutes
2. **Send intake forms** — 48 hours prior
3. **Review history** — Metabolic story focus
4. **Identify goals** — Patient priorities

### Quality Checkpoint
- [ ] Chief complaint documented
- [ ] Previous labs reviewed
- [ ] Current supplements/medications listed
- [ ] Dietary pattern assessed

## Stage 2: Sample Collection

### SOP
1. **Order OAT kit** — Patient or in-office collection
2. **Fatty acid test** — Separate collection if indicated
3. **Instructions provided** — Fasting, timing, storage
4. **Track sample** — Chain of custody

### Quality Checkpoint
- [ ] Patient understands collection protocol
- [ ] Kit expiration date verified
- [ ] Return shipping arranged
- [ ] Lab notified of incoming sample

## Stage 3: Analysis

### SOP
1. **Receive sample** — Log in tracking system
2. **Process OAT** — GC-MS/MS per protocol
3. **Process ** — GC analysis
4. **QC review** — Internal standards valid

### Quality Checkpoint
- [ ] Sample integrity verified
- [ ] QC within acceptable range
- [ ] Results validated by second reviewer

## Stage 4: Reporting

### SOP
1. **Interpret results** — Pattern recognition
2. **Create report** — Standardized template
3. **Schedule review** — 30-minute appointment
4. **Prepare recommendations** — Tiered protocol

### Quality Checkpoint
- [ ] All abnormal markers addressed
- [ ] Pattern interpretation documented
- [ ] Recommendations evidence-based
- [ ] Report proofread

## Stage 5: Treatment Initiation

### SOP
1. **Present findings** — Patient education
2. **Prescribe protocol** — Tiered hierarchy
3. **Provide supplements** — Or prescriptions
4. **Document plan** — In patient record

### Quality Checkpoint
- [ ] Patient understands 
- [ ] Drug interactions checked
- [ ] Cost discussed
- [ ] Timeline established

## Stage 6: Follow-Up

### SOP
1. **Schedule follow-up** — 4 weeks initial
2. **Check-in calls** — Week 1, 2
3. **Assess compliance** — At each contact
4. **Adjust protocol** — Based on response

### Quality Checkpoint
- [ ] Follow-up appointment scheduled
- [ ] Patient knows how to reach clinic
- [ ] Side effects assessed
- [ ] Progress documented
