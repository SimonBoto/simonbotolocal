# Clinic Patient Flow Operations

Six-stage patient flow operationalized: Assessment → Sample → Analysis → Reporting → Treatment → Follow-up. Each stage has standard operating procedures (SOPs) and quality checkpoints.

## Evidence
> "Consistent patient flow ensures no one falls through cracks and every patient gets the same high-quality experience." (Clinical operations)

## Stage 1: Assessment

### SOP
1. **Intake form completion** — Patient completes metabolic history
2. **Metabolic story documentation** — Clinician captures narrative
3. **Symptom prioritization** — Patient ranks top 3 concerns
4. **Goal setting** — Patient defines success

### Quality Check
- [ ] All intake fields complete
- [ ] Metabolic story documented in EMR
- [ ] Patient goals recorded
- [ ] Consent for testing obtained

### Time Target
- **Initial visit:** 45-60 minutes
- **Follow-up:** 30 minutes

## Stage 2: Sample Collection

### SOP
1. **OAT kit provided** — Instructions reviewed
2. **Fatty acid test ordered** — If indicated
3. **Additional labs** — As clinically indicated
4. **Collection timing** — First morning urine for OAT

### Quality Check
- [ ] Patient demonstrates understanding
- [ ] Collection instructions written
- [ ] Lab requisition complete
- [ ] Follow-up scheduled for results

### Pre-Analytical Variables
| Variable | Control |
|----------|---------|
| **Collection time** | First morning, fasting |
| **Medications** | Document supplements |
| **Diet** | Note unusual intake |
| **Exercise** | Avoid strenuous day before |

## Stage 3: Analysis

### SOP
1. **Sample receipt** — Log in tracking system
2. **Chain of custody** — Document handling
3. **Lab processing** — Per protocol
4. **QC review** — Internal controls within range

### Quality Check
- [ ] Sample integrity verified
- [ ] No hemolysis/lipemia
- [ ] QC passed
- [ ] Results within reportable range

### Turnaround Time
- **Standard:** 5-7 business days
- **Stat:** 2-3 business days (if available)

## Stage 4: Reporting

### SOP
1. **Results review** — Clinician interprets before patient contact
2. **Report generation** — Standardized format
3. **Patient scheduling** — Results appointment
4. **Preparation** — Review patient history

### Quality Check
- [ ] All markers reviewed
- [ ] Pattern recognition complete
- [ ] Clinical correlation made
- [ ] Treatment plan drafted

### Report Components
| Section | Content |
|---------|---------|
| **Summary** | One-sentence metabolic story |
| **Findings** | Traffic light visualization |
| **Interpretation** | Pattern explanation |
| **Plan** | Specific recommendations |
| **Timeline** | Follow-up schedule |

## Stage 5: Treatment

### SOP
1. **Results presentation** — 30-minute appointment
2. **Metabolic storytelling** — Explain in patient language
3. **Treatment discussion** — Options and rationale
4. **Prescribing** — Tiered protocol
5. **Education** — Supplements, diet, lifestyle

### Quality Check
- [ ] Patient understands findings
- [ ] Questions answered
- [ ] Written plan provided
- [ ] Supplement list generated
- [ ] Follow-up scheduled

### Tiered Prescribing
| Tier | Timing | Focus |
|------|--------|-------|
| **1** | Weeks 1-4 | Foundation (3-5 supplements) |
| **2** | Weeks 5-8 | Add based on response |
| **3** | Weeks 9-12 | Optimization |

## Stage 6: Follow-Up

### SOP
1. **Scheduled check-ins** — 4-8-16-28 week rhythm
2. **Progress assessment** — Symptom tracking
3. **Adherence review** — Supplement compliance
4. **Plan adjustment** — Based on response

### Quality Check
- [ ] Symptoms tracked
- [ ] Adherence documented
- [ ] Side effects screened
- [ ] Plan adjusted if needed

### Follow-Up Schedule
| Visit | Timing | Focus |
|-------|--------|-------|
| **1** | 4 weeks | Tolerance, early response |
| **2** | 8 weeks | Adherence, symptom change |
| **3** | 12-16 weeks | Retest OAT, assess progress |
| **4** | 6 months | Maintenance, optimization |

## Quality Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| **Pre-analytical error rate** | < 2% | Sample rejection tracking |
| **Report turnaround** | < 5 days | Time from receipt to report |
| **Patient satisfaction** | > 90% | Post-visit survey |
| **Clinical outcome tracking** | 100% | Systematic at 12 weeks |

## Failure Points & Prevention

| Stage | Common Failure | Prevention |
|-------|----------------|------------|
| **Assessment** | Incomplete history | Structured intake form |
| **Sample** | Poor collection | Detailed instructions |
| **Analysis** | QC failure | Equipment maintenance |
| **Reporting** | Delayed results | Workflow tracking |
| **Treatment** | Patient confusion | Written plan, teach-back |
| **Follow-up** | Lost to follow-up | Scheduled appointments |
