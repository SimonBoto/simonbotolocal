> Clinical decisions about protein intake in CKD should be based on carbohydrate context, leading to different recommendations in different metabolic contexts.

## Core Concept

**Key Process:**
```
Clinical decisions based on carb context → Different protein recommendations
```

## Mechanism

| Context | Protein Recommendation | Monitoring |
|---------|----------------------|------------|
| Standard diet | 0.6-0.8 g/kg (restriction) | Standard CKD monitoring |
| Low-carb diet | 0.8-1.0 g/kg (moderate) | GFR, proteinuria, 3-6 months |
| Transitioning | Individualized | Close monitoring |

## Clinical Significance

- Assessment of carbohydrate context precedes protein recommendations
- Patient education on context-specific benefits/risks
- Practical guidelines for real-world application
