# Amino Acids to Fumarate — Phenylalanine and Tyrosine

## Core Claim
**Phenylalanine and tyrosine are both glucogenic and ketogenic — phenylalanine is hydroxylated to tyrosine (requiring BH4 and iron), then tyrosine is catabolized through a multi-step pathway yielding fumarate (glucogenic) and acetoacetate (ketogenic)** — phenylketonuria (PKU) results from phenylalanine hydroxylase deficiency.

## Evidence

### Phenylalanine → Tyrosine
```
Phenylalanine + O₂ + BH₄ → Tyrosine + BH₂ + H₂O
```
| Feature | Detail |
|---------|--------|
| **Enzyme** | Phenylalanine hydroxylase (PAH) |
| **Cofactors** | Tetrahydrobiopterin (BH₄), Iron (Fe²⁺) |
| **Irreversible** | Committed step |

### Tyrosine → Fumarate + Acetoacetate
```
Tyrosine → p-hydroxyphenylpyruvate → Homogentisate → Maleylacetoacetate →
Fumarylacetoacetate → Fumarate + Acetoacetate
```

### Dual Classification
| Product | Classification |
|---------|---------------|
| **Fumarate** | Glucogenic (Krebs cycle → OAA → glucose) |
| **Acetoacetate** | Ketogenic (ketone body) |

### PKU (Phenylketonuria)
| Feature | Detail |
|---------|--------|
| **Defect** | Phenylalanine hydroxylase deficiency |
| **Accumulation** | Phenylalanine (neurotoxic) |
| **Treatment** | Low-phenylalanine diet |
| **Screening** | Newborn screening (Guthrie test) |

## Clinical Implication
- **Both glucogenic and ketogenic:** Rare dual classification
- **PKU:** Most common inborn error of amino acid metabolism
- **BH4 cofactor:** Essential for phenylalanine metabolism
- **METHAP relevance:** Tyrosine is precursor for catecholamines and melanin; PKU requires lifelong dietary management
