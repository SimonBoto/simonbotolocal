# Fatty Acid Nomenclature and Numbering

## Core Claim
**Fatty acids are named using systematic conventions that specify chain length, number and position of double bonds — with the delta (Δ) system counting from the carboxyl end and the omega (ω) system counting from the methyl end — enabling precise communication of structure and metabolic properties across biochemical and clinical contexts.**

## Evidence

### Nomenclature Format
| Component | Meaning | Example |
|-----------|---------|---------|
| **C number** | Total carbons | C18 = 18 carbons |
| **Colon number** | Double bonds | :2 = 2 double bonds |
| **Position** | Double bond location | Δ9 = bond at carbon 9 |
| **Configuration** | Cis or trans | c = cis, t = trans |

### Delta (Δ) System
| Feature | Description |
|---------|-------------|
| **Counting from** | Carboxyl carbon (C1) |
| **Notation** | Δ followed by carbon number |
| **Example** | C18:2Δ9,12 (linoleic acid) |
| **Use** | Biochemical reactions, enzyme specificity |

### Omega (ω) System
| Feature | Description |
|---------|-------------|
| **Counting from** | Methyl carbon (ω carbon) |
| **Notation** | ω followed by position of first double bond |
| **Example** | C18:2ω6 (linoleic acid) |
| **Use** | Nutritional classification, dietary recommendations |

### Common Fatty Acids — Both Systems
| Common Name | Delta | Omega | Structure |
|-------------|-------|-------|-----------|
| **Oleic acid** | C18:1Δ9 | C18:1ω9 | 18C, 1 double bond at C9 |
| **Linoleic acid** | C18:2Δ9,12 | C18:2ω6 | 18C, 2 double bonds |
| **Alpha-linolenic acid** | C18:3Δ9,12,15 | C18:3ω3 | 18C, 3 double bonds |
| **Arachidonic acid** | C20:4Δ5,8,11,14 | C20:4ω6 | 20C, 4 double bonds |
| **EPA** | C20:5Δ5,8,11,14,17 | C20:5ω3 | 20C, 5 double bonds |
| **DHA** | C22:6Δ4,7,10,13,16,19 | C22:6ω3 | 22C, 6 double bonds |

### Shorthand Notation
| Format | Example | Meaning |
|--------|---------|---------|
| **18:0** | Stearic acid | 18 carbons, 0 double bonds |
| **18:1ω9** | Oleic acid | 18C, 1 double bond at ω9 |
| **18:2Δ9,12** | Linoleic acid | 18C, double bonds at C9 and C12 |

## Clinical Implication
- **Precision communication:** Unambiguous structural description
- **Metabolic prediction:** Structure determines function
- **Dietary tracking:** ω system used in nutrition
- **Enzyme specificity:** Δ system explains biochemical behavior
- **METHAP relevance:** Accurate nomenclature essential for fatty acid analysis and metabolic interpretation
