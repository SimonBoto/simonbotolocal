# Amino Acids to Pyruvate — Alanine, Serine, Glycine

## Core Claim
**Six amino acids (alanine, cysteine, serine, glycine, threonine, tryptophan) are catabolized to pyruvate — alanine undergoes direct transamination, while serine and glycine are interconverted via folate-dependent reactions before pyruvate formation** — the glucose-alanine cycle shuttles nitrogen from muscle to liver using alanine as carrier.

## Evidence

### Six Amino Acids → Pyruvate
| Amino Acid | Conversion | Key Enzyme | Notes |
|------------|------------|------------|-------|
| **Alanine** | Direct transamination | Alanine transaminase (ALT) | Glucose-alanine cycle |
| **Cysteine** | → Pyruvate + H₂S | Cysteine desulfurase | Sulfur metabolism |
| **Serine** | → Glycine → Pyruvate | Serine hydroxymethyltransferase (SHMT) | Folate-dependent |
| **Glycine** | → Serine → Pyruvate | SHMT (reverse) | Folate-dependent |
| **Threonine** | → Glycine → Pyruvate | Threonine aldolase | Split pathway |
| **Tryptophan** | → Alanine → Pyruvate | Kynurenine pathway | Also ketogenic |

### Glucose-Alanine Cycle
| Step | Location | Reaction |
|------|----------|----------|
| **Muscle** | Pyruvate + Glutamate → Alanine + α-KG | Transamination |
| **Transport** | Alanine in blood | Nitrogen carrier |
| **Liver** | Alanine + α-KG → Pyruvate + Glutamate | Transamination |
| **Liver** | Pyruvate → Glucose | Gluconeogenesis |

### Serine-Glycine-Folate Interconnection
```
Serine + THF ↔ Glycine + 5,10-methylene-THF (SHMT)
```
- **Folate required:** For interconversion
- **One-carbon metabolism:** Critical for methylation

## Clinical Implication
- **ALT marker:** Alanine transaminase in liver function tests
- **Folate deficiency:** Impairs serine-glycine metabolism
- **Glucose-alanine cycle:** Critical for nitrogen transport
- **METHAP relevance:** Folate and B6 support these pathways; high protein intake increases demand
