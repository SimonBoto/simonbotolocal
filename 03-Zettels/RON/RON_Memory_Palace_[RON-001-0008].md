# RON Memory Palace — The Vault as Mind Architecture

> "The method of loci is a mnemonic strategy that uses visual imagination and spatial memory to organize and recall information."  
> — Ancient tradition, from Simonides to Cicero

---

## The Method

**Simonides of Ceos** (6th century BCE) discovered it:
- A banquet hall collapsed
- He recalled who died by remembering where each sat
- **The insight:** Spatial memory is powerful

**Cicero** (55 BCE) codified it in *De Oratore*:
- Orators memorized speeches by mentally walking through buildings
- Each room = one section of the speech
- Each location = one key point

**The principle:**
1. Choose a familiar space (palace, house, route)
2. Place vivid, striking images at specific locations
3. To recall: mentally walk the space
4. The images trigger the information

**Why it works:**
- Humans are spatial navigators
- We remember places better than abstract lists
- Vivid imagery creates emotional hooks
- The journey provides sequence
