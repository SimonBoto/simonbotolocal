# Article 16: Minimal Exercise Dose for Mortality Reduction

## Key Findings

### Minimum Effective Dose
| Activity Duration | Outcome |
|-------------------|---------|
| **10 min/day** | Significant reduction in all-cause mortality |
| **15 min/day** | Significant reduction in CVD incidence |
| **15-20 min/week vigorous** | 16-40% lower mortality |
| 250 min/week (MVPA) | Optimal target |
| 100 min/week | Possible first target |

### Dose-Response Relationship
- **Steepest benefits at lowest doses** — curve of diminishing returns
- First 15-30 minutes provide greatest marginal benefit
- Additional exercise helps but at gentler rates
- "Some is better than none" messaging critical

### Vigorous vs Moderate Activity
| Intensity | Time Required for Equivalent Benefit |
|-----------|--------------------------------------|
| **Vigorous** | 15-20 min/week |
| **Moderate** | Substantially more time |
| Vigorous more time-efficient for busy individuals |

### Key Statistics
- **25 min/week vigorous activity:** ~46% lower all-cause mortality
- **25 min/week vigorous activity:** ~58% lower CVD mortality

## Clinical Message
- Remove "all-or-nothing" barrier to exercise
- Start with minimal achievable dose
- Any activity provides substantial benefit
- Build up gradually from there
