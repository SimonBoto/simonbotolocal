There are several popular online reference organizing sites and add-ons used for managing citations and references. Some of the most famous ones include:

1. **Zotero**: This is a free, open-source reference management software to help you collect, organize, cite, and share research sources.

2. **Mendeley**: Mendeley is a reference manager and academic social network that helps you organize your research, collaborate with others online, and discover the latest research.

3. **EndNote**: EndNote is a commercial reference management software package used to manage bibliographies and references when writing essays and articles.

4. **RefWorks**: RefWorks is a web-based reference management service that allows users to create personal databases and use them for a variety of research activities.

5. **ReadCube**: ReadCube is a reference manager that offers features for organizing, discovering, and annotating research papers.

6. **Paperpile**: Paperpile is a reference manager specifically designed for Google Docs and offers seamless integration with Google Scholar and PubMed.

7. **Citavi**: Citavi is a reference management software for Windows that supports organizing knowledge, managing tasks, and creating citations and bibliographies.

These platforms offer various features and integrations to help researchers efficiently manage their references and citations. The choice often depends on personal preferences, workflow requirements, and specific features needed for your research.
