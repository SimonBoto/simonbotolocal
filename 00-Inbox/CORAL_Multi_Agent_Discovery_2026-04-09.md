# CORAL: Autonomous Multi-Agent Discovery

## Key Insights

> "The era of autonomous multi-agent discovery is arriving!"

### CORAL Framework
Autonomous multi-agent systems for open-ended scientific discovery that push beyond constrained loops.

### Agent Autonomy Dimensions
| Capability | Description |
|------------|-------------|
| 🔍 **What to explore** | Agents decide research direction |
| 🧠 **What knowledge to store** | Self-directed memory/knowledge management |
| ♻️ **Which ideas to reuse** | Cross-agent knowledge synthesis |
| 🧪 **When to test hypotheses** | Self-directed experimentation |

### Critical Finding: Multi-Agent > Single Agent
- Single autonomous agent outperforms fixed evolutionary search
- **Multi-agent research community** shows biggest gains
- **>50% of breakthroughs** come from building on other agents' discoveries
- Knowledge reuse and collaboration are central to scalable discovery

### Performance
- 10+ difficult tasks: algorithmic discovery, system optimization
- State-of-the-art performance
- **3-10× efficiency improvement**
