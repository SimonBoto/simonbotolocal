# ISSUE-076: Set OpenRouter $5/Day Hard Spend Limit

## Problem
No hard stop on spending. $30 burn in one day was possible because OpenRouter has no daily cap configured.

## Action
Set OpenRouter dashboard spend limit to $5/day.

## Location
https://openrouter.ai/settings → Limits

## Cost Impact
Prevents runaway spending

## Owner
Simos (requires your OpenRouter login)

## Acceptance Criteria
- [ ] Daily limit set to $5
- [ ] Monthly limit considered
- [ ] Screenshot or confirmation logged
