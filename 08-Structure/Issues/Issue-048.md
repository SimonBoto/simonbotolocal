## Implementation Plan

### Phase 1: Daily Health Check (2 hours)
1. `vault-doctor.sh` — generated by K2.6:
   ```bash
   #!/bin/bash
   # Daily vault health check
   
   VAULT="/home/simon/Workspaces/TheOptimizedBrain"
   REPORT="$VAULT/08-Structure/System/health-reports/daily-$(date +%Y-%m-%d).md"
   
   # 1. YAML validity check
   find $VAULT -name "*.md" -exec yq eval '.' {} \; > /dev/null 2>&1
   YAML_INVALID=$?
   
   # 2. Link verification
   python3 $VAULT/08-Structure/System/scripts/link-checker.py
   BROKEN_LINKS=$?
   
   # 3. Orphan detection
   python3 $VAULT/08-Structure/System/scripts/orphan-detector.py
   ORPHAN_COUNT=$?
   
   # 4. Generate report
   cat > $REPORT << EOF
   # Daily Health Report — $(date)
   
   | Check | Status | Count |
   |-------|--------|-------|
   | YAML Valid | $YAML_INVALID | |
   | Broken Links | $BROKEN_LINKS | |
   | Orphan Zettels | $ORPHAN_COUNT | |
   EOF
   ```
