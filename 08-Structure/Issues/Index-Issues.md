# Issues Index

> **Task tracking system for the agent ecosystem**

Issue-based workflow: create, assign, track, complete, learn.

---

## Dashboards

| View | Purpose |
|------|---------|
| [[Issue-Backlog]] | Pending issues (P0/P1/P2/Icebox) |
| [[Issue-Active]] | Currently assigned and in-progress |
| [[Issue-Completed]] | Archive with outcomes |

---

## Creating Issues

1. Copy [[Template-Issue]]
2. Fill metadata (id, title, assignee, energy, priority)
3. Define success criteria
4. Add to [[Issue-Backlog]]
5. When assigned, move to [[Issue-Active]]

---

## Issue Lifecycle

```
BACKLOG → ASSIGNED → ACTIVE → COMPLETED → (auto-zettel)
   ↑______________↓ (if blocked/failed)
```
