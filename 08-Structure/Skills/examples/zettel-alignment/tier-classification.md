# Zettel Tier Classification

## GOLD (10%)
**Characteristics:**
- Complex, multi-concept zettels
- Requires manual review
- High connection density
- Critical for synthesis

**Example:**
- Zettels with 5+ source papers
- Cross-domain connections
- Disputed or evolving claims

**Process:**
1. Manual review required
2. Verify all connections
3. Check source validity
4. Approve individually

## SILVER (25%)
**Characteristics:**
- Medium complexity
- Batch process + spot check
- Standard connections
- Important but not critical

**Example:**
- Standard biomarker zettels
- Single-source but complex
- Well-established concepts

**Process:**
1. Batch add required sections
2. Spot check 20%
3. Fix issues found
4. Approve batch

## BRONZE (65%)
**Characteristics:**
- Standard complexity
- Full batch processing
- Minimal connections
- Routine maintenance

**Example:**
- Simple mechanism zettels
- Single-source, single claim
- Well-understood concepts

**Process:**
1. Batch add required sections
2. Sample check 5%
3. Approve if sample passes
