# Toxicology-Metabolomics Overlap for Biomarker Discovery

**Concept:** The Laboratory of Toxicology and Forensic Chemistry applies toxicological analytical methods to metabolomic biomarker discovery.

**Why Toxicology Lab for Metabolomics:**

### Shared Techniques
| Technique | Toxicology Use | Metabolomics Use |
|-----------|---------------|------------------|
| GC-MS | Drug detection, poisoning | Metabolite profiling |
| Sample prep | Biofluid extraction | Metabolite extraction |
| Quality control | Forensic standards | Metabolic standards |
| Data analysis | Compound ID | Pathway analysis |

### Advantages
1. **Rigorous QC:** Forensic standards ensure reproducibility
2. **Trace detection:** Toxicology sensitivity → low-abundance metabolites
3. **Validation expertise:** Legal standards translate to clinical validation
4. **Instrumentation:** Existing GC-MS infrastructure

### Translation Path
Toxicology (poison detection) → Metabolomics (disease detection) → Clinical practice (diagnostic tool)

**Implication for METHAP:**
Our adipic acid biomarker work benefits from toxicological rigor in method validation.

**Synergy:**
- IR biomarkers discovered via metabolomics
- Toxicological methods ensure clinical-grade reliability
- Forensic standards → diagnostic accuracy
