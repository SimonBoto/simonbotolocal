## Detailed Template Specifications

### 1. Zettel (Conscious) — P0
**Location:** `03-Zettels/03a-Conscious/`
**Naming:** `Title_[Z-{DDD}-{SSSS}].md`

**YAML Requirements:**
```yaml
---
uid: Z-{DDD}-{SSSS}
type: zettel
status: conscious  # ← NEW
stage: working     # ← NEW
title: {{Title}}
domain: {DDD}
tags: [zettel, {topic}]
parent-moc: [[MOC-{DDD}]]
source: [[{LitNote}]]
confidence: high | medium | low
created: {YYYY-MM-DD}
review-by: {YYYY-MM-DD}  # ← Auto: created + 30 days
---
```

**Body:** Standard zettel format (50-200 words, 3 links)

---

### 2. Zettel (Subconscious) — P0
**Location:** `03-Zettels/03b-Subconscious/`
**Naming:** `Title_[Z-{DDD}-{SSSS}].md` (same as conscious)

**YAML Requirements:**
```yaml
---
uid: Z-{DDD}-{SSSS}
type: zettel
status: subconscious  # ← NEW
stage: permanent      # ← NEW
title: {{Title}}
domain: {DDD}
tags: [zettel, {topic}]
parent-moc: [[MOC-{DDD}]]
source: [[{LitNote}]]
confidence: high  # ← Must be high for promotion
created: {YYYY-MM-DD}
promoted: {YYYY-MM-DD}  # ← NEW: promotion date
---
```

**Rules Header (NEW):**
```markdown
> **⚠️ SUBCONSCIOUS ZETTEL — PERMANENT**
> This note represents "ground truth" as of {promoted-date}.
> DO NOT MODIFY. Link only. Add new understanding as new zettels.
```
