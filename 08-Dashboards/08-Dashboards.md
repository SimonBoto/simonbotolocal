## 🔄 Daily Workflow

```
Morning:
    1. Open MISSION_CONTROL — See today's plan
    2. Check 00-COMMAND — P0 objectives
    3. Review 01-QUEUE — Active tasks
    
During Day:
    4. Work from 01-QUEUE
    5. Drafts go to 02-HANDOFF for review
    
Evening:
    6. Update 03-LOG
    7. Plan TOMORROWS_BATTLE_PLAN
```
