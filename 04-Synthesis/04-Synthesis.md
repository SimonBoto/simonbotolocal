# 📖 04-Synthesis — Living Documents

**Purpose:** Evolving documents that aggregate zettels into coherent narratives. Living — not static.

> *"Zettels are the bricks. Synthesis is the architecture."*

---

## 📄 Contents

Synthesis documents:
- Aggregate zettels by topic
- Tell coherent stories
- Evolve as new zettels are added
- Link heavily to source zettels
- Feed into MOCs for navigation

---

## 🔄 Workflow

```
Zettels Accumulate (03-Zettels/)
    ↓
Pattern Emerges
    ↓
Create Synthesis Document
    ↓
Aggregate relevant zettels
    ↓
Tell the story
    ↓
Update as new zettels added
    ↓
Link from MOC (06-Structure/MOCs/)
```

## 📝 Synthesis Template

See: [[06-Structure/Templates/Template_Synthesis_v1.0|Template_Synthesis_v1.0.md]]

**Standard structure:**
1. Core Definition (current understanding)
2. Source Evolution (chronological development)
3. Contributing Zettels (explicit list)
4. Synthesis Gaps (what's missing)
5. Open Questions (tensions, contradictions)
6. Related Syntheses (cross-connections)
7. Update Log (version history)
