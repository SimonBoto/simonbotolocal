## Decision Trees / Checklists

### Pre-Submission Checklist
- [ ] Study design scientifically valid
- [ ] Inclusion/exclusion criteria fair
- [ ] Risk-benefit analysis favorable
- [ ] Informed consent forms complete
- [ ] Data protection plan documented
- [ ] Vulnerable populations protected
- [ ] Independent review obtained

### Informed Consent Elements
- [ ] Purpose of research explained
- [ ] Procedures described
- [ ] Risks disclosed
- [ ] Benefits described
- [ ] Alternatives stated
- [ ] Confidentiality assured
- [ ] Voluntary participation emphasized
- [ ] Contact information provided
