## Chemical Classification

### Aldehyde vs Ketone
| Feature | **Aldehyde** | **Ketone** |
|---------|--------------|------------|
| **Carbonyl position** | Terminal carbon (C1) | Internal carbon |
| **Structure** | R-CHO | R-CO-R' |
| **Example in sugars** | Glucose (aldose) | Fructose (ketose) |
| **Oxidation** | To carboxylic acid | More resistant |

### Keto Acids vs Carboxylic Acids

**Keto acid (α-keto acid):**
- Contains **ketone group** AND **carboxylic acid group**
- Ketone at α-position (adjacent to carboxyl)
- Example: Pyruvate, α-ketoglutarate

**Carboxylic acid:**
- Contains **only** -COOH group
- Example: Acetate, propionate

**Relationship:**
- All keto acids ARE carboxylic acids
- Not all carboxylic acids are keto acids

## Pyruvate: The Central Keto Acid

### Structure and Properties
- **3-carbon α-keto acid**
- **Formula:** CH₃COCOOH
- **Simplest α-keto acid**

### Metabolic Central Role
```
Glucose → ... → Pyruvate → [Multiple fates]
```

**Aerobic:**
```
Pyruvate → Acetyl-CoA → Krebs cycle
```
(PDH complex: requires B1, B2, B3, B5, lipoate)

**Anaerobic:**
```
Pyruvate → Lactate (LDH)
```

**Gluconeogenic:**
```
Pyruvate → Oxaloacetate → Glucose
```

## Propionate: The Gut-Derived Acid

### Sources
- **Dietary fiber fermentation** (gut microbiome)
- **Amino acid catabolism** (methionine, threonine, valine, isoleucine)
- **Dairy products** (natural content)

### Structure
- **3-carbon carboxylic acid** (NOT keto acid)
- **Formula:** CH₃CH₂COOH
- No ketone group (unlike pyruvate)

### Metabolism: Propionate → Glucose
**Gluconeogenic pathway (liver):**

| Step | Reaction | Enzyme | Cofactor |
|------|----------|--------|----------|
| 1 | Propionate → Propionyl-CoA | Propionyl-CoA synthetase | ATP, CoA |
| 2 | Propionyl-CoA → Methylmalonyl-CoA | Propionyl-CoA carboxylase | **Biotin (B7)** |
| 3 | Methylmalonyl-CoA → Succinyl-CoA | **Methylmalonyl-CoA mutase** | **B12 (cobalamin)** |
| 4 | Succinyl-CoA → Krebs cycle → OAA | Various | - |
| 5 | OAA → PEP → Glucose | PEPCK, gluconeogenic enzymes | - |

**Net:** Propionate → **Glucose** (gluconeogenic substrate)

## Vitamin Dependencies

### Critical Cofactors
| Vitamin | Role in Organic Acid Metabolism | Deficiency Effect |
|---------|----------------------------------|-------------------|
| **B7 (Biotin)** | Propionyl-CoA carboxylase | Propionic acidemia |
| **B12 (Cobalamin)** | Methylmalonyl-CoA mutase | Methylmalonic acidemia |
| **B1 (Thiamine)** | Pyruvate dehydrogenase | Lactic acidosis, beriberi |
| **B2 (Riboflavin)** | PDH, α-KGDH | Fatigue, neurological |
| **B3 (Niacin)** | PDH, dehydrogenases | Pellagra |
| **B5 (Pantothenate)** | CoA synthesis | Fatigue, malaise |

### Clinical Pearl
**Methylmalonic acidemia (MMA):**
- B12 deficiency or mutase deficiency
- MMA accumulates → metabolic acidosis
- Neurological damage if untreated
- Treatment: B12 supplementation, diet restriction

## Comparison: Pyruvate vs Propionate

| Feature | **Pyruvate** | **Propionate** |
|---------|--------------|----------------|
| **Class** | α-Keto acid | Carboxylic acid |
| **Carbons** | 3 | 3 |
| **Source** | Glycolysis, amino acids | Gut bacteria, diet |
| **Ketone group** | Yes | No |
| **Primary fate** | Acetyl-CoA, lactate | Glucose (via gluconeogenesis) |
| **Energy yield** | High (aerobic) | Moderate |
| **Key cofactors** | B1, B2, B3, B5 | B7, B12 |

## Physiological Roles

### Pyruvate
1. **Energy production** (aerobic/anaerobic)
2. **Gluconeogenesis substrate**
3. **Amino acid precursor** (transamination → alanine)
4. **Lipogenesis** (acetyl-CoA → fatty acids)

### Propionate
1. **Gluconeogenic substrate**
2. **Gut hormone signaling** (GLP-1, PYY)
3. **Metabolic health** (improves insulin sensitivity)
4. **Cholesterol synthesis** (indirectly via succinyl-CoA)

---

## Clinical Relevance

### Lactic Acidosis
**Pyruvate → Lactate accumulation:**
- Hypoxia (shock, sepsis)
- Thiamine deficiency
- Mitochondrial disorders
- Metformin toxicity (inhibits PDH)

### Organic Acidemias
**Inborn errors of metabolism:**
- Propionic acidemia (biotin-related)
- Methylmalonic acidemia (B12-related)
- Pyruvate dehydrogenase deficiency

### Therapeutic Potential
**Propionate supplementation:**
- Improved glycemic control (studies)
- Gut microbiome modulation
- May reduce cholesterol synthesis
