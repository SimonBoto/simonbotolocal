## Core Concepts

### 1. The Complete Krebs Cycle Amino Acid Table
> "Here's a table that outlines the metabolites of the Krebs cycle and the amino acids that are connected to each through various metabolic pathways."

| Krebs Cycle Metabolite | Connected Amino Acids |
|------------------------|----------------------|
| **Acetyl-CoA** | Leucine, Isoleucine, Tryptophan |
| **Citrate** | None directly |
| **Isocitrate** | None directly |
| **α-Ketoglutarate** | Glutamate, Proline, Arginine, Histidine |
| **Succinyl-CoA** | Methionine, Isoleucine, Valine, Threonine |
| **Succinate** | None directly |
| **Fumarate** | Phenylalanine, Tyrosine |
| **Malate** | None directly |
| **Oxaloacetate** | Aspartate |

### 2. α-Ketoglutarate Entry — The Glutamate Hub
> "Glutamate is directly transaminated to form α-ketoglutarate. Proline, Arginine, and Histidine can be metabolized to glutamate, which in turn can be converted into α-ketoglutarate."

**Four amino acids converge:**
- **Glutamate** → direct transamination to α-ketoglutarate
- **Proline** → glutamate → α-ketoglutarate
- **Arginine** → glutamate → α-ketoglutarate  
- **Histidine** → glutamate → α-ketoglutarate

### 3. BCAA to Succinyl-CoA — Critical Connection
> "Methionine, Isoleucine, Valine, and Threonine can be catabolized to propionyl-CoA, which is then converted into succinyl-CoA through several steps."

**The BCAA-Succinyl-CoA Pathway:**
```
Methionine → propionyl-CoA → methylmalonyl-CoA → succinyl-CoA
Isoleucine → propionyl-CoA → methylmalonyl-CoA → succinyl-CoA
Valine → propionyl-CoA → methylmalonyl-CoA → succinyl-CoA
Threonine → propionyl-CoA → methylmalonyl-CoA → succinyl-CoA
```

**Critical enzyme:** Methylmalonyl-CoA mutase (requires **vitamin B12**)

### 4. Threonine — Dual Metabolism
> Threonine is unique — it can enter both glucogenic and ketogenic pathways.

**Threonine fates:**
- **Glucogenic:** Propionyl-CoA → succinyl-CoA → glucose
- **Ketogenic:** Can also produce acetyl-CoA
- **Both:** Similar to isoleucine, phenylalanine, tyrosine, tryptophan

### 5. Ketogenic vs Glucogenic Classification
> "Leucine, Isoleucine, Tryptophan are ketogenic, meaning they can be broken down into acetyl-CoA."

**Purely ketogenic (2):**
- Leucine
- Lysine

**Both glucogenic and ketogenic (4):**
- Isoleucine
- Phenylalanine
- Tyrosine
- Tryptophan

**Purely glucogenic (14):**
- All others (alanine, arginine, asparagine, aspartate, cysteine, glutamate, glutamine, glycine, histidine, methionine, proline, serine, threonine, valine)

### 6. Anaplerotic Reactions
> "Aspartate can be directly transaminated to produce oxaloacetate, which is a critical step in replenishing Krebs cycle intermediates."

**Anaplerosis (filling up):**
- Pyruvate → oxaloacetate (pyruvate carboxylase)
- Aspartate → oxaloacetate (transamination)
- Glutamate → α-ketoglutarate (transamination)

**Purpose:** Replenish Krebs cycle intermediates when used for biosynthesis

### 7. Malic Acid and Amino Acids
> "Aspartic acid can be converted directly into oxaloacetate, which can then be converted into malate."

**Malate connection:**
- Aspartate → oxaloacetate → malate (reduction)
- Glutamate → α-ketoglutarate → succinyl-CoA → succinate → fumarate → malate
