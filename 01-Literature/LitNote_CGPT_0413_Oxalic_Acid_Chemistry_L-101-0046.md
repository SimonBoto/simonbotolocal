## Core Concepts

### Chemical Properties
- **Molecular formula:** C₂H₂O₄ (dicarboxylic acid)
- **Forms:** Anhydrous and dihydrate (C₂H₂O₄·2H₂O)
- **Appearance:** White crystalline solid, colorless solution in water
- **Structure:** Simplest dicarboxylic acid — two carboxylic acid groups

### Industrial/Manufacturing Uses
| Application | Function |
|-------------|----------|
| Metal cleaning | Rust/oxidation removal |
| Wood treatment | Lightening stained/darkened areas |
| Textile industry | Mordant in dyeing processes |
| Chemical synthesis | Pharmaceutical precursors |
| Organic reactions | Reducing agent |

### Dietary Sources (High Oxalate Foods)
- **Leafy greens:** Spinach, Swiss chard, beet greens
- **Vegetables:** Rhubarb, beets, okra, leeks
- **Nuts/seeds:** Almonds, cashews, sesame seeds
- **Legumes:** Soybeans, navy beans, black beans
- **Grains:** Buckwheat, wheat bran
- **Other:** Chocolate, black tea

### Metabolic Origins — Amino Acid Pathways
| Amino Acid | Pathway | Product |
|------------|---------|---------|
| **Glycine** → | Glyoxylate → | Oxalate |
| **Hydroxyproline** → | Glyoxylate → | Oxalate |

- **Glyoxylate** is the key intermediate
- **Primary hyperoxaluria:** Genetic disorder → excessive oxalate production
- **Collagen turnover:** High hydroxyproline release → increased oxalate

### Ascorbic Acid Connection
| Feature | Oxalic Acid | Ascorbic Acid (Vitamin C) |
|---------|-------------|---------------------------|
| Carboxyl groups | 2 (dicarboxylic) | 1 + lactone ring |
| Solubility | Water-soluble | Water-soluble |
| Reducing properties | Yes | Yes (antioxidant) |
| Plant occurrence | Widespread | Widespread |
| **Metabolic fate** | Oxalate (end product) | Can convert to oxalate |

### Clinical Relevance
- **Mineral binding:** Forms insoluble oxalates with Ca²⁺, Fe²⁺/Fe³⁺
- **Bioavailability impact:** Reduces calcium/iron absorption
- **Kidney stones:** Calcium oxalate precipitation risk
- **Dietary management:** Balance needed — exclusion creates nutritional gaps

## Key Distinctions

### Oxalic Acid vs. Ascorbic Acid
- **Structural similarity:** Both carboxylic acids, both plant-derived
- **Functional difference:** Vitamin C = essential nutrient; oxalic acid = mineral chelator
- **Metabolic link:** Ascorbate CAN convert to oxalate (see [[CGPT_0505_Vitamin_C_to_Oxalate_CGPT-0505])

### Glycolic Acid Connection
- **Glycine → glyoxylate →** can branch to **glycolic acid** OR **oxalic acid**
- Metabolic disorders at glyoxylate level affect both pathways
