# Lit — Inflammatory Bowel Disease — Crohn's vs Ulcerative Colitis Pathophysiology

> **Source:** ChatGPT conversation [[CGPT_0209_Gastroenterology_Registrar_Assessment_CGPT-0209]] (November 18, 2025)  
> **Core Claim:** This clinical synthesis establishes the fundamental pathophysiological distinctions between Crohn's disease and ulcerative colitis, providing a comprehensive framework for differential diagnosis based on distribution patterns, inflammation depth, histological features, serological markers, and treatment approaches.

---

## Core Concepts Extracted

### 1. Pathophysiology Overview Framework

**Category:** Gastrointestinal Pathology

**Key Process:** 
```
Dysregulated immune response → Gut microbiota interaction → Barrier defect → Genetic + environmental triggers → Chronic relapsing inflammation
```

**Mechanism/Steps:**
1. Dysregulated immune activation against gut microbiota
2. Defective intestinal barrier function
3. Genetic susceptibility (NOD2/CARD15, IL23R)
4. Environmental triggers (diet, smoking, medications)
5. Results in chronic relapsing inflammation throughout GI tract

**Purpose/Significance:** Provides the unifying pathophysiological framework that explains the shared clinical features of IBD while distinguishing underlying mechanisms

**Zettel:** [[Pathophysiology_Overview_Framework_Z-001-0585A]]

---

### 2. Crohn's Disease Pathophysiology Framework

**Category:** Gastrointestinal Pathology

**Key Process:** 
```
Transmural inflammation → Skip lesions → Any GI segment → Granulomas → Th1/Th17 immunity
```

**Mechanism/Steps:**
1. Characterized by transmural (full-thickness) inflammation
2. Skip lesions with areas of normal tissue between diseased segments
3. Can affect any part of the GI tract (ileum most common)
4. Non-caseating granulomas present in 40-60% of cases
5. Th1 and Th17 immune profile with elevated IFN-γ, TNF-α, IL-12, IL-23

**Purpose/Significance:** Explains the distinctive clinical presentation, complications (fistulas, strictures), and treatment requirements for Crohn's disease

**Zettel:** [[Crohns_Disease_Pathophysiology_Framework_Z-001-0585B]]

---

### 3. Ulcerative Colitis Pathophysiology Framework

**Category:** Gastrointestinal Pathology

**Key Process:** 
```
Mucosal inflammation → Continuous → Colon only → Crypt abscesses → Th2 immunity
```

**Mechanism/Steps:**
1. Limited to mucosal (superficial) inflammation
2. Continuous involvement without skip lesions
3. Colon only (always involves rectum)
4. Crypt abscesses as characteristic histological feature
5. Th2 immune profile with elevated IL-5, IL-13, and humoral immunity

**Purpose/Significance:** Explains the distinctive clinical presentation, complications (dysplasia), and treatment requirements for ulcerative colitis

**Zettel:** [[Ulcerative_Colitis_Pathophysiology_Framework_Z-001-0585C]]

---

### 4. Comparative Features Framework

**Category:** Clinical Differentiation

**Key Process:** 
```
Distribution: Any GI vs Colon only → Pattern: Skip vs Continuous → Depth: Transmural vs Mucosal → Rectal: Variable vs Universal → Granulomas: Present vs Absent
```

**Mechanism/Steps:**
1. Distribution: Crohn's affects any GI segment; UC only colon
2. Pattern: Crohn's shows skip lesions; UC continuous
3. Depth: Crohn's transmural; UC mucosal only
4. Rectal involvement: Crohn's variable; UC always involves rectum
5. Granulomas: Crohn's present (40-60%); UC absent

**Purpose/Significance:** Provides the clinical criteria for differential diagnosis that guides diagnostic testing and management decisions

**Zettel:** [[Comparative_Features_Framework_Z-001-0585D]]

---

### 5. Serological Markers Framework

**Category:** Diagnostic Medicine

**Key Process:** 
```
p-ANCA+ → UC-like (60-70%) → ASCA+ → Crohn's-like (50-60%) → Diagnostic utility: p-ANCA+/ASCA- → UC → p-ANCA-/ASCA+ → Crohn's
```

**Mechanism/Steps:**
1. p-ANCA: positive in 60-70% of UC, 10-20% of Crohn's
2. ASCA: positive in 50-60% of Crohn's, 10-15% of UC
3. Diagnostic utility: p-ANCA+/ASCA- suggests UC; p-ANCA-/ASCA+ suggests Crohn's
4. Both positive or both negative are less specific

**Purpose/Significance:** Enables non-invasive serological differentiation when clinical presentation is ambiguous

**Zettel:** [[Serological_Markers_Framework_Z-001-0585E]]

---

### 6. Treatment Approach Framework

**Category:** Clinical Management

**Key Process:** 
```
Crohn's: Transmural healing → Fistulas/strictures prevention → Surgery often needed → UC: Mucosal healing → Surgery curative → Rectal therapy important
```

**Mechanism/Steps:**
1. Crohn's treatment targets deeper (transmural) healing, prevention of strictures/fistulas, and often requires surgery
2. UC treatment targets mucosal healing, with surgery (proctocolectomy) being curative
3. Biologic selection: anti-TNF, anti-integrin, anti-IL12/23 for Crohn's; anti-TNF, anti-integrin, JAK inhibitors for UC
4. Rectal therapy is particularly important in UC management

**Purpose/Significance:** Guides therapy selection and management expectations based on disease-specific pathophysiology

**Zettel:** [[Treatment_Approach_Framework_Z-001-0585F]]
