## Entry Point 1: Oxaloacetate

### Amino Acids
- **Asparagine** → Aspartate (asparaginase)
- **Aspartate** → Oxaloacetate (aspartate transaminase)

### Enzymes
**Aspartate transaminase (AST/ALT):**
- Mitochondrial and cytosolic isozymes
- **Vitamin B6** (pyridoxal phosphate) required
- Clinical marker: AST/ALT ratio in liver disease

### Significance
- Direct connection to urea cycle
- Aspartate provides nitrogen for arginine synthesis
- Anaplerotic reaction

---

## Entry Point 2: α-Ketoglutarate

### Amino Acids (4 total)
| Amino Acid | Pathway | Enzyme |
|------------|---------|--------|
| **Glutamate** | Direct transamination | Glutamate dehydrogenase / transaminases |
| **Arginine** | → Glutamate → α-KG | Arginase, then transaminase |
| **Proline** | → Glutamate → α-KG | Proline dehydrogenase, P5C dehydrogenase |
| **Histidine** | → Glutamate → α-KG | Histidase, urocanase, formiminotransferase |

### Central Role
**Glutamate as hub:**
- All four converge on glutamate
- Glutamate ↔ α-ketoglutarate (transamination)
- **Key enzyme:** All B6-dependent

### Vitamin B6 Criticality
**Pyridoxal phosphate (PLP) required for:**
- All transaminases
- Decarboxylases
- Racemases

**Deficiency:** Impaired amino acid metabolism, neurological symptoms

## Entry Point 3: Succinyl-CoA

### Amino Acids (4 total)
| Amino Acid | Pathway | Special Notes |
|------------|---------|---------------|
| **Isoleucine** | → Propionyl-CoA → Methylmalonyl-CoA → Succinyl-CoA | Branched-chain |
| **Methionine** | → Propionyl-CoA → Succinyl-CoA | S-adenosylmethionine first |
| **Valine** | → Propionyl-CoA → Succinyl-CoA | Branched-chain |
| **Threonine** | → α-ketobutyrate → Propionyl-CoA → Succinyl-CoA | Threonine dehydrogenase |

### Critical Vitamin: B12
**Methylmalonyl-CoA mutase requires B12 (adenosylcobalamin):**
```
Methylmalonyl-CoA → Succinyl-CoA (B12-dependent)
```

**B12 deficiency consequence:**
- Methylmalonic acidemia (MMA accumulation)
- Neurological damage (irreversible)
- Megaloblastic anemia

### Branched-Chain Amino Acids (BCAAs)
- Valine, Isoleucine, Leucine
- Share initial catabolic pathway
- Branched-chain α-keto acid dehydrogenase (BCKDH)
- Maple syrup urine disease (BCKDH deficiency)

## Entry Point 4: Fumarate

### Amino Acids
- **Tyrosine** → Fumarate (via homogentisate)
- **Phenylalanine** → Tyrosine → Fumarate

### Enzymes
**Phenylalanine hydroxylase:**
- Converts phenylalanine → tyrosine
- **Requires: Folate, Iron, BH4**
- Deficiency: Phenylketonuria (PKU)

**Tyrosine aminotransferase:**
- B6-dependent
- Rate-limiting for tyrosine catabolism

### Clinical Connection
**Tyrosinemias:**
- Type I: Fumarylacetoacetate hydrolase deficiency
- Type II: Tyrosine aminotransferase deficiency
- Type III: 4-hydroxyphenylpyruvate dioxygenase deficiency

## Entry Point 5: Pyruvate

### Amino Acids (6 total)
| Amino Acid | Conversion | Enzyme | Notes |
|------------|------------|--------|-------|
| **Alanine** | Direct transamination | Alanine transaminase (ALT) | Glucose-alanine cycle |
| **Cysteine** | → Pyruvate + H₂S | Cysteine desulfurase | Sulfur metabolism |
| **Serine** | → Glycine → Pyruvate | Serine hydroxymethyltransferase | Folate-dependent |
| **Glycine** | → Serine → Pyruvate | SHMT reverse | Folate-dependent |
| **Threonine** | → Glycine → Pyruvate | Threonine aldolase | Split pathway |
| **Tryptophan** | → Alanine → Pyruvate | Multiple steps | Kynurenine pathway |

### Folate Connection
**Serine-glycine interconversion:**
```
Serine + THF → Glycine + 5,10-methylene-THF
```
**Enzyme:** Serine hydroxymethyltransferase (SHMT)
**Cofactor:** Tetrahydrofolate (folate)

**Folate deficiency:** Impaired serine/glycine metabolism

## Entry Point 6: Acetyl-CoA (Ketogenic)

### Amino Acids (4 total)
| Amino Acid | Pathway | Classification |
|------------|---------|----------------|
| **Leucine** | → HMG-CoA → Acetyl-CoA + Acetoacetate | **Purely ketogenic** |
| **Lysine** | Complex pathway | **Purely ketogenic** |
| **Tryptophan** | → Alanine (glucogenic) + Acetyl-CoA (ketogenic) | **Both** |
| **Isoleucine** | → Succinyl-CoA (glucogenic) + Acetyl-CoA (ketogenic) | **Both** |

### Purely Ketogenic
**Leucine and Lysine:**
- Cannot be converted to glucose
- Yield acetyl-CoA or acetoacetyl-CoA
- **Only ketogenic amino acids**

### Both (Glucogenic + Ketogenic)
| Amino Acid | Glucogenic Part | Ketogenic Part |
|------------|-----------------|----------------|
| Isoleucine | Succinyl-CoA | Acetyl-CoA |
| Phenylalanine | Fumarate | Acetoacetyl-CoA |
| Tryptophan | Alanine (→ pyruvate) | Acetyl-CoA |
| Tyrosine | Fumarate | Acetoacetyl-CoA |

## Glucogenic vs Ketogenic Summary

### Glucogenic Only (14 amino acids)
Alanine, Arginine, Asparagine, Aspartate, Cysteine, Glutamate, Glutamine, Glycine, Histidine, Methionine, Proline, Serine, Threonine, Valine

### Ketogenic Only (2 amino acids)
**Leucine, Lysine**

### Both (4 amino acids)
Isoleucine, Phenylalanine, Tryptophan, Tyrosine
