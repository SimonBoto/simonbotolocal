## Study Overview: METHAP Trial

### Research Focus
**Identifying novel biomarkers for early insulin resistance detection using metabolomics**

| Aspect | Details |
|--------|---------|
| **Primary analytes** | Urine organic acids + plasma free fatty acids |
| **Target condition** | Early insulin resistance (before HbA1c/fasting glucose abnormalities) |
| **Current tools to improve** | HOMA-IR, HbA1c, fasting glucose (late detection) |

### Study Design
| Element | Specification |
|---------|---------------|
| **Design** | Observational + interventional phases |
| **Sample size** | 200 participants |
| **Control group** | 100 individuals without insulin resistance |
| **Case group** | 100 individuals with insulin resistance |
| **Analytical platform** | GC-MS (gas chromatography-mass spectrometry) |
| **Statistical methods** | PCA, multivariate regression, pathway analysis |

### Intervention Phase (Subset)
| Feature | Details |
|---------|---------|
| **Duration** | 12 weeks |
| **Diet** | Low-carbohydrate diet |
| **Supplements** | Omega-3, Vitamin D, Magnesium, Glutamine, Vitamin C, Probiotics |
| **Outcomes** | Metabolic markers, insulin sensitivity, biochemical parameters |
