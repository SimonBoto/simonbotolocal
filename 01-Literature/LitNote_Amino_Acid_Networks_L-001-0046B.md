## α-Keto Acids: The Carbon Skeletons

### Definition
α-Keto acids are **amino acids without the amino group** — the carbon skeletons remaining after deamination or transamination.

### Key α-Keto Acids in Metabolism

| α-Keto Acid | From Amino Acid | Krebs Entry |
|-------------|-----------------|-------------|
| **Pyruvate** | Alanine | Pyruvate (→ acetyl-CoA or OAA) |
| **Oxaloacetate** | Aspartate | Krebs cycle |
| **α-Ketoglutarate** | Glutamate | Krebs cycle |
| **α-Ketoisocaproate** | Leucine | Ketogenic (→ HMG-CoA) |
| **α-Ketoisovalerate** | Valine | Glucogenic (→ succinyl-CoA) |
| **α-Keto-β-methylvalerate** | Isoleucine | Both (→ succinyl-CoA + acetyl-CoA) |

### Transamination Reversibility
```
Amino Acid + α-Keto Acid ↔ α-Keto Acid + Amino Acid
```
**Example:**
```
Alanine + α-ketoglutarate ↔ Pyruvate + Glutamate
```

**Enzyme:** Aminotransferases (B6-dependent)

## The Glutamate Hub

### Central Position
**Glutamate is the metabolic crossroads of amino acid metabolism:**
- Receives amino groups from most amino acids (via transamination)
- Transfers amino groups to urea cycle
- Entry to Krebs cycle (as α-ketoglutarate)

### Glutamate ↔ α-Ketoglutarate
```
Glutamate + NAD⁺ + H₂O ↔ α-ketoglutarate + NH₄⁺ + NADH
```
**Enzyme:** Glutamate dehydrogenase (GDH)
- **Allosteric regulation:**
  - Activated by: ADP, leucine
  - Inhibited by: ATP, GTP, NADH

### Connected Amino Acids
All converge on glutamate:
- **Arginine** → Glutamate (via ornithine, glutamate-γ-semialdehyde)
- **Proline** → Glutamate (via P5C)
- **Histidine** → Glutamate (via urocanate, formiminoglutamate)

## Glutamine-Glutamate-Arginine Axis

### Glutamine → Glutamate
```
Glutamine + H₂O → Glutamate + NH₃
```
**Enzyme:** Glutaminase
- **Tissue-specific:** Kidney (acid-base), brain (neurotransmitter), immune cells
- **Purpose:** Release ammonia for buffering or urea synthesis

### Glutamate → Glutamine (Reverse)
```
Glutamate + NH₃ + ATP → Glutamine + ADP + Pi
```
**Enzyme:** Glutamine synthetase
- **Location:** Brain (ammonia detoxification), liver
- **Purpose:** Ammonia detoxification, nitrogen transport

### Glutamate → Arginine (Urea Cycle)
```
Glutamate → N-acetylglutamate (NAGS)
NAG activates carbamoyl phosphate synthetase I
Glutamate → Ornithine → Citrulline → Arginine
```

**Aspartate connection:**
- Aspartate + Citrulline → Arginosuccinate
- Provides second nitrogen for urea

## Aspartate: The Simple Connector

### Structure
- **4-carbon dicarboxylic amino acid**
- Simple structure: -OOC-CH₂-CH(NH₂)-COO⁻
- Acidic side chain (pKa ~3.9)

### Synthesis
```
Oxaloacetate + Glutamate ↔ Aspartate + α-ketoglutarate
```
**Enzyme:** Aspartate aminotransferase (AST)

### Roles
1. **Urea cycle nitrogen donor**
2. **Purine/pyrimidine synthesis** (nucleotide backbone)
3. **Krebs cycle anaplerosis** (OAA precursor)
4. **Neurotransmitter** (NMDA receptor co-agonist)

### Evolutionary Significance
**Simos insight:** Aspartate is "extremely simple and evolutionarily important"
- Simple 4-carbon skeleton
- Central to urea cycle (nitrogen disposal)
- Krebs cycle integration
- Protein synthesis component

## Acetyl-CoA: The Energy Exit

### From Ketogenic Amino Acids
**Leucine pathway:**
```
Leucine → α-ketoisocaproate → Isovaleryl-CoA → β-methylcrotonyl-CoA → 
Methylglutaconyl-CoA → HMG-CoA → Acetoacetate + Acetyl-CoA
```

**Lysine pathway:**
- Complex, bifurcated pathway
- Eventually yields acetyl-CoA
- Purely ketogenic

### Acetyl-CoA Fates
1. **Krebs cycle** (if OAA available)
2. **Ketogenesis** (liver, fasting)
3. **Fatty acid synthesis** (fed state)
4. **Cholesterol synthesis**

## Nitrogen Metabolism Integration

### The Glutamine Connection
**Glutamine as nitrogen shuttle:**
- **Muscle → Liver:** Nitrogen export (glucose-alanine cycle)
- **Brain:** Ammonia detoxification
- **Kidney:** Acid-base regulation
- **Immune cells:** Energy and biosynthesis
- **Intestine:** Preferred fuel

### Ammonia Detoxification
**Two main pathways:**
1. **Urea cycle (liver):** Glutamate → Urea (for excretion)
2. **Glutamine synthesis (brain, muscle):** Glutamate + NH₃ → Glutamine
