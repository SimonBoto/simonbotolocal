## The Hormonal Hierarchy

### Hypothalamus — Master Regulator

**Function:** Monitors body needs, maintains homeostasis

**Releasing Hormones:**
| Hormone | Target | Action |
|---------|--------|--------|
| **GnRH** | Pituitary | Releases LH and FSH |
| **TRH** | Pituitary | Releases TSH |
| **CRH** | Pituitary | Releases ACTH |
| **GHRH** | Pituitary | Releases GH |
| **Somatostatin** | Pituitary | Inhibits GH and TSH |
| **Dopamine** | Pituitary | Inhibits prolactin |

### Pituitary — Command Center

**Anterior Pituitary Hormones:**
| Hormone | Target | Action |
|---------|--------|--------|
| **LH** | Testes/Ovaries | Stimulates sex steroid production |
| **FSH** | Testes/Ovaries | Stimulates gamete production |
| **ACTH** | Adrenal cortex | Stimulates cortisol, DHEA |
| **TSH** | Thyroid | Stimulates T3/T4 |
| **GH** | Multiple tissues | Growth and metabolism |
| **Prolactin** | Breast | Lactation (inhibits GnRH) |

**Posterior Pituitary (Storage):**
- **ADH/Vasopressin:** Water balance
- **Oxytocin:** Uterine contraction, milk ejection

### End Organs — Production Sites

**Testes:**
- Leydig cells: Testosterone (stimulated by LH)
- Sertoli cells: Sperm production (stimulated by FSH), Inhibin

**Adrenal Glands:**
- **Cortex:**
  - Zona glomerulosa: Aldosterone
  - Zona fasciculata: Cortisol
  - Zona reticularis: DHEA, DHEA-S, androgens
- **Medulla:** Epinephrine, norepinephrine

**Prostate:**
- **Not a hormone producer**
- Target organ for androgens (DHT primary)
- 5-alpha-reductase converts testosterone → DHT

## The HPG Story

### Step-by-Step Cascade

```
Hypothalamus
    ↓ GnRH pulses
Anterior Pituitary
    ↓ LH and FSH
Testes
    ↓
  - LH → Leydig cells → Testosterone
  - FSH → Sertoli cells → Spermatogenesis + Inhibin
    ↓
Testosterone enters circulation
    ↓
  - Direct actions (androgen receptor)
  - Conversion to estradiol (aromatase)
  - Conversion to DHT (5-alpha-reductase)
    ↓
Target tissues (muscle, bone, brain, prostate)
    ↓
Negative feedback to hypothalamus and pituitary
```

### Feedback Loops

**Negative Feedback:**
- Testosterone → inhibits GnRH and LH
- Estradiol → inhibits GnRH and LH
- Inhibin → selectively inhibits FSH
- Prolactin → inhibits GnRH (hyperprolactinemia causes hypogonadism)

## Integration with HPA Axis

### Stress Response Interference

**Cortisol Effects:**
- Inhibits GnRH pulsatility
- Suppresses LH response to GnRH
- Reduces testosterone production
- **Result:** Stress-induced hypogonadism

**Clinical:**
- Chronic illness → low testosterone
- Cushing's syndrome → hypogonadism
- Critical illness → eugonadal sick syndrome

## Clinical Applications

### Distinguishing Primary vs Secondary Hypogonadism

| Feature | Primary (Testicular) | Secondary (Pituitary/Hypothalamic) |
|---------|---------------------|-----------------------------------|
| **Testosterone** | Low | Low |
| **LH** | High | Low/normal |
| **FSH** | High | Low/normal |
| **Cause** | Testicular failure | Pituitary disease, hypothalamic dysfunction |
| **Examples** | Klinefelter, chemotherapy, trauma | Tumors, iron overload, hyperprolactinemia |

### Pituitary Function Testing

**Stimulation Tests:**
- GnRH stimulation test
- hCG stimulation test (testicular reserve)

**Suppression Tests:**
- Dexamethasone suppression (Cushing's)
