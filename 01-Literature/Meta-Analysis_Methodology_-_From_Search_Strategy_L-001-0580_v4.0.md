# Lit — Meta-Analysis Methodology — From Search Strategy to Publication

> **Source:** ChatGPT conversation [[CGPT_0621_Publishing_Meta_Analysis_Guidelines_CGPT-0621]] (June 02, 2023)  
> **Core Claim:** This methodology synthesis establishes a comprehensive 60-day learning framework for systematic review and meta-analysis, providing the complete methodological pathway from search strategy development through publication.

## Core Concepts Extracted

### 1. 60-Day Learning Framework

**Category:** Research Methodology

**Key Process:** 
```
Phase 1: Understanding (Days 1-15) → Phase 2: Implementation (Days 16-30) → Phase 3: Consolidation (Days 31-45) → Phase 4: Finalization (Days 46-60)
```

**Mechanism/Steps:**
1. Phase 1 focuses on foundational knowledge and tool familiarization
2. Phase 2 emphasizes hands-on practice with real systematic review components
3. Phase 3 involves refinement, validation, and data synthesis
4. Phase 4 concentrates on manuscript writing and publication preparation

**Purpose/Significance:** Provides a realistic, staged timeline for mastering systematic review methodology and completing a publishable meta-analysis

**Zettel:** [[60_Day_Learning_Framework_Z-001-0580A]]

### 2. PRISMA Guidelines Framework

**Category:** Publication Methodology

**Key Process:** 
```
PRISMA 2020 (27-item checklist) → Flow diagram → Protocol registration → Manuscript structure
```

**Mechanism/Steps:**
1. PRISMA 2020 provides the current gold standard for systematic review reporting
2. Flow diagram documents study selection process transparently
3. PROSPERO protocol registration enhances credibility and reduces bias
4. PRISMA-compliant structure ensures all essential elements are included

**Purpose/Significance:** Establishes the publication framework that reviewers and readers expect, ensuring methodological rigor and transparency

**Zettel:** [[PRISMA_Guidelines_Framework_Z-001-0580B]]

### 3. Key Database Selection Criteria

**Category:** Research Methodology

**Key Process:** 
```
PubMed/MEDLINE (biomedical) → Embase (pharmacology) → Cochrane (trials) → Web of Science (citation) → Scopus (interdisciplinary)
```

**Mechanism/Steps:**
1. PubMed/MEDLINE provides broad biomedical coverage with MeSH indexing
2. Embase excels in European journals and drug-related topics
3. Cochrane Library offers high-quality systematic reviews and RCTs
4. Web of Science enables comprehensive citation tracking
5. Scopus provides broad interdisciplinary coverage with citation analysis

**Purpose/Significance:** Ensures comprehensive literature identification by leveraging complementary database strengths and avoiding systematic bias

**Zettel:** [[Key_Database_Selection_Criteria_Z-001-0580C]]

### 4. Quality Assessment Tools Framework

**Category:** Research Methodology

**Key Process:** 
```
Randomized trials: Cochrane RoB 2 → Non-randomized: ROBINS-I → Observational: STROBE/GRADE
```

**Mechanism/Steps:**
1. Cochrane RoB 2 assesses risk of bias in randomized trials with domain-based evaluation
2. ROBINS-I evaluates risk of bias in non-randomized studies
3. STROBE guidelines ensure quality reporting of observational studies
4. GRADE approach assesses confidence in evidence synthesis

**Purpose/Significance:** Provides appropriate quality evaluation tools for different study designs, ensuring methodological rigor in evidence synthesis

**Zettel:** [[Quality_Assessment_Tools_Framework_Z-001-0580D]]

### 5. Statistical Analysis Methodology

**Category:** Biostatistics

**Key Process:** 
```
Effect size: MD/SMD (continuous) → RR/OR/RD (dichotomous) → Heterogeneity: I²/Q → Publication bias: Funnel plot/Egger's test
```

**Mechanism/Steps:**
1. Mean difference and standardized mean difference for continuous outcomes
2. Risk ratio, odds ratio, and risk difference for dichotomous outcomes
3. I² statistic quantifies heterogeneity magnitude (0-25%: low; >75%: considerable)
4. Funnel plot visualization and Egger's test detect publication bias

**Purpose/Significance:** Establishes the statistical foundation for valid meta-analysis interpretation and robust evidence synthesis

**Zettel:** [[Statistical_Analysis_Methodology_Z-001-0580E]]

### 6. Journal Selection Strategy

**Category:** Academic Publishing

**Key Process:** 
```
Tier 1 (high impact) → Tier 2 (specialized) → Tier 3 (field-specific) → Selection criteria: scope, impact, speed, open access
```

**Mechanism/Steps:**
1. Tier 1 journals (Lancet, BMJ, JAMA) offer highest impact but competitive acceptance
2. Tier 2 journals (Systematic Reviews, BMJ Open) specialize in methodology
3. Tier 3 journals provide field-specific audiences and more targeted scope
4. Selection criteria prioritize scope match, impact factor, publication speed, and open access options

**Purpose/Significance:** Optimizes publication chances by strategically matching methodology quality with journal scope and audience

**Zettel:** [[Journal_Selection_Strategy_Z-001-0580F]]
