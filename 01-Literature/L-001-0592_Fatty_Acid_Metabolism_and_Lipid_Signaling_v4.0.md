# Fatty Acid Metabolism and Lipid Signaling

## Core Concepts Extracted

### 1. Beta-Oxidation Mechanisms

**Category:** Energy Metabolism

**Key Process:** 
```
Palmitate (C16:0) + 7CoA + 7NAD⁺ + 7FAD + 7H₂O → 8Acetyl-CoA + 7NADH + 7FADH₂ + 7H⁺
```

**Mechanism/Steps:**
1. Activation: Fatty acid + CoA + ATP → Acyl-CoA + AMP + PPi
2. Transport: Carnitine shuttle transports acyl-CoA into mitochondria
3. Dehydrogenation: Acyl-CoA + FAD → trans-Δ²-Enoyl-CoA + FADH₂
4. Hydration: trans-Δ²-Enoyl-CoA + H₂O → L-3-Hydroxyacyl-CoA
5. Dehydrogenation: L-3-Hydroxyacyl-CoA + NAD⁺ → 3-Ketoacyl-CoA + NADH + H⁺
6. Thiolysis: 3-Ketoacyl-CoA + CoA → Acetyl-CoA + Acyl-CoA (n-2)

**Purpose/Significance:** Provides the fundamental biochemical pathway for mitochondrial fatty acid oxidation, essential for understanding metabolic flexibility and ketosis physiology

**Zettel:** [[Beta_Oxidation_Clinical_Mechanisms_Z-001-1054]]

### 2. Omega-3 and Omega-6 Subclasses

**Category:** Lipid Biochemistry

**Key Process:** 
```
LA (18:2 n-6) → Δ6-desaturase → GLA (18:3 n-6) → Δ5-desaturase → AA (20:4 n-6)
ALA (18:3 n-3) → Δ6-desaturase → SDA (18:4 n-3) → Δ5-desaturase → EPA (20:5 n-3) → Elongase → DHA (22:6 n-3)
```

**Mechanism/Steps:**
1. Omega-6 fatty acids (LA, AA) serve as precursors for pro-inflammatory eicosanoids
2. Omega-3 fatty acids (ALA, EPA, DHA) serve as precursors for anti-inflammatory resolvins
3. Desaturase enzymes (Δ5, Δ6) convert plant-based precursors to longer-chain derivatives
4. Enzyme activity is influenced by genetics, diet, and metabolic health
5. Competition exists between n-3 and n-6 pathways for the same desaturase enzymes
6. Clinical ratios reflect the balance between pro- and anti-inflammatory pathways

**Purpose/Significance:** Enables understanding of fatty acid metabolism pathways and their clinical implications for inflammation and metabolic health

**Zettel:** [[Fatty_Acid_Subclasses_Desaturase_Pathways_Z-001-1055]]

### 3. Clinical Fatty Acid Ratios

**Category:** Clinical Diagnostics

**Key Process:** 
```
AA/EPA Ratio = Arachidonic Acid / Eicosapentaenoic Acid
Omega-6:3 Ratio = Total Omega-6 Fatty Acids / Total Omega-3 Fatty Acids
Omega-3 Index = EPA + DHA / Total Fatty Acids
```

**Mechanism/Steps:**
1. AA/EPA ratio reflects inflammatory potential of cellular membranes
2. Omega-6:3 ratio indicates dietary balance between pro- and anti-inflammatory precursors
3. Omega-3 Index measures cardioprotective fatty acid status
4. Optimal ratios indicate balanced lipid metabolism and reduced inflammation
5. Elevated ratios signal increased risk for inflammatory conditions
6. Ratios provide objective biomarkers of dietary patterns patients may not self-report

**Purpose/Significance:** Provides clinically actionable biomarkers for assessing metabolic health and guiding nutritional interventions

**Zettel:** [[Fatty_Acid_Ratios_Clinical_Interpretation_Z-001-1056]]

### 4. Ketosis and Metabolic Flexibility

**Category:** Metabolic Physiology

**Key Process:** 
```
Low Carbohydrate → ↓ Insulin → ↑ Lipolysis → ↑ Fatty Acids → ↑ Beta-Oxidation → ↑ Acetyl-CoA → ↑ Ketogenesis → ↑ Ketone Bodies
```

**Mechanism/Steps:**
1. Low carbohydrate availability reduces insulin signaling
2. Reduced insulin promotes lipolysis and fatty acid release
3. Increased fatty acid availability drives beta-oxidation
4. Excess acetyl-CoA from beta-oxidation is converted to ketone bodies
5. Ketone bodies (acetoacetate, β-hydroxybutyrate) serve as alternative fuel
6. Metabolic flexibility represents the ability to efficiently switch between glucose and fatty acid oxidation

**Purpose/Significance:** Provides the physiological framework for understanding ketosis as a metabolic adaptation and its clinical applications in metabolic health

**Zettel:** [[Ketosis_Metabolic_Flexibility_Clinical_Z-001-1057]]

### 5. Carnitine Shuttle System

**Category:** Metabolic Transport

**Key Process:** 
```
Acyl-CoA + Carnitine → Acylcarnitine + CoA (CPT1)
Acylcarnitine → Mitochondrial Matrix (translocase)
Acylcarnitine + CoA → Acyl-CoA + Carnitine (CPT2)
```

**Mechanism/Steps:**
1. CPT1 (carnitine palmitoyltransferase 1) catalyzes acyl-CoA to acylcarnitine conversion
2. Carnitine-acylcarnitine translocase transports acylcarnitine across inner mitochondrial membrane
3. CPT2 converts acylcarnitine back to acyl-CoA inside mitochondria
4. Carnitine deficiency impairs fatty acid transport and beta-oxidation
5. Organic acid testing can detect carnitine-related metabolic disorders
6. Supplementation supports fatty acid oxidation in deficiency states

**Purpose/Significance:** Provides the essential transport mechanism understanding needed for interpreting organic acid tests and managing fatty acid oxidation disorders

**Zettel:** [[Carnitine_Shuttle_Fatty_Acid_Transport_Z-001-1058]]
