## Creating Skills

1. Complete a task successfully
2. Extract the reusable pattern
3. Copy [[Template-Skill]]
4. Document: purpose, when to use, steps, verification, failures
5. Add to this index
6. Use and refine

---

## Skill Evolution Rules

| Transition | Requirement |
|------------|-------------|
| v1.0 → v1.1 | 5+ successful uses, refinement identified |
| v1.x → v2.0 | Major rewrite based on pattern analysis |
| v2.0 → stable | 30 days without changes, documented outcomes |
| stable → deprecated | Superseded by v3.0 or better approach |
