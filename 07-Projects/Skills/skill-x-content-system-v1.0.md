# Skill: X Content System v1.0

> **Professional X media management from Zettelkasten**

---

## Philosophy

Every X post is a **derivative** of synthesis, which is an integration of zettels.

**Quality hierarchy:** Zettel → Synthesis → X Post

---

## Content Types

| Type | Format | Source | Frequency |
|------|--------|--------|-----------|
| **Thread** | 5-10 tweets | 1 synthesis | 1x daily |
| **Single** | 1 tweet | 1 zettel | 2-3x daily |
| **Reply** | Contextual | Real-time | As needed |
| **Quote** | Commentary | External + zettel | Curated |

---

## Thread Architecture

### Tweet 1/5 — Hook (Pattern: Contrarian)
```
Everyone believes [common view].

But [evidence-based contradiction]:

[Mechanism hint]

🧵
```

### Tweet 2-4/5 — Evidence (Pattern: Mechanism)
```
[Claim]

→ [Mechanism]
→ [Mechanism]
→ [Mechanism]

Result: [Outcome]
```

### Tweet 5/5 — Action (Pattern: Utility)
```
Simple [category] to [benefit]:

• [Action 1]
• [Action 2]
• [Action 3]

Start with ONE.

[CTA: Follow, RT, Reply]
```

---

## Voice Guidelines

| Element | Rule |
|---------|------|
| **Authority** | "I've seen patients..." / "In clinical practice..." |
| **Accessibility** | No jargon without explanation |
| **Controversy** | Challenge consensus respectfully |
| **Utility** | Every post has actionable takeaway |
| **Brevity** | 280 chars max, punchy sentences |

---

## Workflow

### Daily (RON)
1. **Morning** — Review [[MOC-Social-Content]] queue
2. **Select** — Pick synthesis from 04-Synthesis/
3. **Draft** — Create X post derivative
4. **Schedule** — Use X Premium+ scheduling
5. **Engage** — Reply to comments (30 min)

### Weekly (FORG)
1. **Batch** — Create 7 X posts from syntheses
2. **Queue** — Schedule for week ahead
3. **Report** — Engagement metrics to [[Issue-040]]
