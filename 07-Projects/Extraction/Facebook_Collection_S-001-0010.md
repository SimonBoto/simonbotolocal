## Lessons Learned

### What Worked
- **Batch processing** — GOLD/SILVER/BRONZE tiers
- **Template v2.0** — Consistent YAML structure
- **Git checkpoints** — Every 10 items
- **QMD integration** — Prevented duplicates

### What Didn't
- **Initial bracket naming** — Created 1,900+ file renames
- **Shadow links** — Guessed zettel names without verification
- **PMID gaps** — Many sources lack PubMed IDs

### Applied to Current Work
- FORG agent designed with these lessons
- CGPT extraction uses same tier system
- Bracket naming NEVER AGAIN
