## Creating Skills

1. Complete a task successfully
2. Extract the reusable pattern
3. Copy [[Template-Skill]]
4. Document: purpose, when to use, steps, verification, failures
5. Add to [[Skill-Index]]
6. Use and refine

---

## Evolution Rules

| Transition | Requirement |
|------------|-------------|
| v1.0 → v1.1 | 5+ successful uses |
| v1.x → v2.0 | Major rewrite from pattern analysis |
| v2.0 → stable | 30 days without changes |
