# METHAP Biomarker Integration Panel

## Rationale

Multiple biomarkers provide complementary windows into pathophysiology:
- **VitD**: Immunomodulation status
- **Adipic**: Mitochondrial function
- **Thyroid Antibodies**: Autoimmune activity
- **Inflammatory Cytokines**: Systemic inflammation

## The Integrated Model

```
Hashimoto's Pathophysiology
        ↓
┌─────────────────────────────────────┐
│  Environmental Triggers             │
│  (Stress, infection, toxins)        │
└─────────────┬───────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  Vitamin D Deficiency               │
│  ↓ Treg function                    │
│  ↑ Th17/Th1 activity                │
└─────────────┬───────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  Mitochondrial Stress               │
│  ↓ Beta-oxidation                   │
│  ↑ Adipic/Suberic (omega-oxidation) │
└─────────────┬───────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  Thyroid Autoimmunity               │
│  TPO-Ab, TG-Ab production           │
│  Thyrocyte damage                   │
└─────────────┬───────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  Clinical Hypothyroidism            │
│  Fatigue, weight gain, brain fog    │
└─────────────────────────────────────┘
```

## Intervention Points

| Stage | Biomarker | Intervention |
|-------|-----------|--------------|
| Early | VitD <30 ng/mL | High-dose VitD |
| Metabolic | Adipic >2.0 | Carnitine, CoQ10 |
| Autoimmune | TPO-Ab rising | Selenium, zinc |
| Established | fT3 low | L-thyroxine + above |
