## 🚀 Quick Start

**To work on App Production:**

1. Check `00-ROADMAP.md` for current sprint
2. Pick skill from `01-Skills/`
3. Write requirements in skill's `README.md`
4. Spawn dev agent (FREE/GROQ/PAID as appropriate)
5. RON reviews → integrates
6. Update ROADMAP

**To deploy a skill:**
```bash
# Install to OpenClaw
openclaw skills install ./01-Skills/memory-continuity

# Test
openclaw skills test memory-continuity

# Enable
openclaw skills enable memory-continuity
```
