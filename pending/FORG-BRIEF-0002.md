## Summary
Short ChatGPT convo explaining "communal chat": not a multi-user group chat, but single-user thread with shared system prompts (tone, safety, tools) as backstage instructions. User invites friend expecting join, but system treats as one user; no shared multi-human session. Can simulate collaboration by turn-taking or summarizing across chats.

## Key Points
- Communal = shared AI instructions, not user-shared space.
- Invites don't create group; each user separate thread.
- Simulate multi-person via one keyboard, switching voices.
- Digital campfire metaphor: two at fire, system glow visible to AI only.
