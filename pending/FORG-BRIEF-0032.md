## Summary
Guide to elicit academic/dissertation-style responses from ChatGPT: use formal lang, context/background, detailed analysis w/refs, critical eval. Specify length (e.g., 200/400 words), structure subtopics. Test: micronutrient supp in IR/diabetes - principles (cofactors in metab/insulin sens), deficiencies (obese/T2D pops), multi>mono, trials (VitD/mag improve glycemic), specifics (VitD beta-cell/inflam, mag sens, B-vits energy/DNA, Zn/Se/Cu/Fe antiox/insulin sig). Refs incl Jenkins 2020, Pittas 2019.

## Key Points
- Prompt struc: clear Q + bg + analysis + eval + formal.
- Length: state words, divide topics, prioritize.
- Micronuts: VitD/mag/Cr key; defic in processed diets/obesity.
- Multi synerg > mono; trials mixed but promising.
- Specials: glutamine(?), VitC ox stress, B-complex metab, minerals signaling.
