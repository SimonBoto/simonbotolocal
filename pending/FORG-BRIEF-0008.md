## Summary
Voice convo (Greek): User can't transfer WhatsApp to new phone; asks for username but not distinct. ChatGPT guides: backup chats to Google Drive (Android)/iCloud (iOS) via Settings > Chats > Chat backup > Create. Install WA on new phone, verify same number, restore backup. Steps: 3-dots > Settings > Chats > Backup.

## Key Points
- Backup first on old phone (auto daily/weekly/monthly).
- New phone: install WA, same number, restore when prompted.
- Stable internet for verification; no username needed, phone num suffices.
