# 02-References — README

> **Canonical sources — papers, books, datasets. The foundation of evidence.**

---

## Purpose

References are **canonical sources** — the raw material of knowledge.

This folder contains:
- **Hub/** — Reference notes (processed sources)
- **Zotero/** — Zotero exports and sync
- **Other/** — Non-standard sources
- **Assets/** — PDFs, images, supplementary files

---

## Reference Note Format

**Purpose:** Canonical record of a source, independent of any specific use.

**Contains:**
- Full bibliographic data
- Abstract/summary
- Key findings
- Personal notes
- Links to zettels that cite this source

**Naming:** `Author_Year_Title_R-XXX-XXXX.md`

---

## Relationship to Literature Notes

| Aspect | Reference Note | Literature Note |
|--------|---------------|-----------------|
| **Purpose** | Canonical source record | Processing output |
| **Content** | Bibliographic + abstract | Extracted insights |
| **Links** | Cited by zettels | Links to zettels |
| **Status** | Permanent | Can evolve |

**Flow:** Source → Reference Note → Literature Note → Zettels

---

## Zotero Integration

**Zotero/** folder syncs with Zotero reference manager.

- Export references from Zotero
- Import to vault as needed
- Keep canonical version in Hub/

---

## Rules

1. **One reference = one note** — No duplicates
2. **Complete bibliography** — All fields filled
3. **Link to zettels** — Track where cited
4. **Assets organized** — PDFs in Assets/, named consistently
