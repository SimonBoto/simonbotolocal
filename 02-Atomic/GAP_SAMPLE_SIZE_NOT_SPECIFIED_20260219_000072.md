# Gap: Sample Size and Power Calculation Not Specified

**Missing Element:** Sample size (n) for cases and controls not stated in protocol

**Implications:**
- Cannot assess statistical power
- Feasibility unclear
- Budget planning difficult
- IRB review incomplete

**Standard Requirements:**
| Element | Status | Needed |
|---------|--------|--------|
| Sample size | ❌ Missing | n = ? per group |
| Effect size | ❌ Missing | Expected metabolite change |
| Power | ❌ Missing | Typically 80% |
| Alpha | ❌ Missing | Typically 0.05 |
| Dropout rate | ❌ Missing | Typically 10-20% |

**Estimation from Context:**
- Case-control metabolomics studies: typically 20-50 per group
- Multivariate analysis: minimum 10 samples per variable
- With 100+ metabolites: need substantial sample size

**Recommendation:**
Conduct power analysis based on:
1. Pilot data (if available)
2. Published effect sizes in similar studies
3. Conservative estimate: 50 cases + 50 controls minimum
